# E02: CNN 반발력장 벤치마크 (CIFAR-10)

MLP 53% → CNN 70%+? 반발력장 우위 유지되는가?

## 설정

```
  데이터: CIFAR-10 (32×32 컬러, 10 클래스)
  CNN backbone: Conv2d(3→32→64→128) + BatchNorm + MaxPool + AdaptiveAvgPool → 128-dim
  데이터 증강: RandomHorizontalFlip + RandomCrop(32, padding=4)
  정규화: (0.4914,0.4822,0.4465), (0.2023,0.1994,0.2010)
  학습: 30 에폭, Adam lr=0.001
  5개 모델: Dense, TopK MoE, Repulsion(2극), RepulsionQuad(4극), MetaFixed{1/2,1/3,1/6}
```

## 학습 곡선

```
  Model                            E1     E5    E10    E15    E20    E25    E30
  ---------------------------------------------------------------------------
  CNN+RepulsionQuad             46.2%  63.0%  67.8%  74.8%  73.5%  75.9%  78.1%
  CNN+TopK MoE                  41.7%  63.4%  66.7%  71.5%  75.1%  76.0%  78.1%
  CNN+Repulsion                 49.9%  60.6%  70.5%  72.9%  74.1%  76.5%  78.1%
  CNN+Meta{1/2,1/3,1/6}         47.3%  62.9%  69.9%  73.0%  73.5%  73.0%  77.4%
  CNN+Dense                     52.6%  63.6%  68.3%  72.7%  73.4%  76.9%  77.0%
```

## {1/2,1/3,1/6} 가중치 이탈

```
  초기: [0.5000, 0.3333, 0.1667]
  학습: [0.3399, 0.3470, 0.3130]
  L2 drift: 0.2173
  → 균등(1/3, 1/3, 1/3)으로 수렴!
```

## 핵심 답변

```
  Q1: 반발력장 우위 CNN에서도 유지?
      CNN+Repulsion(78.07%) > CNN+Dense(77.03%) = +1.04%
      → YES, 유지

  Q2: {1/2,1/3,1/6} 여전히 최적?
      CNN+MetaFixed(77.39%) < CNN+RepulsionQuad(78.12%) = -0.73%
      → NO, CNN에서 최하위

  Q3: CNN 개선 폭?
      MLP 평균 52.41% → CNN 평균 77.74% = +25.33%

  Q4: 학습된 가중치 이탈?
      {0.50, 0.33, 0.17} → {0.34, 0.35, 0.31}
      → 균등으로 수렴 (L2=0.217)
```
