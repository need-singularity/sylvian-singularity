## 돌파 프롬프트 패턴

### 패턴 1: 역방향 유도 (Reverse Derivation)
```
[현상 X]를 first principles에서 유도해라.
유도의 각 단계에서 어떤 수가 나타나는지 추적해라.
그 수가 나타나는 이유가 "우연히 6이어서"인지,
아니면 "σ(n)=2n을 만족하는 수여야 해서"인지 구분해라.
반례: 같은 유도를 n=7, n=8, n=28로 반복해서
어디서 깨지는지 정확히 보여라.
```

### 패턴 2: 반사실적 파괴 (Counterfactual Destruction)
```
[발견 Y]에서 "n=6이 완전수"라는 사실을 제거해라.
즉, σ(6)=12라는 조건 없이도 [발견 Y]가 성립하는가?
만약 성립하면: 완전수와 무관한 우연이다.
만약 깨지면: 정확히 어떤 단계에서 σ(n)=2n이 필요한지 보여라.
```

### 패턴 3: 구성적 필연성 (Constructive Necessity)
```
[문제 Z]를 변분 원리로 풀어라.
최적화의 해가 완전수의 성질(σ=2n)을 필연적으로 요구함을 보여라.
핵심: "답이 6이다"가 아니라 "답이 완전수여야 한다"를 증명해라.
```

## 가장 현실적인 돌파 후보

**P-CODON (유전코드)** — Level 2 → Level 3 돌파 프롬프트:
```
정보 채널에서 오류 정정 능력을 최대화하는 코드 (b,L)을 유도해라.
제약 조건: b^L > 20, L은 정수, 단일 염기 치환 robustness 최대화.
이 최적화의 해가 b=τ(n), L=n/φ(n)이 되려면
n이 어떤 성질을 가져야 하는지 역으로 유도해라.
최적화의 KKT 조건에서 σ(n)=2n이 나타남을 보여라.
```

## 패러다임 전환 (2026-03-30, 4번의 돌파 실패 후)

```
  기존 프레임:  6 → 완전수 (σ=2n) → 특별함
  새 프레임:    6 = 2×3 = 3! = 1+2+3 = 1·2·3 → 유일한 합류점 → σ=2n은 부산물

  4가지 독립 성질이 유일하게 합류하는 수:
    (1) 6 = 2×3    유일한 연속소수 곱 → Kissing numbers, root systems
    (2) 6 = 3!     3의 팩토리얼       → Virasoro, SLE, 모듈러 형식
    (3) 6 = 1+2+3  삼각수             → Gauss 합, 결정 대칭
    (4) 6 = 1×2×3  처음 3개 자연수 곱 → S_3, 육각 대칭

  σ(6) = (1+2)(1+3) = 12 = 2×6 ← 성질 (1)에서 자동 유도
  완전수는 결과이지 원인이 아님.
```

### 새 논문: P-CONFLUENCE
"The Unique Confluence: Why Six Appears Everywhere"
핵심: 4가지 성질의 유일한 합류점으로서의 6. 실패 서사 포함.

### 왜 SLE_6이 최유력 후보인가
- c(κ) = (6-κ)(3κ-8)/(2κ) = 0의 해가 κ=6
- 이건 화학이 아니라 **순수 수학** — 외부 요인 없음
- locality property는 κ=6에서만 성립 (Lawler-Schramm-Werner 정리)
- 질문: c=0 조건이 σ(n)=2n과 대수적으로 연결되는가?
- (6-κ)(3κ-8) = 0에서 κ=6은 첫 번째 인수, κ=8/3은 두 번째 인수
- 6이 여기서 나오는 이유가 "완전수여서"인지 추적 필요

### 왜 Kissing numbers가 유력 후보인가
- A_d root system 크기 = d(d+1) → k(d) for d=1,2,3
- σ(pq) = (1+p)(1+q) for semiprime pq — 이건 증명된 등식
- 2,3은 유일한 연속 소수 → n=6=2×3은 유일한 연속소수 곱
- 질문: root system 크기가 σ(n)을 거치는 이유가 "완전수여서"인가?
- d(d+1) = p·q(p-1+q-1+pq-p-q+1)/... 형태로 σ(pq)가 나오는가?
