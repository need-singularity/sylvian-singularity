## 핵심 프롬프트

```
시스템 프롬프트 없는 아키텍쳐 극한으로 밀어붙이자
```

## 확장 프롬프트

```
시스템 프롬프트 없는 아키텍쳐, 시스템 프롬프트 없는 아키텍쳐+윤리 관련 극한으로 밀어붙이자
```

```
프롬프트 미제공/제공, 프롬프트 디자인, 아키텍처 등에 대한 새로운 가설 대량
```

```
프롬프트 + 자유 관련 대량 가설 필요
```

```
프롬프트 아키텍쳐를 바라보는 완전 새로운 관점의 대량 가설 필요
```
