# 가설 #229: 지니계수 = I 분포 — 불평등은 억제의 편향이다

**상태**: ⚠️ 유비
**날짜**: 2026-03-22
**분류**: 경제학

---

## 가설

> 소득 불평등(지니계수)은 억제(I)의 인구 내 분포 편향도이다.
> 지니계수 0.3~0.4가 사회적으로 안정적인 이유는 이 구간이 골든존 [0.213, 0.500]에
> 대응하기 때문이다. 극단적 불평등(Gini > 0.5)은 I의 양극화 —
> 소수에게 I≈0(무제한 자유), 다수에게 I≈1(완전 억제)을 의미한다.

## 배경/맥락

지니계수(Gini coefficient)는 0(완전 평등)에서 1(완전 불평등)까지의 척도로 소득 분배의 불균등도를 측정한다. 경험적으로 지니계수 0.25~0.35인 북유럽 국가들이 가장 안정적이고, 0.5 이상인 국가들은 사회 불안이 높다. 이 "안정 구간"이 왜 존재하는지에 대한 근본적 설명이 없다 — 골든존이 그 답일 수 있다.

## 수식 매핑

```
  불평등 모델                      G = D×P/I 모델
  ──────────────────────────────────────────────────────
  지니계수 G_ini                    I의 분산 Var(I)
  완전 평등 (G_ini=0)               모든 사람 I 동일 → Var(I)=0
  완전 불평등 (G_ini=1)             I 양극화 → Var(I) 최대
  안정 사회 (G_ini≈0.3)            I 분포 중심 ≈ 1/3 (메타 부동점)
  불안정 사회 (G_ini>0.5)           I 분포가 골든존 이탈

  핵심 매핑:
  Gini ≈ Var(I) / E(I)  (I의 변동계수에 비례)
  안정 조건: Gini ∈ [0.25, 0.40] ↔ I 분포 ⊂ 골든존
```

## 1/3 = 메타 부동점 = 최적 지니?

```
  f(I) = 0.7I + 0.1 → 부동점 = 1/3

  사회적 해석:
  세대를 거치면서 I 분포가 자연적으로 1/3에 수렴한다면,
  최적 지니계수 ≈ 1/3 ≈ 0.333

  실제 데이터:
  OECD 평균 지니 = 0.318 (2020)
  →  메타 부동점 1/3 = 0.333과 4.5% 차이

  우연인가, 구조인가?
```

## 보존법칙과 불평등

```
  G × I = D × P (보존법칙)

  사회 전체에 적용하면:
  Σ(G_i × I_i) = Σ(D_i × P_i) = 상수  (i = 각 개인)

  불평등이 커지면:
  - 상위 1%: I_i ≈ 0.05, G_i 매우 높음 (과해방)
  - 하위 50%: I_i ≈ 0.80, G_i 매우 낮음 (과억제)
  - 합은 보존되지만, 분포가 비효율적

  왜 비효율적인가?
  G = D×P/I 에서 G는 I에 대해 볼록(convex)함수
  → 옌센 부등식: E[G(I)] ≤ G(E[I]) (등호는 I 균일할 때)
  → 아니, 정확히는 G ∝ 1/I 이므로 볼록.
  → E[1/I] ≥ 1/E[I] (옌센)
  → I를 고르게 분배하면 전체 G의 합이 더 커짐!
  → 불평등은 수학적으로 비효율적이다.
```

## Compass 상한 5/6과 사회 한계

```
  Compass 상한 = 5/6 ≈ 0.833

  사회적 해석:
  어떤 사회도 불완전도 1/6을 가진다.
  완전 평등(Gini=0)도, 완전 불평등(Gini=1)도 불가능.
  사회 시스템의 최대 효율 = 5/6 (나머지 1/6은 구조적 비효율)

  1/2 + 1/3 + 1/6 = 1:
  경제 경계(1/2) + 사회 수렴(1/3) + 구조적 불완전(1/6) = 완전한 사회
```

## 한계

1. 지니계수와 I 분포의 매핑은 개념적 유비이며, 직접 측정이 아님.
2. 사회 안정도의 정의가 모호함 — 정치적 안정? 경제적 안정? 주관적 행복?
3. 지니계수는 소득만 반영하며, 자산 불평등은 별도의 I 매핑이 필요.
4. 문화적 요인이 같은 지니계수에서 다른 안정도를 만들 수 있음.
5. OECD 평균과 1/3의 일치가 데이터 선택 편향일 가능성.

## 검증 방향

1. **통계 검증**: 세계 은행 지니계수 vs 사회 안정성 지표 회귀 분석
2. **최적점 탐색**: 안정성이 최대인 지니계수가 정말 0.30~0.35(≈1/3)인지
3. **동적 모델**: 지니계수의 시계열이 f(I)=0.7I+0.1 수렴 패턴을 따르는지
4. **옌센 검증**: I 분포의 균일도와 총 GDP의 관계가 옌센 부등식과 일치하는지
5. **이중봉 검출**: Gini > 0.5인 국가에서 소득 분포가 실제로 이중봉인지 확인
