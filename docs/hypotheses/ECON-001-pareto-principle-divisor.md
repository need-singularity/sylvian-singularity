# ECON-001: 파레토 법칙 — 80/20 = τ(6)/sopfr(6)

> **가설**: 파레토 법칙의 80/20 비율이 τ(6)/sopfr(6) = 4/5 : 1/sopfr(6) = 1/5로 표현된다.

## 수치 검증

| 비율 | 값 | P₁ 표현 |
|------|------|---------|
| 80% | 4/5 | τ(6)/sopfr(6) |
| 20% | 1/5 | 1/sopfr(6) |

```
      20%      80%        원인(%)
      │

```

## 한계
- 80/20은 근사적 경험 법칙이지 정확한 비율이 아님
- τ(6)/sopfr(6) 연결은 우연일 가능성
