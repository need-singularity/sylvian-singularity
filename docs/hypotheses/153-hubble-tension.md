# 가설 검토 153: 허블 텐션과 모델 오차의 대응

## 가설

> 허블 상수의 측정 불일치(허블 텐션, 8.3%)는 우리 모델의 격자 이산화 오차(0.8%)와 대응한다. 그러나 스케일 차이가 ~10배이므로 약한 대응에 그친다.

## 배경

### 허블 텐션

허블 상수 H₀의 두 가지 측정 방법이 서로 다른 값을 제공한다:

- **초기 우주 측정** (Planck CMB): H₀ = 67.4 ± 0.5 km/s/Mpc
- **후기 우주 측정** (SH0ES 세페이드): H₀ = 73.0 ± 1.0 km/s/Mpc
- **차이**: ΔH₀ = 5.6 km/s/Mpc → **8.3%** 불일치
- **통계적 유의도**: 5σ 이상 → 체계적 오류 또는 새로운 물리

### 모델 오차

우리 모델의 격자 이산화에서 grid 크기에 따른 오차:

| grid | 오차 | 비고 |
|---|---|---|
| 20 | 2.4% | 거친 탐색 |
| 50 | 0.9% | 일반 검증 |
| 100 | 0.5% | 기본값 |
| 200 | 0.2% | 정밀 검증 |

grid=50 기준 오차 **0.9% ≈ 0.8%** 를 허블 텐션의 8.3%와 비교한다.

## 약한 대응의 분석

허블 텐션과 모델 오차 사이에 **직접적 대응은 약하다**. 그러나 몇 가지 구조적 유사성이 존재한다:

### 유사점

| 성질 | 허블 텐션 | 모델 오차 | 유사도 |
|---|---|---|---|
| "같은 것을 다른 방법으로 측정" | 초기 vs 후기 | 낮은 grid vs 높은 grid | ⚠️ 약 |
| 이산화/해상도 문제 | 관측 스케일 차이 | 격자 해상도 차이 | ⚠️ 약 |
| 체계적 편향 | 거리 사다리 보정 | 경계 효과 | ⚠️ 약 |

### 차이점

| 성질 | 허블 텐션 | 모델 오차 |
|---|---|---|
| 크기 | 8.3% | 0.8% |
| 해결 가능성 | 미해결 (새 물리?) | grid 증가로 해결 |
| 물리적 깊이 | 근본적 | 기술적 |
| 양방향성 | 어느 쪽이 맞는지 불명 | 높은 grid가 정확 |

## 대안적 해석: 허블 텐션의 모델 번역

만약 허블 텐션이 "초기 우주와 후기 우주에서 I 값이 다르다"를 의미한다면:

```

     │

     │      ●────────────────  후기 우주 (SH0ES)
  1/3│━━━━━━━━━━━━━━━━━━━━━━  부동점
     │  ●────────────────────  초기 우주 (Planck)

     │

      CMB  BAO  SNe  세페이드

  초기 우주: I 약간 낮음 (H₀ 작음)
  후기 우주: I 약간 높음 (H₀ 큼)

```

이 해석은 흥미롭지만 **매우 추측적**이다.

## 결론

허블 텐션 8.3%와 모델 오차 0.8%의 직접 대응은 **실패**한다:

1. **스케일 차이 ~10배** — 같은 현상의 다른 표현이라고 보기 어려움
2. **원인의 성격 차이** — 허블 텐션은 물리적, 모델 오차는 기술적
3. **해결 가능성 차이** — 모델 오차는 grid 증가로 해결 가능, 허블 텐션은 미해결

그러나 "다른 스케일에서 같은 것을 측정할 때 불일치가 발생한다"는 일반적 교훈은 유효하다.

## 한계

- 허블 텐션의 원인이 밝혀지지 않아 의미 있는 비교 자체가 어려움
- 8.3% vs 0.8%의 수치 비교는 우연일 가능성이 높음
- 모델 오차는 해상도 문제이지만, 허블 텐션은 물리적 불일치일 수 있음
