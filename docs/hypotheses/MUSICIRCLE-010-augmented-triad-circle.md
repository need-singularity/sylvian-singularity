# MUSICIRCLE-010: 증3화음 원 = τ(6)=4개

> **가설**: 전위불변 증3화음(augmented triad)은 4 = τ(6)개이며, S¹을 P₁/2=3등분한다.

## 수치 검증

```
4 증3화음 (Z₃ 부분군의 coset):

  Aug₀: C  - E  - G#    (0, 4, 8)
  Aug₁: C# - F  - A     (1, 5, 9)
  Aug₂: D  - F# - Bb    (2, 6, 10)
  Aug₃: Eb - G  - B     (3, 7, 11)

  원 위의 배치:
       C
    B     C#     Aug₀: 정삼각형 △
   Bb       D    
  A    (○)   Eb  Aug₁: 회전 +1
   Ab       E    Aug₂: 회전 +2
    G     F      Aug₃: 회전 +3
       F#
```
