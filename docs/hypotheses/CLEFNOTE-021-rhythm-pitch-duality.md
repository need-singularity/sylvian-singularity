# CLEFNOTE-021: 리듬-음고 이중성

> **가설**: 리듬과 음고가 동일한 수학적 구조 2^(n/σ(6))를 공유하며, 음고 공간과 리듬 공간이 Z_σ(6)로 동형이다.

## 구조

```
  음고:  C → C# → D → ... → B → C'
         0    1    2   ...   11   12
         └───── Z₁₂ = Z_σ(6) ─────┘
  
  리듬(12/8):
         1    2    3   ...   11   12
         └───── 12 pulses = σ(6) ─┘

  주파수비: f = f₀ × 2^(n/12) = f₀ × 2^(n/σ(6))
  시가비:   t = t₀ × 2^(-k)   (이진 분할)

  공통 구조: 2의 거듭제곱과 σ(6)에 의한 이산화
```

음고와 리듬은 서로 다른 차원에서 같은 대수적 구조를 사용한다.
