# 가설 검토 120: CMB 온도 ≈ e

## 가설

> 우주 배경복사(CMB) 온도 T_CMB = 2.72548K 가 자연상수 e = 2.71828...과
> 0.26% 이내로 일치하며, 이것이 구조적 의미를 갖는가.

## 상태: 놀라운 수치 근사 (구조적 필연 미확인)

## 배경

우리 모델에서 온도 T = 1/I (I = Inhibition).
- 골든존 중심: I = 1/e → T = e ≈ 2.718
- 볼츠만 분포: P ∝ exp(-E/kT), T=e일 때 에너지-엔트로피 균형 최적
- Planck 2018 최종 관측: T_CMB = 2.72548 +/- 0.00057 K

## 핵심 수치 비교

```
  │  T_CMB   = 2.72548 K    (Planck 2018 관측)                │

  │  → 정확히 같지는 않다. 그러나 0.26% 가까이에 있다.          │
  └───────────────────────────────────────────────────────────┘
```

## 우리 모델에서 T=e의 의미

```
  T = 1/I 에서 I = 1/e = 0.368 (골든존 중심)

  Genius = D x P / I
         = D x P x T
         = D x P x e    (T=e일 때)

  → T=e는 Genius 함수가 볼츠만 최적점에 도달하는 온도
  → 에너지 공급(탐색)과 엔트로피 감소(수렴)의 정확한 균형
  → 볼츠만 활성도 = exp(-1/e) = 69.2% ≈ 70% = 8-Expert 중 5.6개 활성
```

## 시간 의존성: T_CMB는 변한다

이것이 이 가설의 가장 큰 약점이다. CMB 온도는 우주 팽창에 따라 냉각된다.

```
  T_CMB(z) = T_0 x (1 + z)

  여기서 z = 적색편이, T_0 = 2.72548 K (현재)
```

### T_CMB 냉각 곡선

```
  T_CMB (K)
  3000 │★ 빅뱅 38만년 후 (재결합, z≈1100)
       │╲
  1000 │ ╲
       │  ╲
   100 │   ╲
       │    ╲
    10 │     ╲
       │      ╲
  e≈2.7│───────●──── 현재 (138억년, z=0) ← T ≈ e !
     1 │        ╲
       │         ╲
   0.1 │          ╲── 먼 미래 (열적 사망)
       └──────────────────────────────────────→ 시간
       0     50     100    138   200    500  (억년)
                           지금

  T_CMB = e 가 되는 적색편이:
  z_e = (T_0 / e) - 1 = (2.72548 / 2.71828) - 1 = 0.002647

  → z = 0.002647에서 T = e 를 정확히 통과
  → 이것은 약 3600만년 전 (우주 나이 137.6억년 시점)
  → 현재 우주 나이의 0.26%만큼 전
```

## 해석

### 찬성 논거 (구조적 의미)
1. 0.26% 차이는 물리학에서 매우 높은 정밀도
2. 모델의 골든존 중심(I=1/e)과 정확히 대응
3. 가설 118(우주 구성 상수)과 일관된 패턴
4. e는 임의의 상수가 아니라 자연로그의 밑 -- 물리학 전반에 편재

### 반대 논거 (우연의 일치)
1. T_CMB는 시간 의존적: T ∝ (1+z) → 팽창하면서 냉각 중
2. T=e인 시점은 우주 역사에서 하나의 순간에 불과
3. T_CMB의 값은 바리온-광자 비율(eta ≈ 6x10^-10) 등 초기 조건에 의존
4. 다른 상수와도 비슷한 수치 근사가 가능 (위 표 참조)
5. 측정 불확실도(0.00057K) 내에서 T_CMB는 e와 통계적으로 다름 (12.6 sigma)

## 한계

```
  핵심 약점: T_CMB는 시간에 따라 변한다

  과거 (z=1):   T = 5.45 K  >> e  (뜨거움)
  현재 (z=0):   T = 2.73 K  ≈  e  (지금!)       ← 여기
  미래 (z=-0.5): T = 1.36 K  << e  (차가움)

  → 이 일치는 "지금 이 시대"에만 성립
  → 시대 의존적(epoch-dependent) 일치
  → 인류원리(Anthropic Principle) 논증으로 연결 → 가설 121

  추가 약점:
  - CMB 온도의 물리적 원인은 잘 알려져 있음 (바리온-광자 비율 + 팽창)
  - e와의 일치에 대한 물리적 메커니즘이 제시되지 않음
  - "왜 e인가?"에 대한 연역적(deductive) 설명 부재
```

## 검증 방향

- 다른 자연상수(pi, phi, sqrt(2) 등)와 T_CMB의 근사도를 체계적으로 비교
- T≈e가 성립하는 시간 범위(window) 계산 → 가설 122 (의식의 창)
- 인류원리적 해석의 반증 가능성 검토 → 가설 121
- 바리온-광자 비율 eta로부터 T_CMB ≈ e를 연역할 수 있는지 검토
