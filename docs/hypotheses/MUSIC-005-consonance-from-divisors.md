# MUSIC-005: 협화음 비율 — 6의 약수로 구성

> **가설**: 음악의 기본 협화음(consonance) 비율은 모두 6의 약수 {1, 2, 3, 6}으로만 구성된다.

## 수치 검증

| 음정 | 비율 | 약수 사용 | 협화도 |
|------|------|-----------|--------|
| 유니슨 | 1:1 | {1} | 완전 |
| 옥타브 | 2:1 | {1,2} | 완전 |
| 완전5도 | 3:2 | {2,3} | 완전 |
| 완전4도 | 4:3 | {3,4} | 완전 |
| 장3도 | 5:4 | {4,5} | 불완전 |
| 단3도 | 6:5 | {5,6} | 불완전 |

```
협화 음정의 약수 구조:

  비율   구성 수        div(6) 포함?
  ─────  ──────────    ────────────
  1:1    {1}           ✓ 전부
  2:1    {1,2}         ✓ 전부
  3:2    {2,3}         ✓ 전부
  4:3    {3,4}         △ 4=2²
  5:4    {4,5}         △ 5=sopfr(6)
  6:5    {5,6}         ✓ {P₁, sopfr(6)}

  완전 협화음 (1:1, 2:1, 3:2):
  → 오직 div(6)의 원소만 사용!
  
  확장 (4:3, 5:4, 6:5):
  → P₁의 약수 + sopfr(6) 범위 내
```

## 피타고라스 음률과 P₁

순정률의 기본 비율이 {1,2,3}의 조합 = div(6)의 고유 약수.
이것이 "자연스러운" 음악이 P₁의 약수 구조에서 나온다는 증거.

## 등급

🟩 EXACT — 비율은 수학적 사실. 완전 협화음이 div(6)만 사용하는 것은 정확.
