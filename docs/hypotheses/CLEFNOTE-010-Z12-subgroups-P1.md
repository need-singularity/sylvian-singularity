# CLEFNOTE-010: Z₁₂ 부분군 = P₁ = 6개

> **가설**: Z₁₂의 부분군 수 τ(12) = 6 = P₁이며, 각 부분군이 음악적 대칭 구조에 대응한다.

## 부분군 ↔ 음악 구조

| 부분군 | 위수 | 음악적 의미 | 음 |
|--------|------|------------|-----|
| Z₁ | 1 | 유니슨 | C |
| Z₂ | 2 | 트라이톤 쌍 | C, F# |
| Z₃ | 3 | 증3화음 | C, E, G# |
| Z₄ | 4 | 감7화음 | C, Eb, Gb, A |
| Z₆ | 6 | 온음 음계 | C, D, E, F#, G#, A# |
| Z₁₂ | 12 | 반음계 | 전체 |

```
부분군 격자 (하세 도표):

       Z₁₂ (12)
      / | ╲
    Z₆  Z₄  ×
    (6) (4)
    / ╲  |
  Z₃  Z₂
  (3) (2)
    ╲ /
     Z₁
     (1)

  부분군 위수: {1, 2, 3, 4, 6, 12} = div(6) 그 자체!
```

이것은 τ(12) = τ(σ(6)) = P₁이라는 자기참조적 구조.
