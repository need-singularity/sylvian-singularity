# 가설 검토 224: 진화 = 볼츠만 분포

## 상태: ⚠️ 유비

## 분류: 진화론

## 가설

> 진화에서의 돌연변이율은 볼츠만 분포의 온도 T에 매핑된다. I = 1/(kT) 관계에서 높은 돌연변이율(높은 T)은 낮은 I를, 낮은 돌연변이율(낮은 T)은 높은 I를 의미한다. 최적 돌연변이율은 I를 골든존 중심(1/e ≈ 0.368)에 위치시키는 값이다.

## 배경 / 맥락

가설 004에서 볼츠만 분포와 Inhibition의 관계를 I = 1/(kT)로 정립했다. 이를 진화적 맥락으로 확장한다.

열역학에서:
- **높은 T** → 분자 운동 활발 → 무질서 증가
- **낮은 T** → 분자 운동 둔화 → 질서 유지

진화에서:
- **높은 돌연변이율** → 유전적 변이 폭발 → 불안정하지만 적응 잠재력 큼
- **낮은 돌연변이율** → 유전적 안정 → 안정하지만 적응 능력 제한

이 유비를 정량화하면: 돌연변이율 μ = 진화적 온도 T_evo, 그리고 I = 1/(k × T_evo)

관련 가설: 가설 004(볼츠만-억제 온도), 가설 130(볼츠만 k), 가설 221(자연선택)

## 수식 매핑

```
  볼츠만 열역학              진화적 해석
  ──────────────            ──────────────
  온도 T                →   돌연변이율 μ (mutations/bp/generation)
  볼츠만 상수 k          →   진화적 스케일 상수 k_evo
  에너지 E              →   적응 비용 (fitness cost)
  분배함수 Z            →   종 다양성 지수
  엔트로피 S            →   유전적 다양성

  핵심 관계:
  I = 1/(k_evo × μ)

  골든존 조건:
  0.213 ≤ 1/(k_evo × μ) ≤ 0.500
  → 2.0/k_evo ≤ μ ≤ 4.7/k_evo
```

## 핵심 관찰

### 1. 인간의 돌연변이율은 골든존에 위치
인간의 돌연변이율(~10-15 × 10⁻⁹ /bp/gen)은 I ≈ 0.40으로 추정되며, 이는 골든존 내부이면서 1/e(0.368)에 가깝다. 자연선택이 돌연변이율 자체를 골든존으로 최적화한 것일 수 있다.

### 2. RNA 바이러스 = 혼돈 영역의 생존자
RNA 바이러스의 극단적 돌연변이율(I < 0.1)은 혼돈 영역에 해당한다. 이들이 생존하는 이유는 빠른 세대 교체와 높은 P(quasispecies 다양성)로 G를 유지하기 때문이다.

### 3. 돌연변이율의 진화 (μ의 메타-진화)
돌연변이율 자체가 진화한다(mutator/antimutator). 이는 I가 골든존으로 수렴하는 "메타 부동점 1/3" 현상의 생물학적 구현이다 — f(I) = 0.7I + 0.1의 축소 사상.

### 4. Eigen의 오류 재앙 ↔ 골든존 하한
Eigen의 오류 재앙(error catastrophe) 임계값은 돌연변이율이 너무 높아 유전 정보가 붕괴하는 점이다. 이는 우리 모델에서 I = 0.213(골든존 하한) 이탈에 대응한다.

## 볼츠만 분포 에너지 그래프

```
  P(E) 확률
     │
  1.0│■
     │ ■     T_evo 낮음 (μ 낮음, I 높음)
  0.8│  ■    → 좁은 분포 = 보수적 진화
     │   ■
  0.6│    ■■
     │      ■■■           ★ 골든존 T_evo (μ 최적)
  0.4│         ■■■■       → 적절한 분포 = 최적 진화
     │             ■■■■
  0.2│                 ■■■■■■■■■■    T_evo 높음 (μ 높음, I 낮음)
     │                            ■■■ → 넓은 분포 = 혼돈적 진화
  0.0└──┬──┬──┬──┬──┬──┬──┬──┬──→ E (적응 비용)
       0   1  2  3  4  5  6  7  8

  P(E) = exp(-E / T_evo) / Z
  I = 1/(k_evo × T_evo)
```

## 한계

- 돌연변이율 측정의 불확실성이 크며 종 내 변이도 존재
- k_evo 상수의 이론적 근거 부족 (사후 정규화)
- 유효 돌연변이율(중립 돌연변이 제외)은 총 돌연변이율과 다름
- 볼츠만 분포는 평형 상태를 가정하지만, 진화는 비평형 과정
- 게놈 크기에 따라 μ × genome_size가 더 적절한 비교 단위일 수 있음
