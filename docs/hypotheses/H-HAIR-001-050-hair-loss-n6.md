## PART B: Hair Loss Types and Mechanisms (H-HAIR-021 to 030)

### Androgenetic Alopecia (AGA) — Male/Female Pattern

```
  MECHANISM:
    Testosterone → 5α-reductase → DHT (dihydrotestosterone)
    DHT binds androgen receptors in follicle dermal papilla
    → Follicle miniaturization (terminal → vellus)
    → Shortened anagen (growth phase)
    → Progressive thinning → baldness

  KEY ENZYME: 5α-reductase
    Type I: skin, liver
    Type II: hair follicles, prostate (main target for finasteride)
    2 types = phi(6)

  GENETICS:
    AR gene on X chromosome (maternal inheritance)
    20p11 locus (paternal)
    Multiple loci: ~200+ SNPs identified (GWAS)
    Polygenetic — no single "baldness gene"

  NORWOOD SCALE (male): 7 stages (I-VII)
  LUDWIG SCALE (female): 3 stages (I-III)
```

### Alopecia Areata (AA) — Autoimmune

```
  MECHANISM:
    T-cell mediated autoimmune attack on follicle bulb
    Collapse of "immune privilege" of the hair follicle
    NKG2D+ CD8+ T cells infiltrate anagen bulb
    → Abrupt catagen entry → hair loss in patches

  FORMS:
    Alopecia areata (patches)
    Alopecia totalis (entire scalp)
    Alopecia universalis (entire body)
    = 3 forms = divisor of 6
```

### For Female AGA

```
  1. Minoxidil 5% topical (FDA approved for women at 2%)
  2. Spironolactone 100-200mg (off-label, first-line in practice)
  3. Oral minoxidil 0.625-2.5mg (emerging evidence)
  4. Microneedling (same as male protocol)

  6. Iron + vitamin D supplementation (check levels first)
```

### For Alopecia Areata

```
  1. Topical corticosteroids (first line, limited patches)
  2. Intralesional steroid injections (moderate patches)
  3. JAK inhibitors — baricitinib (FDA 2022) or ritlecitinib (FDA 2023)
     → 30-40% achieve SALT50 (50% regrowth) at 36 weeks
  4. Contact immunotherapy (DPCP/SADBE) for extensive AA
  5. Oral mini-pulse steroids (severe, short-term bridge)
  6. Immunomodulators (methotrexate, azathioprine) if JAK unavailable
```

### For Telogen Effluvium

```
  1. Identify and remove trigger (stress, illness, medication, diet)
  2. Nutritional optimization (iron, zinc, biotin, vitamin D)
  3. Time — self-resolving in 6-12 months typically
  4. Minoxidil 5% if chronic (>6 months)
  5. Stress management (meditation, exercise)
  6. Rule out thyroid/hormonal causes (blood work)
```
