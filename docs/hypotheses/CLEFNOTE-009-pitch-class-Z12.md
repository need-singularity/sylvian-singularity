# CLEFNOTE-009: 음고류 군 Z₁₂ = Z_σ(6)

> **가설**: 12평균율의 음고류(pitch class)는 순환군 Z₁₂ = Z_σ(6)를 형성하며, 이 군의 생성원 수 φ(12) = τ(6) = 4이다.

## 수치 검증

| 속성 | 값 | P₁ 연결 |
|------|------|---------|
| 원소 수 | 12 | σ(6) |
| 생성원 수 φ(12) | 4 | τ(6) |
| 생성원 | {1,5,7,11} | |
| 부분군 수 | 6 | P₁ |
| 약수 | {1,2,3,4,6,12} | div(12)=div(σ(6)) |

```
Z₁₂의 부분군 격자:

  Z₁₂ (σ(6) = 12)
  ├── Z₆  (P₁ = 6)     온음 음계
  ├── Z₄  (τ(6) = 4)    감7화음
  ├── Z₃  (P₁/2 = 3)    증3화음
  ├── Z₂  (φ(6) = 2)    트라이톤
  └── Z₁  (1)           유니슨

  부분군 6개 = P₁!
  τ(12) = 6 = P₁ — 12의 약수 개수 자체가 P₁!
```
