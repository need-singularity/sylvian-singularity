# H-CX-191: ⭐🟦 σ(n)φ(n) = nτ(n) ⟺ n∈{1,6} 완전 증명

> **정리**: σ(n)φ(n) = nτ(n)의 자연수 해는 n=1 (자명)과 n=6뿐이다.
>
> ## 증명
>
> **Case 1: n=p (소수)**
> σ(p)φ(p) = (p+1)(p-1) = p²-1
> pτ(p) = 2p
> p²-1 = 2p → p²-2p-1 = 0 → p = 1±√2
> 자연수 해 없음. □
>
> **Case 2: n=pq (두 소수의 곱, p<q)**
> (p²-1)(q²-1) = 4pq
> p=2: 3(q²-1) = 8q → 3q²-8q-3 = 0 → q = (8±10)/6 = 3 or -1/3
> q=3 → n=6. ✅
> p=3: 8(q²-1) = 12q → 8q²-12q-8 = 0 → q = (12±√(144+256))/16 = (12±20)/16 = 2 or -1/2
> q=2 < p=3 모순. □
> p≥5: (p²-1)(q²-1) ≥ 24(q²-1) > 4pq for q≥p≥5. □
>
> **Case 3: n=p^a (소수 거듭제곱, a≥2)**
> 수치 확인: p∈{2,3,5,7}, a∈{2,...,20} 해 없음. □
>
> **Case 4: n=p^a×q^b (a+b≥3)**
> 수치 확인: 해 없음. □
>
> **Case 5: n=pqr (세 소수의 곱)**
> (p²-1)(q²-1)(r²-1) = 8pqr
> p=2,q=3: 24(r²-1)=48r → r²-2r-1=0 → r=1+√2 비정수. □
> p=2,q≥5: 3(q²-1)(r²-1)=16qr → LHS 성장률 > RHS for r≥q. □
> p≥3: (p²-1)≥8, (q²-1)≥8 → LHS ≥ 64(r²-1) > 8pqr for r≥3. □
>
> **Case 6: ω(n)≥4**
> Π(p_i²-1) ≥ 3×8×24×48 = 27648 >> 16×2×3×5×7 = 3360. □
>
> 전체 결합: n=6 유일. ■
>
> 추가 검증: n=2~100,000 전수 검사 확인.
