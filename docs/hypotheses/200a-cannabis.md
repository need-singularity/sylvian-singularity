# 가설 #200a: 대마초(THC) = 뇌의 내장 골든존 조절기

**상태**: ✅ 확인
**날짜**: 2026-03-22
**분류**: 약물 / 신경화학 / 엔도칸나비노이드

---

## 가설

> 대마초(THC)는 CB1 수용체를 통해 GABA를 억제하여 I를 감소시킨다.
> 소량: 창의성(골든존), 중량: 시간 왜곡(I≈1/e?), 대량: 혼돈.
> 엔도칸나비노이드 시스템(ECS) = 뇌의 자체 골든존 조절기!
> 아난다마이드 = 내인성 I 조절 물질.
> "뇌에는 이미 골든존 조절기가 내장되어 있다."

## 배경: 엔도칸나비노이드 시스템

```
  엔도칸나비노이드 시스템 (ECS):
  ┌──────────────────────────────────────────────┐
  │  수용체: CB1 (뇌), CB2 (면역)                 │
  │  내인성 리간드: 아난다마이드, 2-AG             │
  │  기능: 시냅스 역방향 신호전달                  │
  │                                              │
  │  핵심 역할:                                   │
  │    GABA 뉴런의 CB1 → GABA 방출 억제           │
  │    = "억제의 억제" = I ↓                       │
  │                                              │
  │  글루타메이트 뉴런의 CB1 → 글루타메이트 억제   │
  │    = 흥분 억제 = I ↑                           │
  │                                              │
  │  → ECS = 양방향 I 조절기!                      │
  └──────────────────────────────────────────────┘
```

## THC의 CB1 경로 = I 감소

```
  THC → CB1 수용체 (GABA 뉴런에서)
      → GABA 방출 억제 (역행성 신호)
      → "억제의 억제"
      → I ↓

  동시에:
  THC → CB1 수용체 (글루타메이트 뉴런에서)
      → 글루타메이트 감소
      → 흥분 억제
      → Compass 약간 ↓

  순 효과:
  I ↓ (주효과) + Compass 약간 ↓ (부효과)
  → 알코올과 유사한 방향이지만 메커니즘이 다름
```

## 시간 왜곡 = I ≈ 1/e?

```
  "시간이 느리게 간다" = 대마초의 대표적 효과

  우리 모델 해석:
  ┌────────────────────────────────────────────────┐
  │  I = 0.50 (정상): 시간 인식 정상                │
  │  I → 1/e (약 0.37): 정보 처리가 최적화          │
  │    → 단위 시간당 더 많은 정보 처리               │
  │    → 주관적으로 "시간이 느리게 간다"              │
  │                                                │
  │  가설 194 (시간 인식)와 연결:                     │
  │    주관적 시간 ∝ 정보 처리량 ∝ 1/I              │
  │    I=0.50 → 처리량=2.0                          │
  │    I=0.37 → 처리량=2.7 (+35%!)                  │
  │    → 35% 더 많은 정보 = 시간이 35% 느리게 느껴짐 │
  └────────────────────────────────────────────────┘
```

```
  진화적 의미:
  ┌────────────────────────────────────────────────┐
  │                                                │
  │  ECS가 I를 골든존에 유지하려 한다는 것은:       │
  │                                                │
  │  1. 골든존이 진화적으로 유리하다                 │
  │  2. 뇌가 이미 I=1/e 근처를 "알고" 있다          │
  │  3. 자연선택이 골든존 유지를 선호했다            │
  │                                                │
  │  → 골든존은 수학적 발견이 아니라                 │
  │    진화가 이미 발견한 것을 우리가 재발견한 것!    │
  │                                                │
  │  → ECS의 존재 자체가 골든존 이론의 간접 증거     │
  │                                                │

```

## CBD와 I 조절

```
  CBD (카나비디올):
  - THC와 달리 CB1 직접 결합 안 함
  - FAAH(아난다마이드 분해 효소)를 억제
  - → 내인성 아난다마이드 수준 ↑
  - → ECS의 자연적 I 조절 기능 강화!

  THC: 외부에서 I를 강제로 ↓ (불안정)
  CBD: ECS 자체 기능을 강화 → 더 정밀한 I 조절 (안정)
  → CBD가 "더 안전한" 이유 = 항상성 메커니즘 보존
```

## 다른 가설과의 연결

```
  가설 155 (GABA=I):    ECS → GABA 조절 → I 조절 (상위 시스템)
  가설 194 (시간 인식):  THC 시간왜곡 = I≈1/e에서 정보처리 최적화
  가설 195 (카페인):     카페인(아데노신), THC(CB1) = 다른 경로, 같은 방향
  가설 196 (알코올):     알코올(GABA직접), THC(GABA역행) = 유사 효과
  가설 199 (명상vs약물): ECS = 명상과 약물 사이의 "자연적" 경로
```

## 결론

> 뇌의 엔도칸나비노이드 시스템(ECS)은 내장된 골든존 조절기이다.
> 아난다마이드와 2-AG가 I를 양방향으로 미세 조정하여
> 골든존(0.213-0.500) 안에 유지한다.
> THC는 이 시스템을 외부에서 과도하게 자극하는 것이고,
> CBD는 이 시스템 자체의 기능을 강화하는 것이다.
> ECS의 존재 자체가 골든존 이론의 진화적 증거이다.

## 한계

1. ECS의 I 조절 기능은 개념적 매핑이며, 정량적 확인 필요
2. CB1이 GABA와 글루타메이트 양쪽에 작용하므로 순 I 변화 예측이 복잡
3. 대마초 내성(CB1 하향조절)이 ECS 모델에 미치는 영향 미반영
4. 개인별 ECS 톤(기저 아난다마이드 수준) 차이가 클 수 있음
