## Background: The Endocannabinoid System

```
  Endocannabinoid System (ECS):
  ┌──────────────────────────────────────────────┐
  │  Receptors: CB1 (brain), CB2 (immune)        │
  │  Endogenous ligands: anandamide, 2-AG         │
  │  Function: retrograde synaptic signaling      │
  │                                              │
  │  Key role:                                   │
  │    CB1 on GABA neurons → inhibit GABA release │
  │    = "inhibition of inhibition" = I ↓         │
  │                                              │
  │  CB1 on glutamate neurons → inhibit glutamate │
  │    = excitation inhibition = I ↑              │
  │                                              │
  │  → ECS = bidirectional I regulator!           │
  └──────────────────────────────────────────────┘
```

## THC CB1 Pathway = I Reduction

```
  THC → CB1 receptor (on GABA neurons)
      → GABA release inhibited (retrograde signal)
      → "inhibition of inhibition"
      → I ↓

  Simultaneously:
  THC → CB1 receptor (on glutamate neurons)
      → glutamate reduced
      → excitation inhibited
      → Compass slightly ↓

  Net effect:
  I ↓ (main effect) + Compass slightly ↓ (side effect)
  → Similar direction to alcohol but different mechanism
```

## CBD and I Regulation

```
  CBD (cannabidiol):
  - Unlike THC, does not directly bind CB1
  - Inhibits FAAH (anandamide-degrading enzyme)
  - → Raises endogenous anandamide levels
  - → Enhances ECS's natural I regulation!

  THC: forces I ↓ from outside (unstable)
  CBD: strengthens ECS's own function → more precise I regulation (stable)
  → CBD is "safer" because it preserves the homeostasis mechanism
```

## Limitations

1. ECS's I regulation function is a conceptual mapping; quantitative confirmation needed
2. Net I change is complex since CB1 acts on both GABA and glutamate
3. Cannabis tolerance (CB1 downregulation) not reflected in ECS model
4. Individual ECS tone (baseline anandamide level) differences may be large
