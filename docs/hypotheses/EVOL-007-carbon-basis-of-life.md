# EVOL-007: 탄소 기반 생명 — 원소번호 P₁

> **가설**: 알려진 모든 생명체가 원자번호 P₁ = 6인 탄소를 기반으로 한다는 것은 P₁이 화학적 복잡성의 최소 필요조건을 정의하기 때문이다.

## 왜 탄소인가?

```
원소별 결합 능력:

  Z=1 (H):  1결합    불충분
  Z=2 (He): 0결합    비활성
  Z=3 (Li): 1결합    금속
  Z=4 (Be): 2결합    독성
  Z=5 (B):  3결합    사슬 불안정
  Z=6 (C):  4결합    ← τ(6) = 4 = 최적!
  Z=7 (N):  3결합    사슬 불안정  
  Z=14(Si): 4결합    Si-Si 약함

  C의 τ(6)=4 결합:
  ✓ 긴 사슬 형성 (고분자)
  ✓ 분지 구조 (복잡성)
  ✓ 이중/삼중 결합 (다양성)
  ✓ 안정한 C-C 결합 (346 kJ/mol)
```

## 등급

🟩 EXACT — 탄소 기반 생명은 사실, Z=6=P₁
