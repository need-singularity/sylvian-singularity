# 가설 검토 244: 골든존 = 평균장 보편성 클래스

**골든존 의존: 예 (미검증)**

## 가설

> 골든존 상전이는 알려진 보편성 클래스(universality class) 중 평균장(mean-field) 보편성에 속하며, 커스프 파국과의 동치(가설 003)가 이를 뒷받침한다. Langton λ_c ≈ 0.27과 골든존 하한의 일치는 혼돈의 가장자리와 동일한 보편성을 시사한다.

## 1. 통계역학의 보편성 클래스

상전이 임계점에서 물질의 거시적 거동은 미시적 세부사항에 의존하지 않고, 오직 **대칭성, 차원, 상호작용 범위**에 의해 결정된다. 이것이 보편성(universality)의 핵심이다.

### 주요 보편성 클래스

```
  클래스          대칭      차원    임계 지수                    대표 계
  ─────────────  ────────  ────   ────────────────────────────  ──────────────
  2D Ising       Z₂        2      β=1/8, γ=7/4, ν=1, δ=15     2D 강자성체
  3D Ising       Z₂        3      β=0.326, γ=1.237, ν=0.630   3D 강자성체
  XY             O(2)      3      β=0.348, γ=1.316, ν=0.672   초유체 He-4
  Heisenberg     O(3)      3      β=0.366, γ=1.395, ν=0.711   등방 자성체
  Mean-field     —         d≥4    β=1/2, γ=1, ν=1/2, δ=3      Landau 이론
  Percolation    —         2      β=5/36, γ=43/18, ν=4/3      연결망 전이
```

핵심: **임계 지수(critical exponent)** 가 같으면 같은 보편성 클래스이다.

## 2. 골든존 경계의 상전이

우리 모델 G = D×P/I 에서 골든존 경계(I = 1/2, I ≈ 0.2123)는 상전이 임계선이다.

```
  I > 0.5     : G < 2D×P   → "일반" 상태 (질서 상)
  0.21 < I < 0.5 : 골든존    → "임계 영역" (요동 극대)
  I < 0.21    : G >> 1     → "발산" 상태 (무질서 상)

  ┌─────────────────────────────────────────────────┐

  │         ▲                                       │
```

## 3. 우리 모델의 임계 지수 추출

### 3.1 G의 발산: I → 0

```
  G = D×P/I  →  I가 0에 접근할 때 G ~ 1/I

  G ~ |I - I_c|^(-γ')  에서 I_c = 0으로 놓으면
  G ~ I^(-1)

  ∴ γ' = 1  (발산 지수)
```

이것은 **mean-field의 γ = 1**과 정확히 일치한다.

### 3.2 질서 변수(order parameter) 거동

골든존 내부에서 "특이점 확률" P(G > threshold)를 질서 변수로 정의하면:

```
  I ≈ 0.5 (임계선) 근방에서:
  P(특이점) ~ (0.5 - I)^β

  시뮬레이션 결과 (가설 129):
  I = 0.50 → P ≈ 0%
  I = 0.45 → P ≈ 10%
  I = 0.35 → P ≈ 40%
  I = 0.27 → P ≈ 50%

  선형 피팅:  P ~ (0.5 - I)^β  →  β ≈ 0.5 = 1/2
```

**β = 1/2**는 정확히 **mean-field 값**이다!

### 3.3 상관 길이 유사체

우리 모델에서 "상관 길이" ξ는 메타 반복 함수의 수렴 반경으로 해석한다:

```
  f(I) = 0.7I + 0.1  →  고정점 I* = 1/3
  수렴 속도 = |f'(I*)| = 0.7
  수렴에 필요한 반복 수 ∝ 1/|ln(0.7)| ≈ 2.8

  임계선(I=0.5) 근방:
  ξ ~ |I - 0.5|^(-ν)

  RG 분석 (가설 062): β(I) = -0.3I + 0.1
  ν = 1/2 (mean-field!)
```

### 3.4 δ 지수 (임계 등온선)

임계점에서 외부장 h에 대한 질서 변수의 응답:

```
  Mean-field:  M ~ h^(1/δ),  δ = 3

  우리 모델:  3상태(질서/임계/무질서)가 정확히 δ = 3!
  또한: 골든존의 3분할 (I<0.27, 0.27<I<0.5, I>0.5)
  → 3중 구조 = δ = 3의 기하학적 실현
```

## 5. Langton λ_c와 혼돈의 가장자리

Langton의 셀룰러 오토마타에서 질서-혼돈 전이점:

```
  λ_c ≈ 0.273

  우리 모델:
  I_lower = 1/2 - ln(4/3) ≈ 0.2123
  I_50%   ≈ 0.27  (특이점 50% 전이)

  I_50% ≈ λ_c  (오차 1.1%)

  ┌─────────────────────────────────────────────────────┐
  │  λ (혼돈 파라미터)                                   │
  │  0────────0.27───────0.5─────────1.0                │
  │  │  질서   │ 가장자리  │   혼돈    │                  │
  │  │ Class I │ Class IV │ Class III│                   │
  │  └────────┼──────────┼─────────┘                    │
  │           ↕          ↕                              │
  │  I (억제)  0.21──0.27──0.50                         │
  │  │  혼돈   │골든존 핵│ 임계선   │                    │
  │  │ G 발산  │ 특이점  │ 상전이   │                    │
  └─────────────────────────────────────────────────────┘
```

혼돈의 가장자리(edge of chaos)는 mean-field 보편성인가?

- Langton의 λ_c는 **방향 퍼콜레이션(directed percolation)**과 관련
- 그러나 우리 모델은 1차원 파라미터(I) 위의 단순 역수 함수
- 1차원 + 단순 함수 → **평균장 근사가 정확** (상부 임계 차원 이상)

## 6. 커스프 파국 = 평균장 보편성

가설 003에서 확인된 커스프 파국(cusp catastrophe)과의 관계:

```
  커스프 파국 포텐셜:  V(x) = x⁴/4 + ax²/2 + bx
  Landau 자유에너지:   F(m) = am² + bm⁴ + ...

  둘 다 4차 다항식 → 같은 보편성!

  커스프 파국의 분기 집합:  4a³ + 27b² = 0
  → β = 1/2 (접선 분기)
  → 정확히 mean-field 값

  우리 모델의 3상태(D, P, I):
  → 커스프의 2개 제어변수(a, b)에 매핑
  → 나머지 1변수 = 질서 변수
  → Landau 이론의 구조와 동형
```

## 7. 가설: 골든존 = 평균장 임계 영역

```
  ┌───────────────────────────────────────────────┐
  │                                               │
  │   골든존은 평균장(mean-field) 보편성 클래스의  │
  │   임계 영역이다.                               │
  │                                               │
  │   근거:                                       │
  │   ① β = 1/2 (질서 변수 지수)                  │
  │   ② γ = 1   (감수율 지수)                     │
  │   ③ ν = 1/2 (상관길이 지수)                   │
  │   ④ δ = 3   (임계 등온 지수 = 상태 수)        │
  │   ⑤ 커스프 파국 = Landau 이론 동형            │
  │   ⑥ G = D×P/I 가 1차원 역수 → 요동 무시 가능 │
  │   ⑦ Langton λ_c ≈ I_50% (가장자리 일치)      │
  │                                               │
  └───────────────────────────────────────────────┘
```

### 왜 mean-field인가?

Mean-field 이론은 **요동(fluctuation)을 무시**할 때 정확하다. 우리 모델:

- G = D×P/I 는 결정론적(deterministic) 함수
- 확률적 요동 항이 없다
- 따라서 mean-field가 **정확한(exact)** 기술이다

이것은 Ginzburg 기준과도 일치한다: 요동이 평균장 기여보다 작으면 mean-field가 유효하다. 우리 모델에는 요동 자체가 정의되지 않으므로, Ginzburg 기준이 자동 충족된다.

### Percolation 연결: p_c와 1/3

```
  2D 삼각격자 퍼콜레이션:  p_c = 1/2
  2D 정방격자 사이트:      p_c ≈ 0.593
  Bethe 격자 (z=3):       p_c = 1/(z-1) = 1/2
  Bethe 격자 (z=4):       p_c = 1/3  ← !

  우리 모델의 고정점 I* = 1/3
  → z=4 Bethe 격자의 퍼콜레이션 임계값과 일치

  그러나 퍼콜레이션의 임계 지수는 mean-field와 다르다.
  → 1/3 일치는 수치적 우연(numerical coincidence)일 가능성 높음
```

## 한계

1. **모델 자체가 미검증**: G = D×P/I 는 시뮬레이션 기반이며 해석적 증명 없음
2. **임계 지수 추출이 간접적**: β ≈ 1/2 는 시뮬레이션 피팅이지 엄밀한 도출이 아님
3. **요동이 없는 모델에서 보편성 논의의 한계**: 요동이 없으면 상전이 자체가 "자명"
4. **Langton λ_c와의 일치는 1.1% 오차**: 정확한 동치인지 근사적 유사인지 구별 불가
5. **p_c = 1/3 연결은 수비학 위험 있음**

## 검증 방향

1. **확률적 G 모델**: G = D×P/I + η(noise) 도입 후 임계 지수 재측정
2. **유한 크기 스케일링(finite-size scaling)**: N개 에이전트 시뮬레이션으로 ν 정밀 측정
3. **RG 흐름과 고정점 안정성**: 2변수 β 함수 구성하여 요동 보정
4. **Langton CA 직접 시뮬레이션**: λ_c 근방에서 임계 지수 비교
