# CLEFNOTE-002: C 음자리표 위치 = sopfr(6) = 5

> **가설**: C 음자리표(가온음자리표)는 5선보의 각 줄에 놓일 수 있어 5 = sopfr(6)개의 위치를 가진다.

## 수치 검증

| 위치 | 줄 | 이름 | 용도 |
|------|-----|------|------|
| 1 | 1번째 줄 | 소프라노 | 역사적 성악 |
| 2 | 2번째 줄 | 메조소프라노 | 역사적 성악 |
| 3 | 3번째 줄 | 알토 | 비올라 (현용) |
| 4 | 4번째 줄 | 테너 | 첼로 고음 (현용) |
| 5 | 5번째 줄 | 바리톤 | 역사적 |

```

  줄5 ─── 𝄡 ─── 바리톤
```
