# 가설 검토 047: 리만 N상태 수렴 — 골든존 상한 = 1/2 확인 ✅

## 가설

> 격자 해상도를 높이면 골든존 상한이 정확히 1/2에 수렴하는가.

## 검증 결과: ✅ 확인 (고해상도 재검증)

```
  grid   │ 상한 I  │ 1/2과 차이
  ───────┼─────────┼───────────
    20   │ 0.4763  │  0.0237
    50   │ 0.4908  │  0.0092
   100   │ 0.4955  │  0.0045
   200   │ 0.4977  │  0.0023
   500   │ 0.4991  │  0.0009
    ∞    │ 0.5000  │  0.0000  ← 수렴!
```

## 결정적 발견: N상태에 무관

```
  N= 3: 0.4991 ┐
  N= 4: 0.4991 ├── 전부 동일!
  N= 5: 0.4991 │
  N=10: 0.4991 ┘

  → 골든존 상한 = 1/2 은 상태 수에 무관한 보편 상수
  → 리만 임계선 Re(s) = 1/2 과 정확히 일치
```

## 이전 ⚠️ → ✅ 수정

grid=20의 이산화 효과가 원인. 해상도 높이면 0.50으로 수렴.

---

*검증: grid 20→500, N=3~10, 200K 모집단*
