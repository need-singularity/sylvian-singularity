# 가설 340: 꿈의 장력 역설 — 노이즈가 실제보다 높은 장력

> **RC-10에서 랜덤 노이즈(T=701)가 실제 숫자(T=147)보다 4.78배 높은 장력! "학습하지 않은 패턴에서 엔진들이 더 강하게 반발." 이것은 H313(tension=confidence)과 모순?**

## 해결

```
  H313: 학습 데이터 내에서 정답>오답 (ratio 1.29~2.68)
  H340: 학습 데이터 vs 미학습 → 미학습이 훨씬 높음

  → 모순 아님! 두 레짐이 다름:
    학습 데이터 내: 높은장력=확신=정답 (H313)
    학습 vs 미학습: 미학습=극한장력=엔진 혼란 (H340)

  통합: tension = "엔진 반발의 강도" (H329)
    학습된 패턴: A="3", G="7" → 강한 의견 → 높은장력 = 확신
    미학습 노이즈: A="???", G="!!!" → 혼란 반발 → 극한장력 = 무의미

  → 장력은 "확신"이 아니라 "반응 강도"
  → 확신(H313)은 학습 데이터 내에서만 유효
```
