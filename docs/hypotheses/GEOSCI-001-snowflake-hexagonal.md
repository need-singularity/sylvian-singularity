# GEOSCI-001: 눈송이 6겹 대칭 = P₁

> **가설**: 눈송이(ice crystal)의 C₆ 대칭은 물 분자 H-결합 각도에서 필연적으로 발생하며, 자연에서 P₁ = 6이 가장 눈에 띄게 나타나는 현상이다.

## 물리적 메커니즘

```
물 분자 → 사면체 배위 → 육방정계 얼음 Ih → C₆ 대칭

  H-O-H 결합각 = 104.5°
  → 사면체 (sp³) 배위
  → 육방정계 (hexagonal) 결정
  → 6겹 회전 대칭

  눈송이 가지: 정확히 6개
  └→ P₁ = 6
```

| 속성 | 값 | P₁ 연결 |
|------|------|---------|
| 대칭 축 | C₆ | P₁ |
| 가지 수 | 6 | P₁ |
| 배위수 (ice Ih) | 4 | τ(6) |
| O-H 수소결합/분자 | 2 | φ(6) |
