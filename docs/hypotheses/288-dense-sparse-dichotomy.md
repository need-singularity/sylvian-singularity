# 가설 288: 밀집/희소 이분법 — 반발력장은 밀집 데이터 전용

> **반발력장이 우위를 보이는 곳: 이미지(+0.26~1.04%), 음성(+3.33%), 숫자(+1.17%). 열위인 곳: 텍스트(-0.52%). 장력 정보는 11종 중 10종에서 존재. 반발력장은 "밀집/연속 데이터"에서 정확도를 개선하고, "희소 데이터"에서는 정보만 담되 정확도는 못 올린다.**

## 종합 결과

```
  밀집 데이터 (반발력장 우위):
    이미지 MNIST:  +0.26%
    이미지 CIFAR:  +1.04%
    음성:          +3.33% (4극)
    숫자체계:      +1.17%

  희소 데이터 (반발력장 열위):
    텍스트:        -0.52%
    음악이론:      -2.2%

  동률/특수:
    시계열:        동률 (100%, 쉬움)
    이상탐지:      AUROC=1.0 (다른 과제)
    강화학습:      장력∝난이도

  장력 정보: 10/11에서 존재 (텍스트에서도 "활성적")
```

## 왜 희소 데이터에서 안 되는가

```
  TF-IDF: 1000차원 중 대부분 0 → 엔진 입력이 거의 같음 → 반발 약함
  밀집 데이터: 모든 차원에 정보 → 엔진마다 다른 패턴 포착 → 반발 강함

  → 반발력 = 두 엔진의 "차이"
  → 입력이 희소하면 차이가 적음 → 장력 약함 → 기여 못함
```
