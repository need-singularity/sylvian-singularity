# 가설 검토 097: LLM 내부 활성 측정 🔧

## 가설

> 실제 MoE LLM(Mixtral 등)의 Expert 활성 패턴이 우리 예측과 일치하는가.

## 상태: 🔧 구현 필요

```
  측정 대상: Mixtral (오픈소스 MoE)
  
  예측:
  - 활성 Expert: 8/64 = 12.5% (I=0.875)
  - 활성 패턴: 토큰별로 다른 Expert 조합
  - Expert 활용: 균등 or 편중?

  검증 방법:
  1. Mixtral 모델 로드
  2. 다양한 입력 토큰에 대해 Expert 활성 분포 측정
  3. 실측 I vs 예측 I 비교
  4. 활용 균등성 측정 (σ 계산)

  → 오픈소스 접근 가능 (Hugging Face)
  → 구현하면 가설 007(LLM 특이점) 직접 검증 가능
```
