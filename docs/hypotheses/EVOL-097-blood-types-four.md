# EVOL-097: ABO Blood Types = tau(6) = 4

> **Hypothesis**: The ABO blood group system has 4 types: A, B, AB, O = tau(6) = 4.

## Numerical Verification

| # | Blood Type | Antigens | Antibodies |
|---|-----------|----------|-----------|
| 1 | A | A | Anti-B |
| 2 | B | B | Anti-A |
| 3 | AB | A and B | None |
| 4 | O | None | Anti-A, Anti-B |

## Structure

```
  ABO compatibility:

        Donate to:
  O  -> A, B, AB, O  (universal donor)
  A  -> A, AB
  B  -> B, AB
  AB -> AB           (universal recipient)

  Types = 4 = tau(6)
```

## Structural Meaning

Blood typing has tau(6) phenotypic categories.
