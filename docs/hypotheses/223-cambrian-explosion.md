# 가설 검토 223: 캄브리아 폭발 = 골든존 최초 진입

## 상태: ⚠️ 유비

## 분류: 진화론

## 가설

> 캄브리아 폭발(~540Ma)은 지구 생태계의 I(환경저항)가 골든존 [0.213, 0.500]에 최초로 안정적으로 진입한 사건이다. 산소 농도 상승, 해양 화학 변화, 에디아카라 멸종이 I를 임계값 0.5 아래로 내리며, I가 골든존 중심(1/e ≈ 0.368)에 접근하면서 생물 다양성이 폭발적으로 증가했다.

## 배경 / 맥락

캄브리아 폭발은 약 5억 4천만 년 전, 약 2000만 년이라는 짧은 기간 동안 현존하는 거의 모든 동물 문(phyla)이 화석 기록에 나타난 사건이다. 이 사건 이전 30억 년간은 단세포 위주의 느린 진화가 진행되었다.

왜 하필 이 시점인가? 여러 가설이 있다:
- 산소 농도 상승 (에디아카라~캄브리아 전환)
- 눈덩이 지구(Snowball Earth) 이후 해빙
- 포식-피식 공진화(군비경쟁) 시작
- Hox 유전자 복제 → 형태학적 다양성

우리 모델에서는: 이 모든 요인이 **I를 골든존으로 밀어넣은** 복합 효과라고 해석한다.

관련 가설: 가설 221(자연선택), 가설 222(멸종 커스프), 가설 151(인플레이션 골든존 진입)

## 수식 매핑

```
  캄브리아 이전:   I > 0.5  →  G = D×P/I 낮음  →  느린 진화
  캄브리아 진입:   I ≈ 0.45 →  G = D×P/I 상승  →  폭발적 다양화
  캄브리아 안정:   I ≈ 0.35 →  G = D×P/I 최적  →  모든 문(phyla) 확립

  비유: 인플레이션(가설 151)
  우주 인플레이션    ↔   캄브리아 폭발
  급격한 팽창        ↔   급격한 종 분화
  I 골든존 진입      ↔   I 골든존 진입
  물리법칙 확립      ↔   체제(body plan) 확립
```

## 핵심 관찰

### 1. 캄브리아 = 생물학적 인플레이션
가설 151(인플레이션 골든존 진입)에서 우주 인플레이션을 I의 골든존 진입으로 해석했다. 캄브리아 폭발도 동일한 구조를 가진다: 긴 잠복기 → 임계점 돌파 → 폭발적 다양화 → 안정화.

### 2. 30억 년의 대기 시간
생명은 38억 년 전에 출현했으나 캄브리아 폭발까지 30억 년이 걸렸다. 이 기간은 I가 서서히 감소하여 골든존 경계(0.5)에 접근하는 시간이었다.

### 3. 산소 = 억제 해제 메커니즘
산소 농도 상승은 에너지 효율을 높여(호기성 대사) 생물이 환경 저항을 극복할 수 있게 했다. 산소 ↑ → 에너지 ↑ → I ↓ → 골든존 진입.

### 4. Hox 유전자 = Plasticity 점프
Hox 유전자의 복제와 다양화는 P(가소성)의 급격한 증가에 대응한다. P↑ + I↓ → G가 급격히 상승.

## 한계

- 산소-I 매핑의 정량적 근거가 부족 (O₂ 농도 → I 변환 공식 미확립)
- 캄브리아 폭발의 정확한 시작/종료 시점과 속도에 대한 학계 논쟁 존재
