# MUSICIRCLE-007: 옥타토닉 컬렉션 = P₁/2 = 3개

> **가설**: 옥타토닉 음계(diminished scale)의 전위 불변 컬렉션은 3 = P₁/2개이다.

## 수치 검증

```
3 옥타토닉 컬렉션:

  Oct₀: C  C# D# E  F# G  A  Bb    (0,1,3,4,6,7,9,10)
  Oct₁: C# D  E  F  G  Ab Bb B     (1,2,4,5,7,8,10,11)
  Oct₂: C  D  Eb F  F# G# A  B     (0,2,3,5,6,8,9,11)

  각 컬렉션: 8음 = 2τ(6)
  컬렉션 수: 3 = P₁/2
  
  왜 3개? 
  12/4 = 3: 감7화음이 12음을 4=τ(6)등분
  → 3=P₁/2개의 이동 위치만 존재
```
