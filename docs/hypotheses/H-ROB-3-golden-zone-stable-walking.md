# H-ROB-3: Golden Zone = Stable Walking Region

## 가설

> 인간 보행의 안정 영역은 Golden Zone [0.2123, 0.5000]과 대응된다.
> Froude number (Fr = v^2/gL)를 기준으로, 자연 보행속도는 GZ 하한 근처에,
> 걷기-달리기 전이는 GZ 상한(= 1/2)에 정확히 위치한다.

## 배경

Froude number는 보행 역학의 무차원 매개변수로, Fr = v^2/(gL)이다.
여기서 v는 보행 속도, g는 중력가속도, L은 다리 길이이다.
생체역학에서 Fr = 0.5는 걷기-달리기 전이의 임계점으로 알려져 있다.

Golden Zone의 상한은 정확히 1/2 = 0.5000이므로,
Fr_crit = GZ_upper는 **정의적으로 정확**하다.
문제는 이것이 의미있는 대응인지, 단순한 수치 일치인지이다.

관련 가설: H-067 (상수 관계), 042 (entropy ln4 jump), 044 (golden-zone-4state)

## 핵심 상수

```
Golden Zone: [0.2123, 0.5000], center = 1/e = 0.3679, width = ln(4/3)
LIPM: w = sqrt(g/h) = 3.3972 rad/s, T_c = 0.2944 s
자연 보행 속도: ~1.2 m/s, Fr = 0.1727
걷기-달리기 전이: ~2.04 m/s, Fr = 0.5000
```

## 해석

1. **Fr = 0.5 = GZ_upper (정확)**: 걷기-달리기 전이가 Golden Zone 상한에
   정확히 위치한다. 그러나 이것은 **부분적으로 정의적**이다. Fr_crit = 0.5는
   중력과 원심력이 같아지는 조건이고, GZ_upper = 1/2는 Riemann 임계선이다.
   물리적 의미는 "보행 양식 전이 = zone 경계"라는 해석이 가능하다.

2. **Fr_natural ~ GZ_lower (근사)**: 자연 보행속도(1.2 m/s)의 Fr = 0.1727은
   GZ_lower = 0.2123에서 약 19% 차이가 난다. 정확한 일치는 아니지만
   같은 규모(order)에 있다.

3. **1-Recovery ~ 1/e (근사)**: 최적 보행에서 능동적으로 공급해야 하는
   에너지 비율(35%)이 1/e = 36.79%에 근접한다. 차이 = 1.79%.
   이것은 암시적이지만, 경험적 데이터에 따라 변동이 크다.

4. **안정성 마진의 Golden Zone 통과**: 속도 0.9~1.7 m/s에서 stability margin이
   Golden Zone을 통과하며, 이 범위가 "편안한 보행" 범위와 겹친다.

## 제한사항

- Fr_crit = 0.5 = GZ_upper는 정의적 일치 요소가 있음
- LIPM은 실제 보행의 근사, 비선형 효과 무시
- 에너지 회복률 65%는 Cavagna et al. 기준이며, 개인차가 큼
- Fr_natural과 GZ_lower의 19% 차이는 "근사"를 넘어설 수 있음
- Duty factor 0.62는 단일 값, 실제로는 속도 의존적

## 검증 방향

1. 실제 보행 데이터(force plate, motion capture)로 Fr-stability 관계 확인
2. 다양한 체형(다리 길이)에서 GZ 대응 범위 검증
3. 계단, 경사면 등 다양한 지형에서의 안정 영역 분석
4. 로봇 보행 제어기에서 Golden Zone 기반 속도 선택 실험
