# NOTESHAPE-007: 높은음자리표(𝄞) 루프 = φ(6) = 2개

> **가설**: 높은음자리표 𝄞의 닫힌 영역(enclosed region) = 2 = φ(6)개이다.

## 구조

```
𝄞의 2개 루프:

       ╭───╮     ← Loop 2 (상부, 작은 원)
       │   │
       ╰─╮ │
         │ │     ← 교차점
    ╭────╯ │
    │      │
    │   ╭──╯     ← Loop 1 (중앙, G4 감싸기)
    │   │
    ╰───╯

  Loop 1: G4(2번째 줄)를 감싸는 큰 루프
  Loop 2: 상부의 작은 장식 루프

  2 루프 = φ(6) = Euler totient of 6
```

## 위상적 의미

𝄞을 평면 그래프로 보면:
- 꼭짓점(교차점): ~2 = φ(6)
- 변(호): ~4 = τ(6)
- 면(루프 + 외부): 2+1 = 3 = P₁/2
- 오일러 공식: V - E + F = 2 - 4 + 3 = 1 (열린 곡선이므로)
