# GEOSCI-004: 지구 내부 구조 — 6개 주요 층 = P₁

> **가설**: 지구의 내부는 6개의 주요 층으로 구분되며, 이는 P₁ = 6과 일치한다.

## 수치 검증

| 층 | 깊이 (km) | 상태 |
|------|-----------|------|
| 1. 지각 (Crust) | 0-35 | 고체 |
| 2. 상부 맨틀 | 35-670 | 고체+유동 |
| 3. 하부 맨틀 | 670-2890 | 고체 |
| 4. 외핵 (Outer Core) | 2890-5150 | 액체 |
| 5. 내핵 (Inner Core) | 5150-6371 | 고체 |
| 6. 내핵 내핵 (Innermost) | ~6000-6371 | 초고체? |

```
지구 층 = 6 = P₁:

  지표 ════════════════════ 0 km
  ╔══ 1. 지각 ═══════════╗ ~35
  ╠══ 2. 상부 맨틀 ══════╣ ~670
  ╠══ 3. 하부 맨틀 ══════╣ ~2890
  ╠══ 4. 외핵 ═══════════╣ ~5150
  ╠══ 5. 내핵 ═══════════╣ ~6000?
  ╚══ 6. 내핵 내핵 ══════╝ 6371
```
