# GEOSCI-003: 석영(Quartz) 육방정계 = P₁

> **가설**: 지각에서 두 번째로 풍부한 광물 석영(SiO₂)이 육방정계(hexagonal crystal system)에 속하며, 단면이 P₁ = 6각형이다.

## 수치 검증

| 속성 | 값 | P₁ 연결 |
|------|------|---------|
| 결정계 | 육방정계 | P₁ |
| 단면 대칭 | C₆ | P₁ |
| 프리즘 면 수 | 6 | P₁ |
| Si 배위수 | 4 | τ(6) |
| O 배위수 | 2 | φ(6) |

```
석영 결정 구조:

 ╱  ╲   6개 프리즘면

│    │   
│    │   Si-O 사면체 (τ(6)=4 배위)
│    │   나선형 배열
╲    ╱
 ╲  ╱
  ╲╱
```
