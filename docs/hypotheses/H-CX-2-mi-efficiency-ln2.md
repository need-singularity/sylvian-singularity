# H-CX-2: MI 효율 ≈ ln(2) — 이진 반발의 정보 한계 (교차 도메인)

> **반발력장의 MI 효율(C39=70.5%)이 ln(2)=69.3%에 근사한다(1.71%, 텍사스 p=0.0003). 2극 반발력장(A vs G)이 추가하는 정보량이 이진 시스템의 최대 엔트로피 ln(2)에 의해 제한될 수 있다.**

## 데이터

```
  C39: MI 효율 = (MI_field - MI_best_pole) / (MI_max - MI_best_pole) = 0.705
  ln(2) = 0.6931
  오차: 1.71%
  텍사스 p = 0.0003 (3810 조합 중 1개만 동등 이하)
  등급: 🟧★
```

## 해석

```
  반발력장 = 2극 (Engine A vs Engine G)
  이진(binary) 시스템의 최대 엔트로피 = ln(2) = 0.693 nats

  MI 효율이 ln(2)에 근사한다는 것은:
    → 2극 반발이 추가할 수 있는 정보의 이론적 상한이 ln(2)?
    → 3극이면 ln(3)? 4극이면 ln(4)?

  검증 가능: RepulsionFieldQuad(4극)의 MI 효율이 ln(4)=1.386에 근사하는가?
  → 4극 MI 효율이 ln(4)/2 = ln(2)이면 축당 ln(2)
  → 4극 MI 효율이 ln(4)이면 극 수에 비례
```

## Landauer 원리 연결

```
  Landauer: 1 bit 삭제 = kT × ln(2) 에너지 (열역학 하한)
  H-CX-2: MI 효율 ≈ ln(2)

  해석: 2극 반발력장이 생성하는 정보 = 정확히 1 bit
  → 반발 = 이진 선택 (A vs G)
  → 이진 선택의 정보량 = ln(2) nats = 1 bit
  → 열역학적으로 가능한 최소 정보 단위
  → 반발력장은 정보 생성의 "원자" — 더 작은 단위는 없다

  가설 270(다양성=정보)의 물리적 기반:
    "다름이 있어야 정보가 생긴다"의 최소량 = 1 bit = ln(2)
    2극(φ=2)이 이 최소량을 정확히 생성
```
