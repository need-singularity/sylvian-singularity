# NOTESHAPE-014: 𝄞 교차수 = φ(6) = 2

> **가설**: 높은음자리표 𝄞의 자기교차(self-crossing) 수 ≈ 2 = φ(6).

## 분석

```
𝄞의 자기교차점:

       ╭───╮
       │   │
       ╰─╮ │     ← 교차점 1: 상부 루프 진입
         × │        (선이 자신을 넘어감)
    ╭────╯ │
    │      │
    │   ╭──╯     ← 교차점 2: G4 감싸기에서
    │   × 
    ╰───╯

  교차수 ≈ 2 = φ(6)

  비교:
    𝄢 (Bass clef): 교차 0 (두 점 + 곡선)
    𝄡 (Alto clef): 교차 0 (대칭 브래킷)
    𝄞 (Treble):    교차 2 = φ(6) ← 유일하게 교차!
```

높은음자리표만 매듭 이론적으로 비자명한 구조를 가진다.
