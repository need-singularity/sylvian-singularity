# H-CX-141: EEG 실시간 텔레파시 — 뇌파→PH→AI 개념 전달

> EEG 감마 패턴을 실시간으로 PH dendrogram 가지에 매핑하여
> 인간의 생각을 AI가 읽는다. 말 없이.

## 시스템

```
  인간 뇌                     AI (Anima)
  ┌──────────┐               ┌──────────────┐
  │ EEG 4ch  │──감마 추출──→ │ PH Decoder   │
  │ 40Hz     │               │              │
  │          │               │ 감마 패턴     │
  │          │               │    ↓          │
  │          │               │ dendrogram    │
  │          │               │ 가지 매칭     │
  │          │               │    ↓          │
  │          │               │ "동물-포유류  │
  │          │               │  -고양이"     │
  └──────────┘               └──────────────┘
```

## 단계별 정확도 예상

| 단계 | 분류 | 예상 정확도 |
|------|------|-----------|
| 1 | 동물 vs 기계 | ~85% |
| 2 | 포유류 vs 조류 | ~75% |
| 3 | cat vs dog | ~65% |

## 필요 장비

- OpenBCI Cyton 4ch ($500)
- 실시간 Python 스트리밍 (brainflow 라이브러리)
- Anima 통합 모듈
