## Setup

On the circle R/Z = [0,1), consider the maps T_2: x ↦ 2x mod 1 and T_3: x ↦ 3x mod 1.
Furstenberg's theorem (1967): the only closed infinite invariant set under both T_2 and T_3
is R/Z itself. This rigidity arises from the **multiplicative independence of 2 and 3**
— the prime factors of 6.

## Result 1: Fixed points of ×6 = sopfr(6) ⭐⭐⭐

The fixed points of T_6 satisfy 6x ≡ x (mod 1), i.e., 5x ≡ 0 (mod 1):

```
Fix(T_6) = {0, 1/5, 2/5, 3/5, 4/5}
|Fix(T_6)| = 5 = sopfr(6)
```

More generally, |Fix(T_n)| = n-1 for any n. For n=6, n-1 = sopfr(6) = 5,
which is a characterization: **sopfr(n) = n-1 ⟺ n=6** (proved in H-NT-2).

## Result 2: Orbit of 1/6 under (×2,×3) semigroup = sopfr(6) ⭐⭐⭐

The forward orbit of 1/n = 1/6 under the semigroup ⟨T_2, T_3⟩:

```
1/6 →(×2) 1/3 →(×2) 2/3 →(×2) 1/3 (cycle)
1/6 →(×3) 1/2 →(×3) 1/2 (fixed)
1/3 →(×3) 0   →     0   (fixed)
2/3 →(×3) 0
1/2 →(×2) 0
```

Complete orbit: **{0, 1/6, 1/3, 1/2, 2/3}** = 5 elements = sopfr(6).

## Verification

```python
from fractions import Fraction

def orbit_x2_x3(start, max_iter=20):
    """Forward orbit of start under semigroup generated by x2, x3 on Q/Z."""
    visited = set()
    queue = [start]
    while queue:
        x = queue.pop(0)
        if x in visited:
            continue
        visited.add(x)
        for m in [2, 3]:
            y = Fraction((m * x.numerator) % x.denominator, x.denominator)
            if y not in visited:
                queue.append(y)
    return visited

orbit = orbit_x2_x3(Fraction(1, 6))
print(f"Orbit of 1/6: {sorted(orbit)}")   # [0, 1/6, 1/3, 1/2, 2/3]
print(f"|orbit| = {len(orbit)} = sopfr(6) = 5")  # True

```
