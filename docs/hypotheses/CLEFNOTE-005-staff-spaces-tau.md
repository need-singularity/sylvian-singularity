# CLEFNOTE-005: 오선 칸 수 = τ(6) = 4

> **가설**: 5선보의 줄 사이 칸(space)은 4 = τ(6)개이다.

## 수치 검증

```
5선보 구조:

  ─── 줄5 ───
      칸4        ← 4번째 칸
  ─── 줄4 ───
      칸3
  ─── 줄3 ───
      칸2
  ─── 줄2 ───
      칸1        ← 1번째 칸
  ─── 줄1 ───

  줄: 5 = sopfr(6)
  칸: 4 = τ(6)
  합: 9 = 5 + 4 = sopfr(6) + τ(6)
```

높은음자리표에서 칸의 음이름: F-A-C-E (FACE!) = 4음 = τ(6)
