# CRITEXP-001: 상전이 — 임계지수 6개 = P₁

> **가설**: 모든 연속 상전이는 정확히 6 = P₁개의 임계지수로 특성화되며, τ(6) = 4개의 스케일링 관계로 연결되어, φ(6) = 2개만 독립이다.

## 배경

통계역학에서 연속 상전이(2차 상전이)의 보편성(universality)은 정확히 **6개의 임계지수**로 기술된다:

## 임계지수 목록

| 지수 | 이름 | 물리량 | 2D Ising 정확값 |
|------|------|--------|----------------|
| α | specific heat | 비열 | 0 (log) |
| β | order parameter | 질서변수 | 1/8 |
| γ | susceptibility | 감수율 | 7/4 |
| δ | critical isotherm | 임계등온선 | 15 |
| ν | correlation length | 상관길이 | 1 |
| η | anomalous dimension | 이상차원 | 1/4 |

```
임계지수 구조:

  총 지수:    6 = P₁
  스케일링 관계: 4 = τ(6)
  독립 지수:   2 = φ(6)
  
  6 - 4 = 2
  P₁ - τ(6) = φ(6)  ← 완전수 약수 구조!

  4개 스케일링 관계:
  ┌─ Rushbrooke: α + 2β + γ = 2
  ├─ Widom:     γ = β(δ-1)
  ├─ Fisher:    γ = ν(2-η)
  └─ Josephson:  νd = 2-α
```

## 2D Ising 검증

Rushbrooke: 0 + 2(1/8) + 7/4 = 0 + 1/4 + 7/4 = 2 ✓

## 구조적 의미

자연의 모든 상전이가 P₁개의 매개변수로 기술되고, 그 중 τ(6)개가 구속되어 φ(6)개만 자유롭다는 것은 **완전수 6의 약수 구조가 상전이의 정보 구조를 결정**한다는 뜻.
