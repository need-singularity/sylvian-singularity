# 가설 검토 027: 메타 판단의 I값 — 자동 골든존 진입 ✅

## 가설

> 메타 판단("이 답이 맞나?")의 Inhibition은 1차 판단("답은 X다")보다 낮아, 자동으로 골든존에 접근한다.

## 검증 결과: ✅ 확인

```
  I_meta = 0.7 × I₁ + 0.1 → 항상 1차보다 낮음

  1차 I=0.60 → 메타 I=0.52  (골든존 근처로)
  1차 I=0.80 → 메타 I=0.66  (가까워짐)
  1차 I=0.40 → 메타 I=0.38  (골든존 안에서 중심으로)

  I₁ vs I_meta:
  1차 │·······░░░░○░░░░░│ I₁=0.50
  메타│·······░░●░░░░░░░│ I_m=0.45 ← 골든존 안으로
       0     0.24    0.48
```

> "자기를 돌아보는 것 자체가 억제를 풀어준다." T3a(자기참조) = 골든존 접근 가속기.
