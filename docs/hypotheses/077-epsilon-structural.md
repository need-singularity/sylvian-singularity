# 가설 검토 077: ε = (1-a)×(1/6) — 구조적 결정 ✅

## 가설

> 호기심 강도 ε=0.05가 모델 파라미터에서 유도 가능한가.

## 검증 결과: ✅ 구조적으로 결정됨

```
  I* = (0.1 - ε) / 0.3 = 1/6 이 되려면:
  ε = 0.1 - 0.3×(1/6) = 0.1 - 0.05 = 0.05

  ε = (1-a) × (1/6) = 0.3 × (1/6) = 1/20

  → 호기심 강도 = (1-수축률) × 블라인드스팟
  → 구조적으로 결정됨 (추측이 아닌 유도)
```
