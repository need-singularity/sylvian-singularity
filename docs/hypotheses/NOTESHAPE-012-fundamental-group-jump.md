# NOTESHAPE-012: 기본군 점프 — π₁ = 0 → π₁ = Z

> **가설**: ●→○ 전환에서 기본군이 자명군 0에서 정수군 Z로 점프한다. 이는 "구멍 주위를 감는 것"이 가능해짐을 의미한다.

```
  기본군 전이:

  ● (D²):  π₁ = 0
    → 모든 닫힌 경로가 점으로 수축 가능
    → "탈출구 없음" = 짧은 음의 에너지 포획

  ○ (Ann): π₁ = Z  
    → 구멍 주위를 n번 감는 경로가 비자명
    → 감김수(winding number) = 정수 n
    → "울림/공명" = 긴 음의 에너지 순환

  물리적 해석:
    ● = 에너지가 갇힌 상태 (trivial topology)
    ○ = 에너지가 순환하는 상태 (Z winding)
    
  감김수 n ∈ Z:
    n=0: 수축 가능 (자명)
    n=1: 한 바퀴 감김 (기본 공명)
    n=2: 두 바퀴 (2차 공명 = φ(6)차)
```
