# EVOL-081: Blood Cell Lineages = P1/2 = 3

> **Hypothesis**: Blood has 3 main cellular components: red blood cells, white blood cells, platelets = P1/2 = 3.

## Numerical Verification

| # | Cell Type | Function | Count/uL |
|---|-----------|----------|---------|
| 1 | RBC (erythrocytes) | O2 transport | 4-6M |
| 2 | WBC (leukocytes) | Immunity | 4-11K |
| 3 | Platelets (thrombocytes) | Clotting | 150-400K |

## Structure

```
  Blood composition:

  Plasma (55%)
  --------

  Platelets (<1%)]

  Cell types = 3 = P1/2
```

## Structural Meaning

Blood contains P1/2 formed element categories.
