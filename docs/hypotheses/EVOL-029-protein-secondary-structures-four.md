# EVOL-029: Protein Secondary Structures = tau(6) = 4

> **Hypothesis**: There are 4 levels of protein structure: primary, secondary, tertiary, quaternary = tau(6) = 4.

## Numerical Verification

| # | Level | Description |
|---|-------|-------------|
| 1 | Primary | Amino acid sequence |
| 2 | Secondary | Alpha-helix, beta-sheet |
| 3 | Tertiary | 3D fold |
| 4 | Quaternary | Multi-subunit assembly |

## Structure

```
  Protein structure hierarchy:

  Sequence -> Helix/Sheet -> 3D Fold -> Complex
  Primary    Secondary     Tertiary   Quaternary
    1            2             3          4

  Levels = 4 = tau(6)
```

## Structural Meaning

Protein organization has tau(6) hierarchical levels.
