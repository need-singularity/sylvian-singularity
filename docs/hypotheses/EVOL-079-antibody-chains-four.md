# EVOL-079: Antibody Polypeptide Chains = tau(6) = 4

> **Hypothesis**: Each antibody (IgG) has 4 polypeptide chains: 2 heavy + 2 light = tau(6) = 4.

## Numerical Verification

| Chain | Count | Size |
|-------|-------|------|
| Heavy (H) | 2 | ~50 kDa |
| Light (L) | 2 | ~25 kDa |
| Total | 4 = tau(6) | |

## Structure

```
  Antibody (IgG) structure:

       Fab          Fab

    VL  VH     VL  VH
    |    |      |    |

         |          |

  Light chains = 2 = phi(6)
  Heavy chains = 2 = phi(6)
  Total = 4 = tau(6)
```

## Structural Meaning

Antibody architecture = phi(6) heavy + phi(6) light = tau(6) chains.
