# CLEFNOTE-022: 음표 머리 형태 = τ(6) = 4

> **가설**: 음표 머리의 기본 형태는 4 = τ(6) 종류이다.

| # | 형태 | 용도 |
|---|------|------|
| 1 | 빈 타원 (○) | 온음표, 2분음표 |
| 2 | 채운 타원 (●) | 4분음표 이하 |
| 3 | 다이아몬드 (◇) | 하모닉스 |
| 4 | X 형태 (×) | 타악기, 음소거 |

```
음표 머리 4형태 = τ(6):

  ○  빈 머리  ─── 긴 음가 (2박 이상)
  ●  채운 머리 ─── 짧은 음가 (1박 이하)
  ◇  다이아몬드 ─── 특수 주법
  ×  X표      ─── 비음고 타악

  빈/채운 이중성 = φ(6) = 2
```
