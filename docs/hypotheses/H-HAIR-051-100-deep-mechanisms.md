### The 5α-Reductase Enzyme

```
  Testosterone + NADPH → DHT + NADP+
  Catalyzed by 5α-reductase (SRD5A)

  5α-reductase structure:
    Transmembrane enzyme
    3 isoforms: SRD5A1, SRD5A2, SRD5A3
    SRD5A2: primary in hair follicles (finasteride target)
    SRD5A1: skin, liver (dutasteride also blocks this)
    SRD5A3: converts polyprenol to dolichol (unrelated to hair)

  Steroid A-ring: 6-membered carbon ring!
    Testosterone has 4 rings: A(6C) + B(6C) + C(6C) + D(5C)
    = 3 six-membered rings + 1 five-membered ring

  5α-reduction: double bond in A-ring (C4=C5) is reduced
    → C4-C5 single bond → planar → non-planar A-ring
    → STRONGER androgen receptor binding
```

### Wnt/β-Catenin Pathway (Master Hair Growth Signal)

```
  The Wnt pathway is THE master regulator of hair follicle cycling.

  Core Wnt pathway components:
    1. Wnt ligands (Wnt3a, Wnt7a, Wnt10b — 3 key ones for hair)
    2. Frizzled receptors (FZD — 10 family members)
    3. LRP5/6 co-receptors (2 types = phi(6))
    4. Dishevelled (DVL — 3 family members)
    5. GSK-3β (destruction complex)
    6. β-catenin (effector)

  → 6 core signaling nodes in the Wnt pathway!

  Wnt ON:
    Wnt → FZD + LRP6 → DVL → inhibits GSK-3β → β-catenin accumulates
    → enters nucleus → activates hair growth genes

  Wnt OFF:
    No Wnt → GSK-3β active → phosphorylates β-catenin → degradation
    → no hair growth gene activation → catagen entry
```
