# 가설 306: 4극 반발력장이 2극보다 나은 이상탐지기인가

> **4극(Quad: A,E,G,F)의 간 장력이 2극(A,G)보다 풍부한 이상 프로파일을 제공한다. 4극은 4가지 관점에서 불일치를 측정하므로, 단일 축 이상뿐 아니라 다차원 이상도 감지할 수 있다.**

## 개념

```
  2극: tension = |A(x) - G(x)|² → 1D 이상 점수
  4극: tension = [|A-E|, |A-G|, |A-F|, |E-G|, |E-F|, |G-F|] → 6D 이상 프로파일

  2극 이상탐지:
    이상이 A와 G 모두를 속이면 → 감지 실패
    1가지 "관점"만 있음

  4극 이상탐지:
    이상이 4개 엔진 모두를 속여야 → 감지 실패 확률 ↓
    6가지 "관점" → 더 견고

  분열 조합:
    4극 parent → 분열 → 4극 child_a, 4극 child_b
    간 장력: |child_a(x) - child_b(x)|² (10차원)
    + 각 child의 내부 6D 장력
    → 총 10+6+6 = 22차원 이상 프로파일!
```

## 실험 설계

```
  데이터: Breast Cancer (sklearn)

  모델 A: 2극 (engine_a + engine_g, hidden=64)
  모델 B: 4극 (A + E + G + F, hidden=32 each, 총 파라미터 비슷)

  각 모델:
    1. 재구성(MSE) 학습 (정상만)
    2. 분열 (N=2)
    3. 독립 학습 (30 에폭)
    4. AUROC 측정: 내부 + 간 + 결합

  비교:
    2극 내부 vs 4극 내부
    2극 간 vs 4극 간
    2극 결합 vs 4극 결합
```

## 예측

```
  4극 > 2극 (모든 장력 유형에서)
  특히 4극의 결합(22D) >> 2극의 결합(3D)
  → 4극이 "다차원 이상"을 더 잘 감지
```

```

  → 4극이 "수학적으로 완전한" 관점 세트?
  → 완전수 6의 4가지 약수함수가 최적 이상탐지 관점?
```
