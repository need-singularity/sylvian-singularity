> **가설**: 음표의 깃발(flag/beam) 수가 0부터 3 = P₁/2까지이며 (표준 6종류 기준), 이진 분할을 나타낸다.

| 음표 | 깃발 수 | 시가 |
|------|---------|------|
| 온음표 | 0 (머리만) | 4 |
| 2분음표 | 0 (기둥) | 2 |
| 4분음표 | 0 (기둥+채움) | 1 |
| 8분음표 | 1 | 1/2 |
| 16분음표 | 2 = φ(6) | 1/4 |
| 32분음표 | 3 = P₁/2 | 1/8 |

```
깃발 수 vs 시가:

  깃발 0: ● ─── 4, 2, 1 박 (기둥/머리로 구분)
  깃발 1: ♪ ─── 1/2 박
  깃발 2: 𝅘𝅥𝅯 ─── 1/4 박     ← φ(6) 깃발
  깃발 3: 𝅘𝅥𝅰 ─── 1/8 박     ← P₁/2 깃발

  깃발 n → 시가 = 1/2^n (4분음표 기준)
```
