# 가설 검토 011: Z_max 수렴값 = 86σ

## 가설

> Z-Score 상한이 약 80σ에 수렴한다.

## 검증 결과: 80σ가 아닌 86σ

```
  G_max = D_max × P_max / I_min = 0.99 × 0.99 / 0.05 = 19.60
  Z_max = (G_max - μ) / σ = (19.60 - 0.306) / 0.224 = 86.3σ

  n별 수렴:
  10K:   84.1σ
  50K:   86.0σ
  200K:  86.5σ
  1M:    86.3σ  ← 수렴

  Z_max
  87│         ●──●
  86│      ●
  85│
  84│   ●
  83│
    └──────────────
      10K  50K  200K  1M
```

## 결론

> Z_max ≈ 86σ. 단순히 $G_{max}/\sigma$로 결정되는 비율. 80σ는 격자 스캔의 이산화 효과.
