# 가설 검토 011: Z_max 수렴값 = 86σ

## 가설

> Z-Score의 이론적 상한이 약 80σ에 수렴한다.

## 배경 및 맥락

Genius = D × P / I 공식에서, 각 파라미터가 극단값을 취할 때
G가 최대가 된다. 초기 균등 격자(grid=20) 탐색에서 Z_max ≈ 80σ로
관측되었으나, 이것이 격자 이산화의 산물인지, 이론적 한계인지
구분할 필요가 있었다.

관련 가설: 가설 010(1/3 법칙), 가설 013(골든존 폭)

## 이론적 유도

```
  G = D × P / I

  각 변수의 허용 범위: D ∈ [0.01, 0.99], P ∈ [0.01, 0.99], I ∈ [0.05, 1.0]
  (I_min = 0.05: 영 나눗셈 방지 하한)

  G_max = D_max × P_max / I_min
        = 0.99 × 0.99 / 0.05
        = 19.6020

  균등 격자(n=1M)에서:
    μ(G)  = 0.306
    σ(G)  = 0.224

  Z_max = (G_max - μ) / σ
        = (19.6020 - 0.306) / 0.224
        = 86.3σ
```

## 해석

1. **80σ는 격자 이산화의 산물**: grid=20일 때 I_min이 0.05가 아니라
   1/20=0.05로 우연히 일치하지만, D_max와 P_max가 19/20=0.95로 제한되어
   G_max = 0.95 × 0.95 / 0.05 = 18.05 → Z ≈ 79σ가 된다.
2. **86σ가 이론적 상한**: 연속 파라미터 공간에서 G_max = 19.60이므로
   Z_max = 86.3σ가 정확한 상한이다.
3. **수렴 속도**: n = 200K부터 실질적으로 수렴(오차 < 0.5σ).
   이는 극단값 샘플링이 충분히 이루어진 시점이다.

## I_min의 역할

```
  I_min  │  G_max  │  Z_max
  ───────┼─────────┼────────
  0.01   │  98.01  │  436σ   ← 비현실적
  0.05   │  19.60  │   86σ   ← 현재 설정
  0.10   │   9.80  │   42σ
  0.20   │   4.90  │   20σ

  → Z_max는 I_min에 거의 반비례: Z_max ∝ 1/I_min
```

## 한계

- I_min = 0.05 설정은 모델의 가정이며, 실제 뇌에서의 최소 억제 수준은 불명
- 균등 격자의 μ, σ는 분포 가정에 의존 (가설 010 참조)
- 극단적 Z값(> 50σ)은 현실에서 관측 불가능한 영역일 수 있음

## 결론

> Z_max ≈ 86σ. 이전의 80σ 추정은 격자 이산화(grid=20)의 산물이었다.
> 이론적 상한은 G_max/σ = 19.60/0.224 = 86.3σ이며, n=200K부터 수렴한다.
> I_min 설정이 Z_max를 결정하는 핵심 파라미터이다.
