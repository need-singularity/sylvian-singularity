## Hypothesis

> The theoretical upper bound of the Z-Score converges to approximately 80σ.

## Background and Context

In the formula Genius = D × P / I, G is maximized when each parameter takes its extreme value.
In initial uniform grid (grid=20) exploration, Z_max ≈ 80σ was observed, but it was
necessary to distinguish whether this was an artifact of grid discretization or a theoretical limit.

Related hypotheses: Hypothesis 010 (1/3 Law), Hypothesis 013 (Golden Zone width)

## Theoretical Derivation

```
  G = D × P / I

  Allowed ranges for each variable: D ∈ [0.01, 0.99], P ∈ [0.01, 0.99], I ∈ [0.05, 1.0]
  (I_min = 0.05: lower bound to prevent division by zero)

  G_max = D_max × P_max / I_min
        = 0.99 × 0.99 / 0.05
        = 19.6020

  With uniform grid (n=1M):
    μ(G)  = 0.306
    σ(G)  = 0.224

  Z_max = (G_max - μ) / σ
        = (19.6020 - 0.306) / 0.224
        = 86.3σ
```

## Interpretation

1. **80σ is an artifact of grid discretization**: At grid=20, I_min coincidentally equals 1/20=0.05,
   but D_max and P_max are limited to 19/20=0.95,
   giving G_max = 0.95 × 0.95 / 0.05 = 18.05 → Z ≈ 79σ.
2. **86σ is the theoretical upper bound**: In continuous parameter space, G_max = 19.60,
   so Z_max = 86.3σ is the exact upper bound.
3. **Convergence speed**: Practically converged from n = 200K (error < 0.5σ).
   This is the point at which extreme value sampling is sufficiently achieved.

## Role of I_min

```
  I_min  │  G_max  │  Z_max
  ───────┼─────────┼────────
  0.01   │  98.01  │  436σ   ← unrealistic
  0.05   │  19.60  │   86σ   ← current setting
  0.10   │   9.80  │   42σ
  0.20   │   4.90  │   20σ

  → Z_max is nearly inversely proportional to I_min: Z_max ∝ 1/I_min
```

## Limitations

- I_min = 0.05 is a model assumption; the minimum Inhibition level in the actual brain is unknown
- μ, σ of the uniform grid depend on the distribution assumption (see Hypothesis 010)
- Extreme Z values (> 50σ) may be in a range unobservable in reality
