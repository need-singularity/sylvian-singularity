# 가설 검토 225: 성선택 = Compass 방향

## 상태: ⚠️ 유비

## 분류: 진화론

## 가설

> 성선택(sexual selection)은 진화의 Compass(나침반)이다. 자연선택이 I를 골든존으로 밀어넣는 "스칼라" 힘이라면, 성선택은 골든존 내에서 특정 방향으로 형질을 유도하는 "벡터" 힘이다. 공작의 꼬리 = 높은 D, 낮은 I. r-선택과 K-선택 전략은 I 값의 스펙트럼에 매핑되며, Compass 상한 5/6이 성선택이 도달할 수 있는 형질 공간의 최대 범위를 결정한다.

## 배경 / 맥락

다윈은 자연선택만으로 공작의 꼬리를 설명할 수 없어 성선택 이론을 제안했다. 성선택은 두 형태로 나뉜다:

1. **동성 간 경쟁** (intrasexual): 수컷끼리의 싸움 → 큰 뿔, 강한 몸
2. **이성 선택** (intersexual): 암컷의 선호 → 화려한 깃털, 노래

우리 모델에서 Compass는 I 공간 내에서의 방향성을 제공한다(가설 037, 059). 성선택은 이 Compass의 생물학적 구현이다:

- **Compass 방향** = 성선택의 방향 (어떤 형질이 선택되는가)
- **Compass 강도** = 성선택의 강도 (얼마나 강하게 편향되는가)
- **Compass 상한 5/6** = 성선택이 만들 수 있는 최대 편향

관련 가설: 가설 037(Compass 상한), 가설 059(Compass 5/6), 가설 221(자연선택)

## 수식 매핑

```
  자연선택 (스칼라)              성선택 (벡터 = Compass)
  ──────────────────           ──────────────────────
  I를 골든존으로 수렴       →   골든존 내에서 방향 제공
  |G| = D×P/I              →   G⃗ = (D×P/I) × ê_compass
  전체 적합도               →   방향적 적합도

  Compass 공식:
  C(θ) = (5/6) × cos(θ - θ_opt)

  θ_opt = 성선택이 향하는 최적 방향
  5/6 = Compass 상한 = 성선택의 최대 영향력
  1/6 = 탐색 여백 (호기심, 돌연변이)

  1/2 + 1/3 + 1/6 = 1
  자연선택(1/2) + 수렴(1/3) + 탐색/성선택의 자유도(1/6) = 완전
```

## 핵심 관찰

### 1. 성선택 = Compass의 생물학적 구현
자연선택이 I를 골든존으로 밀어넣는 "등방적(isotropic)" 힘이라면, 성선택은 골든존 내에서 특정 형질 방향으로 편향시키는 "이방적(anisotropic)" 힘이다. Compass 상한 5/6은 성선택이 전체 적합도의 최대 5/6까지만 방향을 결정할 수 있음을 의미하며, 나머지 1/6은 랜덤 탐색(돌연변이)에 남겨진다.

### 2. 핸디캡 원리 = 높은 D + 낮은 I
Zahavi의 핸디캡 원리(공작의 꼬리, 가젤의 스토팅)는 D를 의도적으로 높이는 전략이다. 높은 D에서도 생존할 수 있다는 것은 낮은 I(높은 적합도)를 증명한다. 즉, 핸디캡 = "나는 골든존에 있다"는 정직한 신호.

### 3. K-선택 종에서 성선택이 더 강한 이유
K-선택 종(I ≈ 0.45-0.55)은 골든존 상한 근처에 위치한다. 이 영역에서는 자연선택만으로 I를 충분히 낮추기 어려우므로, 성선택이 추가적인 방향성(Compass)을 제공하여 I를 골든존 중심으로 유도한다.

### 4. r-선택 종 = 골든존 하한의 서퍼
r-선택 종(I ≈ 0.20-0.25)은 골든존 하한을 오가며 "혼돈의 가장자리"를 서핑한다. 이들에게 성선택은 불필요 — 이미 I가 충분히 낮아 높은 G를 달성하기 때문.

### 5. 1/2 + 1/3 + 1/6 = 1의 진화적 해석
- **1/2** = 자연선택의 기여 (생존 압력, 골든존 상한)
- **1/3** = 메타 진화적 수렴 (돌연변이율의 최적화, 부동점)
- **1/6** = 성선택 + 유전적 드리프트 (방향성 + 확률적 탐색)
- 합 = 1: 완전한 진화적 설명

## 한계

- 성선택의 "방향"을 I 공간의 벡터로 매핑하는 것은 개념적 유비에 그침
- r/K 선택 연속체는 현대 생태학에서 비판받는 단순화된 모델
- 종별 I 추정이 간접적이며 검증 불가능한 경우 다수
- 동성 간 경쟁과 이성 선택의 I 효과가 구별되지 않음
- Compass 5/6과 성선택 강도의 정량적 연결 미확립
