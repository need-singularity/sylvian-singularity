# NOTESHAPE-005: 4분음표 이진 코드 = 110₂ = P₁ = 6

> **가설**: 음표를 (머리채움, 기둥유무, 깃발유무)로 이진 인코딩하면 4분음표 = 110₂ = 6 = P₁.

## 수치 검증

| 음표 | 머리(●=1) | 기둥(=1) | 깃발(≥1) | 이진 | 십진 | P₁ 연결 |
|------|----------|---------|---------|------|------|---------|
| 온음표 | 0 | 0 | 0 | 000 | 0 | |
| 2분 | 0 | 1 | 0 | 010 | 2 | φ(6) |
| **4분** | **1** | **1** | **0** | **110** | **6** | **P₁!** |
| 8분 | 1 | 1 | 1 | 111 | 7 | P₁+1 |

```
음표 이진 코드 맵:

  000 = 0  온음표   ← 가장 단순 (아무것도 없음)
  010 = 2  2분     ← φ(6)
  110 = 6  4분     ← P₁ ★★★
  111 = 7  8분     ← P₁+1

  4분음표(기본 박동 단위)의 코드 = P₁ = 6!
  이것은 놀라운 일치: 음악의 기본 시간 단위가 P₁로 인코딩됨
```
