# CLEFNOTE-003: 전체 음자리표 체계 = P₁ = 6

> **가설**: G, F, C 음자리표를 합치면 Middle C가 놓이는 위치가 총 6 = P₁개이다 (soprano C, mezzo C, alto C, tenor C, treble 아래 보표, bass 위 보표).

## 수치 검증

| # | 음자리표 | Middle C 위치 |
|---|---------|--------------|
| 1 | Soprano C | 1번째 줄 위 |
| 2 | Mezzo C | 2번째 줄 위 |
| 3 | Alto C | 3번째 줄 위 |
| 4 | Tenor C | 4번째 줄 위 |
| 5 | Treble | 아래 첫째 덧줄 |
| 6 | Bass | 위 첫째 덧줄 |

```
Middle C의 6가지 표기 = P₁:

  Bass:     ─── ─── ─── ─── ───  C는 위 덧줄
  Tenor C:  ─── ─── ─── ─C─ ───  4번째 줄
  Alto C:   ─── ─── ─C─ ─── ───  3번째 줄
  Mezzo C:  ─── ─C─ ─── ─── ───  2번째 줄
  Soprano C: ─C─ ─── ─── ─── ───  1번째 줄
  Treble:   ─── ─── ─── ─── ───  C는 아래 덧줄

  총 6가지 = P₁
```
