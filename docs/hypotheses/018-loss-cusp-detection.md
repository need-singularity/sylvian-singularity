# 가설 검토 018: Loss 2차미분 급변 = 커스프 전이 감지 ✅

## 가설

> 학습 중 Loss의 2차미분(d²L/dt²)이 임계값을 초과하면 커스프 전이점으로 감지할 수 있다.

## 검증 결과: ✅ 감지 가능

```
  삽입된 전이점: epoch 35, 70
  감지된 전이점: epoch 34, 35, 69, 70 부근

  Loss 2차미분:
   30│
   35│               ████  ← 전이 1 감지
   40│
   65│
   70│               ████  ← 전이 2 감지
   75│
```

## 결론

> ✅ 2차미분 임계값(2.5σ)으로 전이점 감지 가능. 커스프 모니터 구현의 기반.
