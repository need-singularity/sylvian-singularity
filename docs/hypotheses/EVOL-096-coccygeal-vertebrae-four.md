# EVOL-096: Coccygeal Vertebrae = tau(6) = 4

> **Hypothesis**: The coccyx (tailbone) typically consists of 4 fused vertebrae = tau(6) = 4.

## Numerical Verification

| Vertebra | Status |
|----------|--------|
| Co1 | Largest, sometimes mobile |
| Co2 | Fused |
| Co3 | Fused |
| Co4 | Smallest |

## Structure

```
  Coccyx:

  [Co1]
  [Co2]  fused
  [Co3]
  [Co4]

  Count = 4 = tau(6) (most common)
```

## Structural Meaning

The vestigial tail has tau(6) fused vertebrae.

## Grade

🟧 WEAK -- Most commonly 4, but ranges from 3 to 5

## Limitations
- Significant individual variation (3-5 segments)
