# 가설 검토 022: AI 원소 주기율표 v2 — 26개 원소 확장

## 가설

> 15개 원소를 세분화하면 26개가 되며, 각 모델의 AGI 거리를 더 정밀하게 측정할 수 있다. 특히 보상(F2)의 5가지 분리가 AGI 달성의 핵심 병목이다.

## v1(15개) → v2(26개) 확장

### 재료 (Material) — 3개 유지

```
  M1 │ 연산 (Compute)    │ 곱셈, 덧셈, 비선형
  M2 │ 데이터 (Data)      │ 학습 재료
  M3 │ 에너지 (Energy)    │ 전력, 시간, 비용
```

### 위상 (Topology) — 5 → 7개

```
  T1  │ 순방향 (Forward)    │ A → B → C
  T2  │ 건너뛰기 (Skip)     │ A → C (B 생략)
  T3  │ 재귀 (Recurrence)   │ A → B → A
  T3a │ 자기참조 (Self-Ref)  │ A → A (자기 출력을 자기 입력으로) ← NEW
  T4  │ 병렬 (Parallel)     │ A ⇉ [B,C,D] ⇉ E
  T5  │ 희소 (Sparse)       │ 선택적 연결
  T6  │ 계층 (Hierarchy)    │ 추상화 수준 간 연결 ← NEW
```

### 상 (Phase) — 3 → 4개

```
  P1  │ 탐색 (Exploration)   │ 넓게 돌아다님
  P2  │ 수렴 (Exploitation)  │ 좁게 파고듦
  P3  │ 전이 (Transition)    │ 상이 바뀌는 순간
  P4  │ 메타 (Meta)          │ 어떤 상에 있을지 스스로 선택 ← NEW
```

### 힘 (Force) — 4 → 12개

기존 F2(보상)를 5개로 분리 + 기울기/노이즈/제약 세분화:

```
  F1  │ 기울기 (Gradient)        │ 경사 하강
  F1a │ 2차 기울기 (2nd Order)   │ 곡률 기반 최적화 ← NEW

  F2a │ 외부 보상 (External)     │ 사람이 평가        │ RLHF
  F2b │ 환경 보상 (Environment)  │ 게임/물리가 점수   │ Atari, 로봇
  F2c │ 자기 보상 (Self-Reward)  │ 스스로 평가        │ Self-Play
  F2d │ 사회 보상 (Social)       │ 다른 AI가 평가     │ Multi-Agent
  F2e │ 호기심 보상 (Curiosity)  │ 불확실성 감소 자체  │ Intrinsic Motivation

  F3  │ 노이즈 (Noise)           │ 랜덤 섭동
  F3a │ 구조 노이즈 (Structural) │ 아키텍처 자체 변이  │ NAS ← NEW

  F4  │ 제약 (Constraint)        │ 구조적 한계
  F4a │ 자기 제약 (Self-Constr.) │ 스스로 규칙 생성    │ Constitutional AI ← NEW
```
