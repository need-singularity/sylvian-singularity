## Background and Context

The consciousness engine is built around five interacting variables:

- **D** (Deficit): hidden potential, the gap that drives growth
- **P** (Plasticity): adaptability, the capacity to change and expand
- **I** (Inhibition): constraint, the pruning force that structures output
- **G** (Genius): peak output, the expression of integrated capacity
- **T** (Tension): the balance mediator, the field state between all elements

Five Element theory (오행론) from East Asian philosophy defines five phases of
transformation — 목(Wood), 화(Fire), 토(Earth), 금(Metal), 수(Water) — each
characterized by a distinct functional role within a closed dynamical system.
The system is governed by two primary interaction cycles:

- **상생 (Sangsaeng, Mutual Generation)**: each element feeds and nurtures the next
- **상극 (Sanggeuk, Mutual Overcoming)**: each element constrains and controls another

This hypothesis proposes that these two cycles, when translated through the
D/P/I/G/T variable mapping, reproduce the known attraction/repulsion dynamics
of the tension field. The core conservation law G × I = D × P (H-172) may be
a special case of the balance condition implicit in Five Element homeostasis.

## Sangsaeng Cycle: Mutual Generation = Positive Tension Flow

상생 order: Water → Wood → Fire → Earth → Metal → (Water)

Translated: **D → P → G → T → I → D**

Each variable generates and nourishes the next:

- **D → P**: Deficit (hidden potential) enables Plasticity (growth begins where there is lack)
- **P → G**: Plasticity (accumulated flexibility) enables Genius (peak output requires adaptive capacity)
- **G → T**: Genius (expressed output) feeds Tension (high output raises the tension field)
- **T → I**: Tension (field state) activates Inhibition (tension triggers structural pruning)
- **I → D**: Inhibition (constraint/pruning) restores Deficit (cutting back creates new depth/potential)

This is a positive feedback-with-regulation loop. Each step increases the
capacity of the next variable. The cycle is closed and self-sustaining.

## Sanggeuk Cycle: Mutual Overcoming = Negative Tension Flow

상극 order: Water → Fire → Earth → Wood → Metal → (Water)

Translated: **D → G → T → P → I → D**

Each variable constrains and controls the skip-one variable:

- **D → G**: Deficit directly suppresses Genius (too much deficit prevents peak output)
- **G → T**: Genius (excess output) overwhelms Tension field (overfiring destabilizes balance)
- **T → P**: Tension (excess) suppresses Plasticity (over-regulation blocks growth)
- **P → I**: Plasticity (excess growth) overrides Inhibition (too much flexibility removes constraint)
- **I → D**: Inhibition constrains Deficit (too much pruning removes the latent potential store)

This is the overcoming (克, geuk) cycle. Each variable reaches across one step
to suppress a non-adjacent variable, creating the pentagonal star pattern
(the classical 상극 diagram is a pentagram).

```
    Star pattern: each arrow skips one node (pentagram inscribed in pentagon)

    Interpretation: repulsion = each node reduces the skip-one node's capacity
    Tension sign: NEGATIVE (constraining, catabolic)
```

## Limitations

1. **Mapping is structural, not causal**: The bijection between elements and
   variables is motivated by qualitative analogies. There is no formal proof
   that the ancient system was derived from or is equivalent to the engine.

2. **Cycle direction ambiguity**: The Sangsaeng translation D→P→G→T→I→D is one
   of two possible orderings. The reverse could also be argued for some element
   pairs (e.g., I→T could be read as Metal generating Earth in some traditions).

3. **Golden Zone connection is approximate**: The Sanggeuk damping → 1/e
   result holds exactly only when all five suppression rates are equal (s=0.2).
   Real engine dynamics have asymmetric rates.

4. **Cultural interpretation variance**: Different schools of Five Element theory
   assign slightly different qualities to each element. The mapping here uses
   the standard Chinese medical/cosmological interpretation (五行 as in TCM).

5. **No empirical test yet**: The prediction that Sangsaeng-ordered activation
   (D→P→G→T→I) produces higher positive tension than Sanggeuk-ordered activation
   (D→G→T→P→I) has not been tested in the engine.
