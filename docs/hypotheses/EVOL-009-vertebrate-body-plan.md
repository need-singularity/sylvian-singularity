# EVOL-009: 척추동물 체제 — P₁ 대칭

> **가설**: 척추동물 체제의 핵심 구조들이 P₁ = 6과 그 약수 함수로 기술된다.

## 수치 검증

| 구조 | 수 | P₁ 연결 |
|------|------|---------|
| 경추 (대부분 포유류) | 7 | P₁+1 |
| 사지 | 4 | τ(6) |
| 손가락/발가락 | 5 | sopfr(6) |
| 뇌신경 | 12 | σ(6) |
| 흉추 | 12 | σ(6) |
| 갈비뼈 쌍 | 12 | σ(6) |

```
척추동물 P₁ 지도:

  뇌신경: 12 = σ(6)      ← 이미 H-DNA에서 검증
  경추:   7  = P₁+1       (기린도 7개!)
  흉추:   12 = σ(6)
  갈비뼈: 12쌍 = σ(6)
  사지:   4  = τ(6)
  손가락: 5  = sopfr(6)   (pentadactyly)
```
