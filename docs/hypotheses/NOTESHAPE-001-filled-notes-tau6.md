# NOTESHAPE-001: 채운 음표(●) = τ(6) = 4종

> **가설**: 채운 머리(●)를 사용하는 표준 음표는 4 = τ(6)종이다: 4분, 8분, 16분, 32분음표.

## 수치 검증

| # | 음표 | 머리 | 기둥 | 깃발 |
|---|------|------|------|------|
| 1 | 4분음표 (♩) | ● | | | 0 |
| 2 | 8분음표 (♪) | ● | | | 1 |
| 3 | 16분음표 | ● | | | 2=φ(6) |
| 4 | 32분음표 | ● | | | 3=P₁/2 |

```
채운 음표의 위상:

  ● = D² (디스크)
  π₁(D²) = 0 (단순연결)
  χ(D²) = 1

  ●  ●  ●  ●
  ♩  ♪  𝅘𝅥𝅯  𝅘𝅥𝅰
  1  ½  ¼  ⅛ 박

  4종 = τ(6) = 6의 약수 개수
```
