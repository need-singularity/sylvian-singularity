# NOTESHAPE-003: 총 표준 음표 = P₁ = 6종

> **가설**: 표준 음표 = 채운(τ(6)=4) + 빈(φ(6)=2) = P₁ = 6종.

## 핵심 등식

```
  τ(6) + φ(6) = P₁
  4 + 2 = 6

  채운(●): 32분, 16분, 8분, 4분  = τ(6) = 4종
  빈(○):  2분, 온음표            = φ(6) = 2종
  ────────────────────────────────────────────
  합계:                           = P₁  = 6종

  이것은 완전수의 항등식:
    τ(n) + φ(n) = n 은 n=6에서만 성립하지 않지만,
    음표 체계에서 τ(6)+φ(6)=P₁이 실현됨!
```
