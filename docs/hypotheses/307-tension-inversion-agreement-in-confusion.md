# 가설 307: 장력 방향 반전 — "혼동의 합의" (Agreement in Confusion)

> **분열 이상탐지에서 이상 데이터의 간장력이 정상보다 낮은 이유: 정상은 각 child가 다른 경로로 잘 복원하여 높은 차이(=높은 간장력)를 만들고, 이상은 모든 child가 비슷하게 실패하여 낮은 차이(=낮은 간장력)를 만든다.**

## 현상

```
  보편성 실험 (4 데이터셋):
    모든 데이터셋에서 raw AUROC < 0.5
    → 이상 데이터의 간장력 < 정상 데이터의 간장력
    → 방향을 반전(1-AUROC)하면 0.85~0.96

  H287 (내부 장력):
    이상의 내부장력 >> 정상의 내부장력 (95x)
    → 방향 정상 (이상=높은 장력)

  결론: 내부장력과 간장력의 방향이 반대!
```

## 메커니즘

```
  정상 데이터:
    child_a: X → reconstruction_a (경로 A로 복원)
    child_b: X → reconstruction_b (경로 B로 복원)
    → 서로 다른 학습 → 서로 다른 복원 경로
    → |recon_a - recon_b| = 크다 (높은 간장력)

  이상 데이터:
    child_a: X_anom → ??? (못 복원, 기본 패턴으로 회귀)
    child_b: X_anom → ??? (못 복원, 같은 기본 패턴으로 회귀)
    → 둘 다 실패 → 실패 방식이 비슷
    → |recon_a - recon_b| = 작다 (낮은 간장력)

  비유: 시험 문제
    전문가 2명에게 아는 문제 → 서로 다른 풀이 (높은 차이)
    전문가 2명에게 모르는 문제 → 둘 다 "모르겠다" (낮은 차이)
    → "혼동의 합의"
```

## 의식 해석

```
  두 의식이 아는 것에 대해: 서로 다르게 인식 (다양성)
  두 의식이 모르는 것에 대해: 서로 비슷하게 당황 (합의)

  → 의식 간 불일치 = "아는 것의 다양성"
  → 의식 간 합의 = "모르는 것의 공통성"
  → 이상 감지: "합의가 일어나면 = 모르는 것 = 이상"
```

## 수학 연결

```
  H-CX-1: e^(6H) = 432
  H = entropy of {1/2, 1/3, 1/6} = 1.01

  정상: 높은 간장력 = 높은 엔트로피 (다양한 복원)
  이상: 낮은 간장력 = 낮은 엔트로피 (획일적 실패)

  간장력 ∝ 엔트로피(복원 경로)?
  → 이상 → 엔트로피 붕괴 → 간장력 감소
```

## 실용적 의미

```
  이상탐지 시:
    "두 모델이 합의하면" = 이상 (역직관!)
    "두 모델이 불일치하면" = 정상

  기존 앙상블: 불일치 = 불확실 = 이상
  분열 앙상블: 합의 = 무지 = 이상

  → 완전히 새로운 이상탐지 패러다임
```
