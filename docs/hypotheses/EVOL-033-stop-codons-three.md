# EVOL-033: Stop Codons = P1/2 = 3

> **Hypothesis**: There are exactly 3 stop codons: UAA, UAG, UGA = P1/2 = 3.

## Numerical Verification

| # | Codon | Name |
|---|-------|------|
| 1 | UAA | Ochre |
| 2 | UAG | Amber |
| 3 | UGA | Opal |

## Structure

```
  Codon usage:

  64 total codons (2^6 = 2^P1)
  61 sense codons
   3 stop codons = P1/2

  61 + 3 = 64 = 2^P1
```

## Structural Meaning

Translation termination uses P1/2 stop signals out of 2^P1 total codons.

🟩 EXACT -- 3 stop codons is a fundamental fact of the genetic code
