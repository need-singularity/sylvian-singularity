# FIB-001: Fibonacci-Sigma 항등식 — F₁₂ = 144 = σ(6)²

> **가설**: F_σ(6) = σ(6)². 즉 피보나치 수열의 σ(6)=12번째 항이 정확히 σ(6)의 제곱이다. n≤29에서 이를 만족하는 합성수는 n=6뿐이다.

## 수치 검증

```
Fibonacci 수열:
  F₁=1, F₂=1, F₃=2, F₄=3, F₅=5, F₆=8,
  F₇=13, F₈=21, F₉=34, F₁₀=55, F₁₁=89,
  F₁₂ = 144

  σ(6) = 1+2+3+6 = 12
  F₁₂ = 144 = 12² = σ(6)²  ✓ EXACT!
```

## 유일성 검증 (n ≤ 29)

| n | σ(n) | F_σ(n) | σ(n)² | 일치? |
|---|------|--------|-------|-------|
| 1 | 1 | 1 | 1 | ✓ (자명) |
| 2 | 3 | 2 | 9 | ✗ |
| 3 | 4 | 3 | 16 | ✗ |
| 4 | 7 | 13 | 49 | ✗ |
| 5 | 6 | 8 | 36 | ✗ |
| **6** | **12** | **144** | **144** | **✓ ⭐** |
| 7 | 8 | 21 | 64 | ✗ |
| 8 | 15 | 610 | 225 | ✗ |
| 9 | 13 | 233 | 169 | ✗ |
| 10 | 18 | 2584 | 324 | ✗ |
| 11 | 12 | 144 | 144 | ✓ (σ=12 공유) |
| 12 | 28 | 317811 | 784 | ✗ |
| 28 | 56 | ~10^11 | 3136 | ✗ |

```
F_σ(n) = σ(n)² 해:

  n=1:  자명 (σ=1, F₁=1=1²)
  n=6:  σ=12, F₁₂=144=12²  ← 최소 비자명해 = P₁!
  n=11: σ=12, F₁₂=144=12²  (소수, σ(11)=12와 동일)

  n=6은 합성수 중 유일한 해.
  n=11은 σ(11)=1+11=12=σ(6)이므로 같은 메커니즘.
```

## 왜 n=6에서만?

F_m = m²이 되려면 m=1 또는 m=12만 가능 (Fibonacci 완전제곱수는 F₁=1, F₂=1, F₁₂=144만 제곱수).
따라서 σ(n)=12인 n만 해가 되며, σ(n)=12인 합성수는 n=6뿐.
