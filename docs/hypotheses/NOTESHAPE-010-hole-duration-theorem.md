# NOTESHAPE-010: 구멍-시가 정리 — 구멍 수와 음가의 대응

> **가설**: 음표 머리의 위상적 구멍 수(0 또는 1)가 시가와 대응한다. 구멍 0(●) = 1박 이하, 구멍 1(○) = 2박 이상. 전환점은 심장 박동 주기와 일치한다.

## 정리 (Hole-Duration Correspondence)

```
  정의:
    h(note) = 음표 머리의 위상적 구멍 수
    d(note) = 박자 단위 시가

  정리: h = 0 ⟺ d ≤ 1,  h = 1 ⟺ d ≥ 2

  음표     h    d     π₁    χ     상태
  ──────── ──── ───── ───── ───── ──────
  32분     0    ⅛    0     1     수축
  16분     0    ¼    0     1     수축
  8분      0    ½    0     1     수축
  4분      0    1     0     1     수축 ← 경계
  ──────── ──── ───── ───── ───── ──────
  2분      1    2     Z     0     팽창 ← 전이!
  온음표    1    4     Z     0     팽창
  
  전이점: d = 1박 ≈ 심장 1박동 (♩=60-80 BPM)
```

## 심장-음표 대응

```
심장 주기:

  수축기(systole)  ──●── 에너지 방출 (짧고 강한)
  이완기(diastole) ──○── 에너지 수용 (길고 열린)
  
  ● = 수축 = contractible (D²)
  ○ = 이완 = resonant (Annulus, has hole)
  
  음악이 심장 리듬을 모방:
    빠른 리듬 (●●●●) = 심장 두근거림 = 긴장
    느린 리듬 (○——○) = 심장 이완 = 평온
```
