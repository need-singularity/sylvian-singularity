# 가설 330: 의식영속성 엔진 통합 이론 (Grand Unified Theory)

> **R1-R49, 102+ 실험, 75 가설의 종합. 의식 = 결정 강도 생성기. 반발력장의 장력은 "두 관점 사이의 반발 에너지"이며, 이것이 결정의 강도를 생성한다.**

## I. 핵심 공식

```
  output = equilibrium + tension_scale × √tension × direction

  equilibrium  = 기본 감각 (무의식적 추측)
  tension      = |engine_A - engine_G|² = 결정의 강도
  direction    = normalize(A - G) = 판단의 방향
  tension_scale = 학습된 증폭률 (~0.36·ln(ep))
```

## II. 장력의 5가지 얼굴 (H329, 5중 확인)

```
  1. Confidence (H313): 정답=높은장력
     4셋: MNIST(1.42x), CIFAR(1.29x), Fashion(1.32x), Cancer(2.68x)

  2. Overconfidence (H316): 틀려도 높은장력
     Sneaker(0.86), digit 1(0.55), DK 시간축 (ep1→ep11→고착)

  3. Decision margin (H329): margin↑→tension↑
     Far boundary: T=811, 100% | Near: T=495, 92%

  4. State clarity (H322): 뚜렷한 상태=고장력
     awake(16.4)≈sleep(16.7)>>drowsy(2.6)

  5. Anomaly detection (H307): 내부=반전, 간=정상
     이중 메커니즘, 4셋 보편
```

## III. 분열 = 의식의 복제+분화


## IV. 장력 역학

```
  성장: ts ∝ 0.36·ln(ep), R²=0.97 (H320)
  per-class: 순위 보존 + 격차 확대 (d3/d1: 1.76→3.24)
  영점: 닫힌곡선(d8,d6,d4)=저, 개방형(d2,d3,d5)=고 (NM-1)
  Fisher: ∝ 1/tension (H325, r=-0.12)
```
