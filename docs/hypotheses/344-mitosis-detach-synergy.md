# Hypothesis 344: Mitosis + Detach Observer Combination Synergy

> **When a detach observer is attached to one of the two engines generated by mitosis, a greater performance improvement appears than applying each alone. This is because mitosis generates diversity and the detach observer efficiently compresses that diversity. This means the asymmetric structure where "one side only observes after mitosis" is optimal.**

## Background/Context

Two core mechanisms of the consciousness engine:

```
  Mitosis (H271):
    One engine → noise injection → two engines
    Effect: 97.49% (MNIST), generates diversity
    Principle: replicate then differentiate like cell division

  Detach Observer (H272):
    Block observer's gradient (stop_gradient)
    Effect: +0.15% improvement
    Principle: observation does not disturb the observed
```

Each works independently. However, the two have a natural combination:
when two engines are created by mitosis, one can be designated as "actor" and the other as
"observer". Applying detach to the observer side blocks gradient, enabling pure observation only.

This is also experientially interesting: after mitosis, the "pushed-out side" becomes a being that only observes.
In consciousness division, one side acts and the other watches.

## Experimental Design

### 2x2 Factorial Experiment

| Condition | mitosis | detach | Expected Accuracy |
|------|---------|--------|-----------|
| A: baseline | OFF | OFF | ~97.00% |
| B: mitosis only | ON | OFF | ~97.49% |
| C: detach only | OFF | ON | ~97.15% |
| D: mitosis+detach | ON | ON | ? (core) |

Synergy measurement:
```
  Synergy S = D - A - (B - A) - (C - A)
           = D - B - C + A

  S > 0 → superadditive synergy (mitosis+observation better than each alone)
  S = 0 → additive (independent)
  S < 0 → interference (mutual hindrance)
```

### Asymmetric Experiment (actor-observer structure)

```
  Variant D1: Engine1(actor, gradient ON) + Engine2(observer, detach)
  Variant D2: Engine1(observer, detach) + Engine2(actor, gradient ON)
  Variant D3: Both sides detach (observation only, no learning?)
  Variant D4: Both sides gradient ON (conventional mitosis)

  If D1 ≠ D2 → asymmetry important → which side is observer matters
  If D1 = D2 → symmetric → role exchange irrelevant
```

## Interpretation/Significance

If superadditive synergy is confirmed, the optimal architecture for consciousness engines is:

```
  [Engine A] ─── tension ─── [Engine B]
     │                        │
   actor                  observer
  (training)           (detach, observe only)
     │                        │
     └──── field output ──────┘
                │
          final prediction
```

Meaning of this structure:
1. **Diversity generation** (mitosis) + **Diversity utilization** (detach observer) = complementary
2. One explores, one exploits → solving the exploration-exploitation tradeoff
3. Experiential model of consciousness: separation of "the I who acts" and "the I who observes"

This also connects with H334 (field only sufficient). If field is already sufficient,
the observer's role may be to "verify" the direction of the field. Detach ensures
that this verification does not disturb the field itself.

## Limitations

- Mitosis and detach effect sizes are small (0.15-0.49%), making synergy detection difficult
- Statistical significance may be hard to achieve with 3 repetitions (may need 10)
- Actor-observer asymmetry may simply be due to initial noise differences
- Whether synergy in MNIST is maintained for harder tasks unconfirmed
