# MUSICIRCLE-006: 헥사토닉 순환 = τ(6) = 4개

> **가설**: Neo-Riemannian 이론에서 헥사토닉 순환(hexatonic cycle)은 정확히 4 = τ(6)개이다.

## 수치 검증

```
4 헥사토닉 순환 (Richard Cohn):

  H₀: C — c — Ab — g# — E — e — C      (Northern)
  H₁: Db — db — A — a — F — f — Db      (Eastern)
  H₂: D — d — Bb — bb — F# — f# — D    (Southern)
  H₃: Eb — eb — B — b — G — g — Eb      (Western)

  각 순환: 6음 = P₁ (hex = 6!)
  순환 수: 4 = τ(6)
  총 음: 4 × 6 = 24 = 2σ(6)
```

| 속성 | 값 | P₁ 연결 |
|------|------|---------|
| 순환 수 | 4 | τ(6) |
| 각 순환의 음 수 | 6 | P₁ |
| 총 음 (중복 포함) | 24 | 2σ(6) |
| 각 순환의 화음 수 | 6 | P₁ |
| P, L 교대 횟수 | 6 | P₁ |

τ(6)개의 P₁음 순환 = 완전수의 자기참조적 구조!
