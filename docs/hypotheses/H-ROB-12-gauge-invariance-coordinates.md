# H-ROB-12: Gauge Invariance = Coordinate Independence

## 가설 (Hypothesis)

> 센서 기반 로봇 내비게이션 컨트롤러는 전역 좌표 회전(global gauge transformation)에
> 대해 불변이지만, 국소 센서 회전(local gauge transformation)에 대해서는
> 불변이 깨진다. 이 구조는 H-CX-415의 신경망 가중치 공간 게이지 불변성
> (global error 1.7e-07)과 동일한 SO(2) 대칭이다.

## 배경 (Background)

게이지 불변성(gauge invariance)은 물리학의 근본 원리이다.
좌표계를 전체적으로 회전시켜도 물리 법칙이 변하지 않는 것이 전역 게이지 대칭이고,
각 점에서 독립적으로 회전시키면 깨지는 것이 국소 게이지 대칭이다.

H-CX-415는 신경망 가중치 공간에서 전역 게이지 변환의 오차가 1.7e-07임을 보였다.
이 가설은 로봇 내비게이션에서도 동일한 전역/국소 게이지 구조가 존재함을 검증한다.

관련 가설: H-CX-415 (Global Gauge Invariance)

## 이론적 분석

```
  전역 회전 R(theta)를 모든 객체에 적용:
    장애물: o_i -> R * o_i
    로봇:   p   -> R * p
    방향:   h   -> h + theta
    목표:   g   -> R * g

  센서 판독값은 상대 위치에 의존:
    dist(R*p, R*o in direction d+theta) = dist(p, o in direction d)

  따라서 전역 회전에서 센서 입력은 동일 -> 출력 동일
  (정사각형 경계 클리핑이 없다면 정확히 불변)

  국소 회전 (방향만 변경):
    dist(p, o in direction d+theta) != dist(p, o in direction d)
    -> 센서 입력 변경 -> 행동 변경 -> 불변성 깨짐
```

## 해석 (Interpretation)

전역 게이지 불변성의 **이론적 정확성**은 확인되었다. pi/2, pi, 3pi/2에서
오차가 정확히 0인 것은 센서 기반 컨트롤러가 상대 좌표만 사용하기 때문이다.

비직각 회전(pi/6, pi/4, pi/3)에서 오차가 발생하는 것은 아레나의 정사각형 경계가
SO(2) 대칭을 D4(이면체군)로 제한하기 때문이다. 이것은 **진정한 대칭 파괴**이며,
물리학의 자발적 대칭 파괴(SSB)와 유사한 메커니즘���다.

국소 회전에서 일관된 큰 오차(1.3 평균)는 국소 게이지 불변성이 깨짐을 명확히 보여준다.
Cohen's d = 1.68은 "매우 큰" 효과 크기이다.

H-CX-415와의 구조적 유사성:
- 둘 다 SO(2) 대칭
- 둘 다 전역 변환에 불변, 국소 변환에 비불변
- 차이점: H-CX-415는 연속 대칭이 정확히 보존, 로봇은 경계에 의해 이산 부분군으로 제한
