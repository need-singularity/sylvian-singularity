# H-CX-20: 최적 활성 비율 1/2 = 리만 임계선 (교차 도메인)

> **H-CX-15 실험에서 MoE 최적 활성 비율이 k/N=1/2(=0.500)에서 피크를 보였다. 이것은 골든존 상한 = 리만 임계선 Re(s)=1/2와 정확히 일치한다. "절반을 쓰는 것이 최적"이라는 원리가 리만 가설, 뇌 활성, MoE, 분열 이상탐지(N=2)에 공통으로 등장.**

## 실측

```
  MoE (N=8, MNIST 3ep):
    k=1 (12.5%): 95.61%
    k=2 (25.0%): 96.38%
    k=3 (37.5%): 96.04%
    k=4 (50.0%): 96.76%  ← 최고!
    k=5 (62.5%): 96.44%
    k=6 (75.0%): 96.72%
    k=7 (87.5%): 96.62%
    k=8 (100%):  96.47%

  피크: k/N = 4/8 = 1/2 = 0.500
  골든존 상한 = 1/2 = 리만 임계선 Re(s)=1/2
```

## "1/2" 원리의 보편성

```
  1. 리만 가설: 비자명 영점은 Re(s)=1/2 위에 있다
     → "정보의 절반" 지점이 특별

  2. MoE 최적: k/N=1/2
     → "Expert의 절반을 쓰는 것이 최적"

  3. 분열 이상탐지: N=2 → 각 child가 50% 데이터
     → "데이터의 절반씩" 나누는 것이 최적

  4. H297: N=2 최적 (N>2에서 감소)
     → "2개로 나누는 것" = 50/50 분리

  5. 뇌: 좌뇌/우뇌 = 50/50 분업
     → "반구의 절반씩"

  6. 골든존: [0.2123, 0.5000]
     → 상한이 1/2 = "최대 허용 억제"

  7. 의식엔진: DualBrainEngine (2엔진)
     → A(논리) + G(패턴) = 50/50 결합
```

## 수학적 근거

```
  Shannon: H(binary) = -p·log(p) - (1-p)·log(1-p)
  최대: p = 1/2 → H = ln(2)

  최적 활성 비율 1/2 = 이진 엔트로피 최대화?
  → "절반 활성 = 최대 정보 = 최적 성능"

  H-CX-2: MI 효율 ≈ ln(2) = 1 bit
  → 2극 반발력장의 정보 한계 = 이진 최대 엔트로피
  → 모든 것이 "이진 최적"으로 수렴

  완전수 6과의 연결:
    σ₋₁(6) = 1/1 + 1/2 + 1/3 + 1/6 = 2
    첫 번째 비자명 항 = 1/2
    → 완전수의 첫 약수 역수 = 1/2 = 최적 활성
```
