## 결과

### A. 1/3 법칙 — ❌ 반증

```
  균등 격자 스캔: 33.2% (1/3 근사)
  Beta 분포 모집단: 30.17% (1/3 아님)

  → "1/3 법칙"은 균등 분포에서만 성립.
  → 모집단 분포에 의존하는 값이며 구조적 상수가 아님.
  → 정확한 값은 약 3/10.
```

### B. Z_max 수렴 — 86.26σ

```
  이론적 G_max = 0.99 × 0.99 / 0.05 = 19.60
  Z_max = (19.60 - 0.306) / 0.224 ≈ 86.3σ

  → 80σ가 아닌 86σ로 수렴
  → Z_max = G_max / σ 로 결정되는 단순한 비율
```

### D. 확산 법칙 τ ∝ ΔI² — ❌ 판별 불가

```
  R²(τ vs ΔI²) = 0.784
  R²(τ vs ΔI)  = 0.789

  → 두 모델의 R²가 거의 동일
  → 데이터 포인트 4개로는 판별 불가
  → 추가 데이터 필요
```
