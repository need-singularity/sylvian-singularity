# EVOL-018: Cell Junction Types = P1/2 = 3

> **Hypothesis**: Animal cells have 3 major junction types: tight junctions, anchoring junctions, gap junctions = P1/2 = 3.

## Numerical Verification

| # | Junction | Function |
|---|---------|----------|
| 1 | Tight junctions | Seal between cells |
| 2 | Anchoring junctions | Mechanical attachment |
| 3 | Gap junctions | Communication channels |

## Structure

```
  Epithelial cell junctions:

  ----[Tight]----    seal
  ----[Anchor]---    attach
  ----[Gap]------    communicate

  Types = 3 = P1/2
```

## Structural Meaning

Intercellular communication requires exactly P1/2 junction categories.
