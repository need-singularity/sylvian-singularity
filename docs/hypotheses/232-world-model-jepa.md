# 가설 검토 232: JEPA = Deficit 기반 학습 ⚠️

**분류**: 월드모델/AI
**상태**: ⚠️ 유비

## 가설

> Yann LeCun의 JEPA(Joint Embedding Predictive Architecture)는
> Deficit(결손)을 의도적으로 만들어 학습하는 구조이며,
> 마스킹 비율이 곧 I(Inhibition)에 매핑된다.
> JEPA의 최적 마스킹 비율은 골든존에 위치한다.

## 배경/맥락

JEPA는 Yann LeCun이 제안한 차세대 AI 아키텍처 패러다임이다.
기존 생성 모델(픽셀/토큰 예측)과 달리 추상 표현 공간에서 예측한다.

```
  ┌──────────────────────────────────────────────────────────────┐
  │  기존 접근        vs           JEPA                         │
  ├──────────────────────────────────────────────────────────────┤
  │  픽셀 예측                    추상 표현 예측                 │
  │  모든 세부 복원               핵심 의미만 예측               │
  │  GAN, Diffusion              Joint Embedding               │
  │  과적합 위험                  의미적 일반화                  │
  │  D = 0 (결손 없음)           D > 0 (의도적 결손)            │
  └──────────────────────────────────────────────────────────────┘
```

핵심: JEPA는 입력의 일부를 **마스킹(masking)**하여 의도적으로 결손을 만들고,
나머지로부터 마스킹된 부분의 **추상 표현**을 예측한다.

## 수식 매핑: JEPA → G=DxP/I

```
  JEPA 구성요소              우리 모델        매핑 근거
  ─────────────────         ──────────      ──────────────────
  마스킹 비율 (mask%)    →   Deficit (D)     입력 정보의 의도적 결손
  예측기 학습률           →   Plasticity (P)  표현 공간 적응 속도
  정규화/EMA 강도         →   Inhibition (I)  과적합 억제, 표현 붕괴 방지
  예측 정확도             →   Genius (G)      학습된 표현의 품질
```

### D, P, I 매핑 테이블

```
  ┌─────────────────────┬───────┬─────────────────────────────────┐
  │ JEPA 메커니즘       │ 변수  │ 설명                            │
  ├─────────────────────┼───────┼─────────────────────────────────┤
  │ 마스킹 (Masking)    │ D     │ 입력의 일부를 제거 = 결손 생성  │
  │ 예측기 (Predictor)  │ P     │ 잠재 표현에서 마스크 복원 시도  │
  │ EMA 타겟 인코더     │ I     │ 타겟이 천천히 변함 = 억제       │
  │ VICReg 정규화       │ I     │ 표현 붕괴 방지 = 과해방 억제    │
  │ 학습된 표현 품질    │ G     │ 다운스트림 성능 = 천재성        │
  └─────────────────────┴───────┴─────────────────────────────────┘
```

## V-JEPA vs I-JEPA 비교

```
  ┌────────────┬────────────────────┬────────────────────┐
  │            │ I-JEPA (이미지)    │ V-JEPA (비디오)    │
  ├────────────┼────────────────────┼────────────────────┤
  │ 마스킹     │ 공간 패치 75%     │ 시공간 블록 80-90% │
  │ D 추정     │ 0.75              │ 0.80-0.90          │
  │ 모달리티   │ 2D 이미지         │ 3D 비디오          │
  │ I 추정     │ ≈0.25 (해석 2)    │ ≈0.15 (해석 2)     │
  │ 골든존?    │ 하한 근처          │ 골든존 밖 (아래)   │
  │ 성능       │ ImageNet SOTA      │ 비디오 이해 강력   │
  │ 보존법칙   │ GxI = 0.75xP      │ GxI = 0.85xP      │
  └────────────┴────────────────────┴────────────────────┘

  관찰: 마스킹이 높을수록(D↑) I가 낮아져 골든존 밖으로
  → JEPA도 현재 LLM처럼 골든존 최적화가 안 되어 있다?
```

## 핵심 통찰

1. **JEPA의 마스킹 = Deficit의 인공적 생성** — 뇌의 실비우스열 결여와 구조적 유사
2. **의도적 결손이 학습을 촉진** — 아인슈타인의 뇌에서 결손이 천재성으로
3. **현재 JEPA는 D가 너무 높고 I가 너무 낮음** — 골든존 밖
4. **마스킹 비율 최적화 = 골든존 진입** — mask 30-50%가 최적일 수 있음

## 한계

- JEPA의 mask%와 I의 정확한 매핑 함수가 불명확
- EMA momentum과 I의 관계가 비선형적일 수 있음
- I-JEPA/V-JEPA가 실제 "월드모델"인지는 논쟁 중
- 마스킹 비율 최적화 실험이 아직 골든존 관점에서 수행되지 않음

## 검증 방향

1. I-JEPA에서 마스킹 비율을 30%, 37%(1/e), 50%로 변경하여 성능 비교
2. EMA momentum을 변화시키며 실효 I 값 추정
3. JEPA 학습 dynamics에서 I의 시간 변화 추적
4. 골든 MoE 구조를 JEPA 인코더에 적용
