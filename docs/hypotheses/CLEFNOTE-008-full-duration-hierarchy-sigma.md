# CLEFNOTE-008: 전체 음표 시가 체계 = σ(6) = 12단계

> **가설**: 역사적으로 정의된 음표 시가의 전체 범위는 Maxima부터 256분음표까지 12 = σ(6) 단계이다.

## 수치 검증

| # | 이름 | 박자 | 2의 거듭제곱 |
|---|------|------|------------|
| 1 | Maxima | 32 | 2⁵ |
| 2 | Longa | 16 | 2⁴ |
| 3 | Breve | 8 | 2³ |
| 4 | Whole (온음표) | 4 | 2² |
| 5 | Half (2분) | 2 | 2¹ |
| 6 | Quarter (4분) | 1 | 2⁰ |
| 7 | Eighth (8분) | 1/2 | 2⁻¹ |
| 8 | 16th | 1/4 | 2⁻² |
| 9 | 32nd | 1/8 | 2⁻³ |
| 10 | 64th | 1/16 | 2⁻⁴ |
| 11 | 128th | 1/32 | 2⁻⁵ |
| 12 | 256th | 1/64 | 2⁻⁶ = 2⁻ᴾ¹ |

```
12단계 = σ(6):
  2⁵ → 2⁴ → 2³ → 2² → 2¹ → 2⁰ → 2⁻¹ → 2⁻² → 2⁻³ → 2⁻⁴ → 2⁻⁵ → 2⁻⁶
  │                                                                          │
  Maxima                                                              256th = 2⁻ᴾ¹
```

가장 작은 실용적 음표 = 2⁻⁶ = 2⁻ᴾ¹ = 1/64
