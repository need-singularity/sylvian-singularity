# GEOSCI-002: 주상절리 육각형 단면 = P₁

> **가설**: 주상절리(basalt columns, Giant's Causeway)의 단면이 우세하게 육각형인 것은 열수축 균열의 에너지 최소화가 P₁ = 6각형을 선택하기 때문이다.

## 수치 검증

| 속성 | 값 | P₁ 연결 |
|------|------|---------|
| 우세 단면 | 육각형 | P₁ |
| 최적 분할 | 벌집 정리 (2001 Hales) | 6각형 |
| 평균 변 수 | 5.5~6.2 (실측) | ≈P₁ |

```
열수축 균열 패턴:

  용암 냉각 → 수축 응력 → 균열 네트워크
  
  에너지 최소화 = 최소 둘레/넓이 비
  → 정육각형! (Honeycomb conjecture, Hales 2001)
  
  실측 다각형 분포:
  4각: ~5%
  5각: ~25%
  6각: ~50% ← 우세! = P₁
  7각: ~15%
  8각: ~5%
```
