## Background and Context

The braid group B_n on n strands is generated by n-1 elementary crossings
sigma_1, ..., sigma_{n-1} subject to the braid relations:

    sigma_i sigma_j = sigma_j sigma_i        |i - j| >= 2   (far commutativity)
    sigma_i sigma_{i+1} sigma_i = sigma_{i+1} sigma_i sigma_{i+1}  (Yang-Baxter)

There is a natural surjection B_n -> S_n sending each sigma_i to the
transposition (i, i+1). The kernel is the pure braid group P_n.

S6 is the *only* symmetric group S_n (n != 6) where Out(S_n) is nontrivial.
For all n != 6, Out(S_n) = 1. For n = 6, Out(S6) = Z/2Z.

This is well-established (see e.g. Janusz-Rotman, or the classification by
Holder). The outer automorphism exchanges the conjugacy class of transpositions
with the class of products of three disjoint transpositions (both have 15 elements).

## The Outer Automorphism Structure

The outer automorphism of S6 swaps two conjugacy classes of equal size:

```
  Transpositions (2-cycles):           Triple-transpositions (2+2+2 cycles):
  ────────────────────────             ────────────────────────────────────
  (1 2), (1 3), ..., (5 6)            (1 2)(3 4)(5 6), ...
  Count = C(6,2) = 15                 Count = 6!/(2^3 * 3!) = 15

  Both classes have size 15 — the Garside length!

  Under the outer automorphism phi:
    phi: {transpositions} <---> {triple-transpositions}
    phi: {3-cycles}       <---> {3-cycles of type (abc)(def)}

  This creates "exotic" representations of S6 (and hence of B6)
  that have no analogue for any other B_n.
```

## Garside Element and the Number 15

The Garside element Delta in B_n is the positive lift of the longest
permutation w_0 in S_n. Its word length in generators is:

    len(Delta) = C(n, 2) = n(n-1)/2

For n = 6: C(6,2) = 15.

Note: 15 = sigma(6) + 3 = 12 + 3. While not a clean identity,
15 itself appears as both Garside length and conjugacy class size.
