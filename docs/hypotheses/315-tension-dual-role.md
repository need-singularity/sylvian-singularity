# 가설 315: 장력의 이중 역할 — 확신(항상) + 정규화(저데이터)

> **장력은 항상 confidence를 측정하지만(H313), 데이터가 부족할 때 추가로 regularizer 역할을 한다. H283에서 N=100일 때 +5.5pp 효과는 장력이 과적합을 방지하기 때문이고, N=60000일 때 무효과는 충분한 데이터로 정규화가 불필요하기 때문.**

## 통합 모델

```
  장력의 역할 = f(데이터량):
    저데이터 (N<1000): confidence + regularizer → 정확도 향상
    고데이터 (N>10000): confidence only → 정확도 중립

  왜?
    저데이터: 모델이 과적합 경향 → 장력(2엔진 반발)이 다양성 강제
    → Dropout과 유사한 정규화 효과 → 일반화 향상
    고데이터: 충분한 데이터로 과적합 없음 → 정규화 불필요
    → 장력은 여전히 confidence를 측정하지만, 정확도에 영향 없음

  비유: 의식의 역할
    초보자: 의식적 주의가 필수 (실수 방지 = 정규화)
    전문가: 의식적 주의 선택적 (자동 처리 충분 = confidence만)
```

## 검증 실험

```
  MNIST N = {100, 300, 1000, 3000, 10000, 60000}
  각 N에서:
    1. 장력 O vs 장력 X → delta(정확도) = 정규화 효과
    2. T(정답)/T(오답) ratio = confidence 효과
    3. delta > 0이면 "regularizer", ratio > 1이면 "confidence"

  예측:
    모든 N에서 ratio > 1 (항상 confidence)
    N<1000에서만 delta > 0 (저데이터에서만 regularizer)
```
