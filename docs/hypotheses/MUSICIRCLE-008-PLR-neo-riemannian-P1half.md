# MUSICIRCLE-008: Neo-Riemannian P,L,R = P₁/2 = 3 연산

> **가설**: Neo-Riemannian 이론의 기본 변환 P(Parallel), L(Leading-tone), R(Relative)은 3 = P₁/2개이다.

## 수치 검증

| 변환 | 이름 | 효과 | 반음 이동 |
|------|------|------|----------|
| P | Parallel | 장↔단 (같은 근음) | 3도: 1 반음 |
| L | Leading-tone | 장→단 (장3도 위) | 1 반음 |
| R | Relative | 장↔관계단조 | 2 반음 |

```
P, L, R의 원 위의 작용:

  C major ─P─ C minor
    │              │
    L              R
    │              │
  E minor ─P─ E major
    │              │
    R              L
    │              │
  Ab major ─P─ Ab minor

  3 연산 = P₁/2
  P² = L² = R² = 1 (각각 자기역원 = 대합)
  → 이면군 Dih 구조 생성
```
