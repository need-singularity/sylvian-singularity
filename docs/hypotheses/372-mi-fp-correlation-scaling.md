# 가설 372: MI-FP 상관 N-Scaling — 0.705는 구조적 상수인가

> **H-CX-25의 MI효율=FP상관=0.705 일치가 N=10(Fashion)에서만 관측되었다. N이 커지면(N=100, N=1000) 이 값이 수렴하는가, 아니면 우연인가? ln(2)=0.693과의 1.7% 오차가 N→∞에서 0으로 수렴하면 구조적 상수.**

## 배경/맥락

```
  H-CX-25 핵심:
    C39: MI 효율 = 0.705 (MNIST, N=10 클래스)
    H318: r(tension, knn_acc) = 0.705 (Fashion, N=10 클래스)
    일치: 0.705 = 0.705 (오차 0.0%)

  문제:
    N=10 → Pearson r의 SE = 1/√8 = 0.35
    95% CI: [0.0, 1.4] → 매우 불확실!
    → 더 큰 N에서 검증 필수
```

## 예측

```
  가설 A (구조적): 0.705 ≈ ln(2) = 0.693
    N→∞에서 MI효율 → ln(2) 수렴
    이유: Landauer 원리 — 1bit 정보 처리의 열역학적 한계
    예측: N=100에서 MI효율 ∈ [0.68, 0.72]

  가설 B (우연): 0.705는 N=10의 우연
    N이 달라지면 값도 달라짐
    예측: N=100에서 MI효율 ∉ [0.68, 0.72]
```

## 실험 설계

```
  N을 체계적으로 증가시키며 두 메트릭 동시 측정:

  | 데이터셋        | N    | MI효율 | r(T,knn) | 비고 |
  |-----------------|------|--------|----------|------|
  | MNIST           | 10   | 0.705  | ?        | 기존 |
  | Fashion-MNIST   | 10   | ?      | 0.705    | 기존 |
  | EMNIST-Letters  | 26   | ?      | ?        | 새로 |
  | EMNIST-Balanced | 47   | ?      | ?        | 새로 |
  | CIFAR-100       | 100  | ?      | ?        | 핵심 |
  | ImageNet subset | 1000 | ?      | ?        | 이상적 |

  각 N에서:
    1. RepulsionField 2극 학습
    2. MI 효율 = (MI_field - MI_best_pole) / (MI_max - MI_best_pole)
    3. r(tension, knn_acc) per-class 상관
    4. 두 값 비교 + ln(2)와의 오차
```

## 성공/실패 기준

```
  성공 (구조적 상수):
    N=100에서 MI효율 ∈ [0.68, 0.72]
    AND r(T,knn) ∈ [0.68, 0.72]
    AND 두 값의 차이 < 5%

  부분 성공:
    한쪽만 수렴 → 독립 메커니즘

  실패 (우연):
    N=100에서 두 값 모두 [0.68, 0.72] 밖
    → H-CX-25를 ⬛로 하향
```
