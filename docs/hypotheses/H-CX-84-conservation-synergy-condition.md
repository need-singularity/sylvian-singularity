# H-CX-84: 위상 보존 = 시너지 조건

> ts×H0 곱이 보존되는 데이터셋(Fashion, CIFAR)에서만
> 의미있는 시너지가 발생한다. 보존 = 위상-장력 균형 = 시너지 가능.

## 배경

- H-CX-79: ts×H0 보존 — Fashion CV=0.08, CIFAR CV=0.04 (보존), MNIST CV=0.14 (비보존)
- 통합 시너지: Fashion +17.8%p, CIFAR +5.4%p, MNIST +2.1%p

## 예측

1. CV(ts×H0)와 synergy의 음의 상관 (CV 작을수록 시너지 큼)
2. 보존 임계값 CV < 0.1 이하에서만 시너지 > 3%p
