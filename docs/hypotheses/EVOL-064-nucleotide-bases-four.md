# EVOL-064: DNA/RNA Bases per Molecule = tau(6) = 4

> **Hypothesis**: Both DNA and RNA use exactly 4 different bases each = tau(6) = 4.

## Numerical Verification

| # | DNA Base | RNA Base |
|---|---------|---------|
| 1 | Adenine | Adenine |
| 2 | Guanine | Guanine |
| 3 | Cytosine | Cytosine |
| 4 | Thymine | Uracil |

## Structure

```
  Base alphabet:

  DNA: {A, G, C, T}  = 4 = tau(6)
  RNA: {A, G, C, U}  = 4 = tau(6)

  Shared: 3 = P1/2
  Unique: 1 each (T vs U)
```

## Structural Meaning

The genetic alphabet size = tau(6) in both DNA and RNA.
