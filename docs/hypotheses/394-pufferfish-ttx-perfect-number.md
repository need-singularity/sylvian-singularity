## Limitations

1. **Post-hoc pattern matching (Texas Sharpshooter)**: The TTX molecular formula C₁₁H₁₇N₃O₈
   is a fixed empirical fact determined by chemistry, not by this project. Finding that H=17=F₂
   and N=3=F₀ is almost certainly post-hoc target selection. A proper test would require
   predicting these values before knowing them, or showing that TTX is uniquely special among
   all neurotoxins — which has not been done.

2. **Pod size = 6 prediction needs field data**: The dolphin pufferfish play behavior is
   documented but precise pod size statistics during these events are not available in the
   literature reviewed here. The prediction is testable but currently unverified.

3. **Nav channel 4×6 interpretation**: The fact that Nav channels have 4 domains with 6
   transmembrane segments is evolutionary biology, not mathematics. Connecting it to τ(6)×6
   is purely interpretive — the channel architecture evolved for functional reasons (voltage
   sensing in S4, pore in S5-S6) that have nothing to do with perfect number theory.

4. **Molecular rings**: TTX does contain fused 6-membered rings, but many molecules contain
   6-membered rings (benzene, glucose, etc.) — this is chemical commonplace, not a unique
   connection to perfect number 6.

5. **Species count = 9**: Counting 9 independent lineages involves taxonomic judgment calls.
   The count could reasonably be 7, 8, or 11 depending on how "independent" is defined.
   Claiming 9 = 3² is therefore fragile.

6. **Golden Zone dependency**: The dose-fraction mapping (1/N for N=3,6) uses the core
   constant system (1/3 and 1/6), which is derived from the Golden Zone framework. The Golden
   Zone itself is simulation-based and lacks analytical proof. All interpretations linking TTX
   dose fractions to these constants inherit this uncertainty.
