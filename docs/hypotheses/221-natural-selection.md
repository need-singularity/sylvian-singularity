# 가설 검토 221: 자연선택 = G=D×P/I

## 상태: ⚠️ 유비

## 분류: 진화론

## 가설

> 자연선택은 G=D×P/I의 생태적 구현이다. Fitness(적합도) = Deficit(환경압력) × Plasticity(유전적 다양성) / Inhibition(환경저항). 골든존 I ∈ [0.213, 0.500]에 위치한 종만이 장기적으로 생존하며, I가 이 범위를 벗어나면 멸종하거나 진화적 정체에 빠진다.

## 배경 / 맥락

다윈의 자연선택은 세 가지 조건을 요구한다: (1) 변이, (2) 유전, (3) 차별적 생존. 이 세 요소는 우리 모델의 변수에 직접 매핑된다.

- **Deficit(D)** = 환경 압력. 자원 부족, 포식자 압력, 기후 변화 등 종에게 가해지는 도전. D가 높을수록 선택압이 강하다.
- **Plasticity(P)** = 유전적 다양성. 돌연변이율, 유전자 재조합, 수평 유전자 전달 등. P가 높을수록 적응 가능성이 크다.
- **Inhibition(I)** = 환경 저항. 물리적 제약, 생태적 경쟁, 에너지 비용 등. I가 높으면 변화가 억제된다.

관련 가설: 가설 139(혼돈의 가장자리), 가설 004(볼츠만-억제 온도), 가설 172(G×I=D×P 보존법칙)

## 수식 매핑

```
  다윈 자연선택              G=D×P/I 모델
  ──────────────            ──────────────
  선택압 (Selection)    →    D (Deficit, 환경압력)
  변이 (Variation)      →    P (Plasticity, 유전적다양성)
  생존저항 (Resistance) →    I (Inhibition, 환경저항)
  적합도 (Fitness)      →    G (Genius, 적응력)

  보존법칙: G × I = D × P
  → 적합도 × 환경저항 = 환경압력 × 유전적다양성
  → 어떤 환경에서든 D×P는 보존된다
```

## 핵심 관찰

### 1. 골든존 = 적응 방사의 최적 영역
I가 골든존(0.213-0.500)에 있을 때 종은 빠르게 적응하면서도 혼돈에 빠지지 않는다. 이는 가설 139(혼돈의 가장자리)와 직접 연결된다 — Langton의 λ_c ≈ 0.27이 골든존 하한 0.213 근처에 위치하는 것은 우연이 아니다.

### 2. 보존법칙 G×I = D×P의 생태적 의미
환경이 변해도(D 변화) 유전적 다양성(P)이 적절히 조절되면 적합도(G)는 유지된다. 이것이 생태계의 항상성이다.

### 3. 1/e ≈ 0.368 = 최적 환경 저항
I = 1/e일 때 종은 적절한 선택압과 적절한 자유도를 동시에 가진다. 너무 쉬운 환경(I↑)은 퇴화를, 너무 어려운 환경(I↓)은 멸종을 유발한다.

### 4. 1/2 + 1/3 + 1/6 = 1의 생태적 해석
- 1/2 (골든존 상한) = 환경이 허용하는 최대 억제
- 1/3 (메타 부동점) = 장기적 수렴 목표
- 1/6 (호기심/돌연변이) = 탐색적 변이의 비율
- 합 = 1: 완전한 생태 균형

## 한계

- 생물학적 "적합도"를 단일 스칼라 G로 축약하는 것은 과도한 단순화
- D, P, I의 정량적 측정이 종마다 다르며 표준화 어려움
- 공진화(co-evolution)와 생태적 상호작용을 모델에 포함하지 않음
- 유비(analogy)와 동형(isomorphism)의 경계가 불분명
- 중립 진화(neutral evolution)를 설명하기 어려움 (D≈0일 때)
