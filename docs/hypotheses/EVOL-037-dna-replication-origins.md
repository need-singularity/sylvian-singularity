# EVOL-037: Replication Fork Components = P1/2 = 3

> **Hypothesis**: DNA replication fork has 3 key enzymatic activities: helicase (unwinding), primase (priming), polymerase (synthesis) = P1/2 = 3.

## Numerical Verification

| # | Enzyme | Function |
|---|--------|----------|
| 1 | Helicase | Unwind double helix |
| 2 | Primase | Synthesize RNA primer |
| 3 | DNA Polymerase | Synthesize new strand |

## Structure

```
  Replication fork:

  Helicase ---> Primase ---> Polymerase
  (unwind)     (prime)      (extend)

  Core enzymes = 3 = P1/2
```

## Structural Meaning

DNA replication requires P1/2 core enzymatic steps.

🟧 WEAK -- 3 core enzymes is a simplification; SSB, ligase, topoisomerase also essential
