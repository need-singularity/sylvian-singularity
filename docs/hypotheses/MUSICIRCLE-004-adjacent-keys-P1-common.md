# MUSICIRCLE-004: 인접 조의 공통음 = P₁ = 6

> **가설**: 5도권에서 인접한 두 조는 정확히 P₁ = 6개의 공통음을 공유한다.

## 수치 검증

```
C 장조:  C  D  E  F  G  A  B     (7음)
G 장조:  G  A  B  C  D  E  F#    (7음)
                                  
공통:    C  D  E     G  A  B     = 6개 = P₁!
차이:         F→F#                = 1음

F 장조:  F  G  A  Bb C  D  E     (7음)
C 장조:  C  D  E  F  G  A  B     (7음)

공통:    C  D  E  F  G  A        = 6개 = P₁!
차이:         Bb→B                = 1음
```

| 관계 | 공통음 수 | P₁ 연결 |
|------|----------|---------|
| 인접 (5도) | 6 | P₁ |
| 2단계 | 5 | sopfr(6) |
| 3단계 | 4 | τ(6) |
| 4단계 | 3 | P₁/2 |
| 5단계 | 2 | φ(6) |
| 6단계 (대척) | 2 | φ(6) |

```
공통음 수 vs 5도권 거리:

  거리: 0  1  2  3  4  5  6
  공통: 7  6  5  4  3  2  2
          P₁ sp τ P/2 φ  φ
          
  → 거리 1~5의 공통음 = {6,5,4,3,2} = {P₁,sp,τ,P₁/2,φ}
  → P₁의 산술함수가 역순으로 나타남!
```
