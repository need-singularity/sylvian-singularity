## PART C: Mechanical Tension Theory (H-HAIR-146 to 155)

### The Galea Aponeurotica Hypothesis

```
  The "mechanical tension" theory of AGA (Choy et al. 2022):

  Skull growth → galea aponeurotica (scalp fascia) stretches
  → Mechanical tension on follicles in vertex and frontal areas
  → Reduced blood flow (compression of perforating arteries)
  → Hypoxia → DHT upregulation (compensatory vasodilation attempt)
  → DHT → miniaturization

  Evidence:
    ✓ AGA pattern matches galea tension distribution
    ✓ Areas NOT attached to galea (temporal, occipital) don't bald
    ✓ Scalp exercises / massage reduce tension → some benefit
    ✗ Castrated men (no DHT) don't bald regardless of tension
    ✗ Women bald in different pattern despite same galea
```

## PART D: Aging and Senescence (H-HAIR-156 to 165)

### Follicle Aging Mechanisms

```
  Hair aging involves multiple parallel processes:

  1. STEM CELL EXHAUSTION
     Bulge stem cells decrease with age
     → Fewer hairs per cycle, thinner hairs
     → Eventually: follicle converts to epidermal cells

  2. OXIDATIVE STRESS
     Reactive oxygen species (ROS) damage DP cells
     → Bcl-2 depletion → apoptosis
     → Melanocyte death → graying

  3. TELOMERE SHORTENING
     Stem cell telomeres shorten with each division
     → Replicative senescence → no more follicle cycling
     Telomere repeat = TTAGGG = 6 nt (H-DNA-022!)

  4. HORMONAL SHIFTS
     Age → ↑DHT sensitivity, ↓estrogen (women)
     → Miniaturization acceleration after 40-50

  5. INFLAMMATION (INFLAMMAGING)
     Chronic low-grade inflammation
     → Perifollicular microinflammation
     → Fibrosis → permanent follicle loss

  6. EPIGENETIC DRIFT
     Methylation changes accumulate
     → Gene expression shifts away from growth program
     → Matches bald scalp methylation signature
```
