# CLEFNOTE-023: 덧줄(Ledger Lines) = φ(6) 대칭

> **가설**: 큰보표에서 Middle C는 정확히 1개 덧줄을 사용하며, 위아래 대칭으로 2 = φ(6) 방향으로 확장된다.

```

  ─── 줄5 ───
  ─── 줄4 ───
  ─── 줄3 ───
  ─── 줄2 ───
  ─── 줄1 ───
  ── C4 ──     ← 아래 1번째 덧줄 (Middle C)

  낮은음자리표:
  ── C4 ──     ← 위 1번째 덧줄 (Middle C)
  ─── 줄5 ───
  ─── 줄4 ───
  ─── 줄3 ───
  ─── 줄2 ───
  ─── 줄1 ───

  확장 방향: 2 = φ(6) (위, 아래)
  Middle C: 양쪽 보표의 정확한 대칭점
```
