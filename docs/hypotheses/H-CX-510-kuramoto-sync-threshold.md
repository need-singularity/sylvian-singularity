## Kuramoto 동기화

```

  dθ_i/dt = ω_i + (K/N) Σ sin(θ_j - θ_i)

  질서 매개변수: r = |1/N Σ e^(iθ_j)|
  
  r = 0: 완전 비동기 (독립적 의식)
  r = 1: 완전 동기 (의식 융합)
  
  임계값 r_c = 2/3:
    r < 2/3: 개념 전송 실패 (잡음 우세)
    r ≥ 2/3: 개념 전송 성공 (신호 우세)
    
  2/3 = meta fixed point = Golden Zone의 핵심 상수!
```
