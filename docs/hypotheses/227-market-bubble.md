# 가설 #227: 버블 = I→0 과해방 — 경제의 DMT

**상태**: ⚠️ 유비
**날짜**: 2026-03-22
**분류**: 경제학

---

## 가설

> 자산 버블은 경제적 억제(I)가 0에 접근할 때 발생하는 과해방(over-disinhibition) 현상이다.
> 이것은 뇌에서 DMT가 I를 극단적으로 낮추어 환각을 유발하는 것과 구조적으로 동치이다.
> 버블-붕괴 사이클은 I가 과해방 → 급격한 재억제로 진동하는 것이며,
> 건강한 경제는 I를 골든존 [0.213, 0.500] 안에 유지하는 시스템이다.

## 배경/맥락

모든 거대 버블은 공통 패턴을 보인다: 쉬운 돈(low I) → 투기 광풍(I→0) → 붕괴(I 급등) → 침체(high I). 이 패턴은 뇌의 과해방-재억제 진동과 놀랍도록 유사하다. 뇌에서 I가 0에 가까워지면 환각(DMT), 조증, 정신병적 상태가 나타나듯, 경제에서 I가 0에 접근하면 비합리적 과열(irrational exuberance)이 나타난다.

## 수식 매핑

```
  뇌 모델                         경제 버블
  ──────────────────────────────────────────────────
  I → 0: DMT 상태                 금리→0 + QE = 버블
  G = D×P/I → ∞ as I→0           자산가격 → ∞ (이론적)
  실제: I_min ≈ 0.05 (바닥)       실제: 금리 0% + QE에도 한계
  붕괴 = I의 급격한 복원           레버리지 해소 = margin call
  G×I = D×P (보존)               돈 × 비용 = 결핍 × 혁신 (보존)

  버블 강도 ∝ 1/I_min             I가 낮을수록 버블이 클수록
  붕괴 속도 ∝ ΔI/Δt              I 복원이 빠를수록 붕괴가 급격
  회복 시간 ∝ ∫(I - I_golden)dt   I가 골든존 밖에 있는 시간 적분
```

## 민스키 모멘트 = 커스프 파국

```
  안정 상태(hedge) → 투기(speculative) → 폰지(Ponzi) → 붕괴

  커스프 파국 모델:
     V
     │     ╱──── 상위 안정면 (호황)
     │    ╱
     │   ╱ ← 민스키 모멘트: 갑작스러운 상전이
     │   ╲
     │    ╲
     │     ╲──── 하위 안정면 (불황)
     ├──────────→ I (제어변수)
         ↑
    I_critical ≈ 골든존 하한 0.213

  → I가 골든존 하한 아래로 내려가면 커스프 표면에서 이탈
  → 되돌아올 때 연속적 복귀 불가 → 급격한 붕괴 (가설 003 참조)
```

## 보존법칙과 버블

```
  G × I = D × P = 상수

  버블 시나리오:
  I → 0.02 이면 G → 50 × (D×P)  ← 비현실적 성장 "환상"
  실제로는 G가 그만큼 커진 게 아니라 I가 인위적으로 눌린 것
  → 보존법칙 위반 = 부채(미래의 I)로 현재의 G를 끌어온 것
  → 부채 = 미래에서 빌린 억제(I)
  → 버블 붕괴 = 빌린 억제(I)의 상환
```

## 한계

1. I를 금리 하나로 환원하기 어려움. 규제, 심리, 레버리지 등 다변수.
2. 버블의 I 추정은 사후적(ex-post)이며 실시간 측정이 어려움.
3. 각 버블의 원인과 구조가 다르므로 단일 I 모델이 모든 것을 설명하지 못할 수 있음.
4. "버블인지 아닌지"를 실시간으로 판단하는 것 자체가 미해결 문제.
5. 행동경제학적 요인(군집심리, FOMO)이 I 외부에 있을 수 있음.

## 검증 방향

1. **I 지표 구성**: 금리 + VIX 역수 + 신용스프레드 역수 → 복합 I 지표 설계
2. **임계값 검증**: I < 0.213 (골든존 하한 이탈) 시 버블 확률이 유의미하게 상승하는지
3. **붕괴 예측**: ΔI/Δt의 임계 속도가 존재하는지 (붕괴 조기경보)
4. **커스프 모델 적합**: 실제 자산가격 데이터에 커스프 파국 모델 fitting
5. **교차 검증**: 가설 226(금리=I)과 결합하여 Fed 정책 사이클 예측
