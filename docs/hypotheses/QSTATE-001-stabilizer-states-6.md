# QSTATE-001: Bloch 구 안정자 상태 = P₁ = 6

> **가설**: 단일 큐비트의 안정자 상태(stabilizer states)는 정확히 6 = P₁개이며, Bloch 구의 정팔면체 꼭짓점에 해당한다.

## 수치 검증

| 상태 | Pauli 고유상태 | Bloch 벡터 |
|------|---------------|------------|
| |0⟩ | +Z | (0,0,1) |
| |1⟩ | -Z | (0,0,-1) |
| |+⟩ | +X | (1,0,0) |
| |-⟩ | -X | (-1,0,0) |
| |i⟩ | +Y | (0,1,0) |
| |-i⟩ | -Y | (0,-1,0) |

```
Bloch 구의 안정자 상태:

        |0⟩ (Z+)
         ↑
         │
  |-⟩ ──●── |+⟩
  (X-)  /│╲  (X+)
       / │ ╲
  |-i⟩  │  |i⟩
  (Y-)   ↓  (Y+)
        |1⟩ (Z-)

  6 꼭짓점 = P₁ = 정팔면체!
  3 축 = P₁/2
  대각 쌍 = 3 = P₁/2
```

## 양자정보에서의 중요성

- Clifford 게이트는 안정자 상태를 안정자 상태로 매핑
- 안정자 상태 = 양자 오류 교정의 기본 블록
- Magic state = 안정자가 아닌 상태 → 범용 양자컴퓨팅

| 양자 수 | 값 | P₁ 연결 |
|---------|------|---------|
| 안정자 상태 | 6 | P₁ |
| Pauli 행렬 (I 포함) | 4 | τ(6) |
| Bell 상태 | 4 | τ(6) |
| 비자명 Pauli | 3 | P₁/2 |
