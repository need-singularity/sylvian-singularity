# 가설 281: 장력 시간적 인과 — 장력이 학습의 선행 지표

> **학습 중에 특정 클래스의 장력이 먼저 높아지면, 그 클래스의 정확도가 뒤따라 향상되는가? 장력이 학습의 선행 지표(leading indicator)라면, 장력 모니터링으로 학습 진행을 예측 가능.**

## 검증 방법

```
  RepulsionFieldQuad를 MNIST에서 학습하면서 매 에폭:
    - 숫자별 평균 장력 기록
    - 숫자별 정확도 기록
  분석:
    - 장력(t) → 정확도(t+1)? (시차 상관)
    - 장력이 먼저 오르는 숫자가 먼저 정확도 향상?
```
