# Hypothesis H-CX-419: N=6 Engines = Optimal Collective Intelligence

## 가설

> 완전수 6의 약수 구조 {1, 2, 3, 6}은 최대 하위그룹 유연성을 제공하며,
> N=6 엔진 앙상블이 accuracy x diversity 곱(Pareto front)에서 최적을 달성한다.
> 6의 고유약수 {1, 2, 3}의 합 = 6 (완전수)이므로, 하위그룹 크기 1, 2, 3으로
> 분할 가능한 유일한 소규모 앙상블이다.

## 배경/맥락

- H-CX-297에서 N=2가 이상치 탐지에 최적임을 확인

- 고유약수 {1, 2, 3}은 합=6으로 완전수 조건 충족
- 질문: 일반 분류 과제에서 N=6이 특별한 위치를 차지하는가?

## 해석

1. **N=6은 Acc x Div 최적이 아니다.** N=4 (tau(6))가 최적.
2. **N=3이 순수 accuracy 최적** (N=12와 동률). 3은 6의 고유약수.
3. **N=4 = tau(6)가 Pareto 최적점.** 이는 완전수 6의 약수의 수가 최적 앙상블 크기를 결정함을 시사.
4. **약수 유연성과 voting pattern은 비례**: N=6은 45개 패턴 (N<=7 중 최다), N=12는 55개.
5. **N=6의 특성**: 최대 유연성(flex=3)을 가진 최소 N. accuracy plateau의 시작점.
6. 위상 전이는 N=1->2 (앙상블 출현)와 N=3->4 (약수 구조 전이)에서 발생.

## 한계

- sklearn digits는 단순한 10-class 문제. 더 복잡한 데이터셋에서 N=6 효과가 달라질 수 있음
- Bootstrap 다양성은 진정한 구조적 다양성과 다름
- Acc x Div 곱이 유일한 최적성 지표는 아님
- 단일 실행 결과 (통계적 불확실성 미측정)

## 결론

가설 부분 지지. N=6 자체가 최적은 아니지만, **tau(6)=4가 Pareto 최적**, **6의 고유약수 {3}이 accuracy 최적**,
N=6이 **최대 약수 유연성을 가진 최소 앙상블**이라는 구조적 특성 확인.
완전수 6의 약수 구조가 앙상블 최적화에 간접적으로 연결됨.
