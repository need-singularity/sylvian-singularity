# GEOSCI-005: 자연의 육각형 편재성 — P₁ = 6의 지문

> **가설**: 자연에서 독립적으로 발생하는 육각형 현상이 최소 6 = P₁개 이상이며, 이는 에너지 최소화와 최밀충전이 P₁각형을 선호하기 때문이다.

## 육각형 현상 목록

| # | 현상 | 원인 | 스케일 |
|---|------|------|--------|
| 1 | 눈송이 | H-결합 사면체 → 육방정계 | μm |
| 2 | 벌집 | 밀랍 최소화 (벌집 정리) | cm |
| 3 | 주상절리 | 열수축 균열 | m |
| 4 | 벤젠 C₆H₆ | π전자 공명 | Å |
| 5 | 그래핀 | sp² 결합 120° | nm |
| 6 | 토성 북극 | 유체역학 (Rossby wave) | 10⁴ km |
| 7 | 거북 등딱지 | 성장 패턴 | cm |
| 8 | 파리 겹눈 | 광학 최적화 | μm |

```
육각형이 나타나는 이유:

  1. 최밀충전 (2D): 육각형이 면적/둘레 비 최대
  2. 에너지 최소: Plateau 법칙 → 120° 삼중점
  3. 대칭 + 안정: 정육각형 = 유일한 자기타일링 정다각형
  
  스케일 범위: Å ~ 10⁴ km (10¹³배!)
  → P₁ = 6은 스케일 불변 원리
```
