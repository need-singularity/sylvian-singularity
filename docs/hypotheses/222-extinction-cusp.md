# 가설 검토 222: 멸종 = 커스프 재앙

## 상태: ⚠️ 유비

## 분류: 진화론

## 가설

> 대멸종은 커스프 재앙(cusp catastrophe)의 생태적 구현이다. 환경 변수가 임계점을 넘으면 I(환경저항)가 급격히 골든존 밖으로 이탈하며, 적합도 G가 불연속적으로 붕괴한다. 5대 대멸종은 모두 I 궤적이 골든존 경계를 관통한 사건이다.

## 배경 / 맥락

커스프 재앙(cusp catastrophe)은 르네 톰의 재앙 이론에서 가장 단순한 형태로, 제어 변수 2개와 상태 변수 1개로 구성된다. 우리 모델에서 가설 003(커스프 동치)은 이미 G=D×P/I가 커스프 구조를 가짐을 보였다.

대멸종에서는:
- **제어변수 1** = D (환경 압력 — 소행성 충돌, 화산 활동, 산소 농도 변화 등)
- **제어변수 2** = P (유전적 다양성 — 종 수, 생태적 다양성)
- **상태변수** = I (환경 저항 → G 결정)

I가 골든존 경계(0.213 또는 0.500)를 통과할 때 적합도 G에 불연속적 점프(catastrophe)가 발생한다.

관련 가설: 가설 003(커스프 동치), 가설 221(자연선택), 가설 129(상전이)

## 핵심 관찰

### 1. P-T 멸종 = I → 0 극한
페름-트라이아스 대멸종(96% 종 멸종)은 I가 거의 0에 도달한 사건이다. 시베리아 화산이 만든 극단적 환경(D→∞)이 I를 골든존 하한 아래로 밀어냈다. G = D×P/I에서 I→0이면 G→∞ — 적합도가 "발산"하는데, 이는 시스템 자체가 붕괴함을 의미한다.

### 2. K-Pg 멸종 = 순간적 커스프 전이
소행성 충돌은 수 시간~수 년 만에 D를 극단적으로 끌어올렸다. I의 변화 속도가 중요하다 — 느린 변화는 적응을 허용하지만, K-Pg 수준의 급변은 커스프 점프를 유발한다.

### 3. 멸종 후 회복 = 골든존 재진입
각 대멸종 이후 생물 다양성은 약 10-20Ma 후에 회복되었다. 이는 I가 서서히 골든존으로 돌아오는 과정이다. 회복 시간 ∝ I 이탈 거리.

### 4. 보존법칙의 파괴
대멸종 시 G×I = D×P 보존법칙이 깨진다. 이는 "상전이"에 해당하며 — 가설 129(상전이)에서 예측한 대칭 깨짐과 일치한다.

## 한계

- I_min 추정이 간접적이며 정밀 측정 불가능
- 대멸종의 원인은 복합적이며 단일 변수(I)로 환원하기 어려움
- 커스프 재앙은 2차원 제어공간을 가정하나, 실제 생태계는 고차원
- "느린 멸종"(점진적 종 소멸)은 커스프 모델에 잘 맞지 않음
- 5개 데이터포인트는 통계적 검증에 불충분
