# EVOL-004: 텔로미어 반복 단위 = P₁ = 6 bp

> **가설**: 인간 텔로미어의 반복 단위 TTAGGG는 정확히 6 = P₁ 염기쌍이다.

## 수치 검증

| 속성 | 값 | P₁ 연결 |
|------|------|---------|
| 반복 단위 | TTAGGG | 6 bp = P₁ |
| 반복 횟수 | ~2500 | |
| 총 길이 | ~15000 bp | = 2500 × P₁ |
| G-quadruplex 면 | 4 | τ(6) |

```
텔로미어 구조:

  5'─TTAGGG─TTAGGG─TTAGGG─...─TTAGGG─3'
     ╰──6bp──╯╰──6bp──╯╰──6bp──╯   ╰──6bp──╯
      = P₁    = P₁    = P₁        = P₁
     
  G-quadruplex (텔로미어 끝):
    G─G
    │ │
    G─G     ← τ(6) = 4개 G 적층
    │ │
    G─G
    │ │
    G─G
```

## 진화적 보존

- 텔로미어 반복 단위 6bp는 **척추동물 전체에서 보존**
- 식물: TTTAGGG (7bp) — 약간 다름
- 효모: TG₁₋₃ (가변)
- 척추동물의 6bp 보존은 진화적으로 의미 있음
