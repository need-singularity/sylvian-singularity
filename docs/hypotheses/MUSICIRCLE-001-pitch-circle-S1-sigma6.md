# MUSICIRCLE-001: 음고류 원 S¹ — σ(6)=12 점

> **가설**: 음고류 공간은 위상적으로 원 S¹이며, 12=σ(6)개의 등간격 점이 배치된다. S¹ 위의 Z₁₂ 작용은 반음 이동(transposition)에 해당한다.

## 위상적 구조

```
  음고류 원 S¹ (σ(6)=12 nodes):

          C (0)
       B (11)  C# (1)
     A# (10)    D (2)
    A (9)         D# (3)
     G# (8)     E (4)
       G (7)   F (5)
          F# (6=P₁)
          
  ↺ 시계방향 = +1 반음 (크로매틱 상행)
  σ(6) = 12 등분점
  
  위상적 성질:
    π₁(S¹) = Z        기본군 = 정수군
    H₁(S¹) = Z        첫째 호몰로지
    χ(S¹) = 0          오일러 특성
    S¹ ≅ R/Z           실수/정수 몫
```
