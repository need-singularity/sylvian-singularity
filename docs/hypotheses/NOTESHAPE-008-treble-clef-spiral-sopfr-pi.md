# NOTESHAPE-008: 𝄞 나선 감김 = sopfr(6)π/φ(6) 라디안

> **가설**: 높은음자리표의 총 감김각 ≈ 5π/2 = sopfr(6)·π/φ(6) 라디안이다.

## 수치 검증

```
𝄞의 감김 분석:

  구간         회전 (라디안)
  ──────────── ──────────────
  하부 꼬리     ~π (반회전)
  중앙 G-감싸기  ~2π (1회전)
  상부 루프     ~2π (1회전)
  ──────────── ──────────────
  총합          ~5π/2 = 2.5회전

  5π/2 = sopfr(6) × π / φ(6)
        = 5π/2

  또는: 총 회전 = 2.5 = sopfr(6)/φ(6) = 5/2
```
