# EVOL-070: Lung Lobes = sopfr(6) = 5

> **Hypothesis**: Human lungs have 5 lobes total: 3 right + 2 left = sopfr(6) = 5.

## Numerical Verification

| Lung | Lobes | Names |
|------|-------|-------|
| Right | 3 | Superior, Middle, Inferior |
| Left | 2 | Superior, Inferior |
| Total | 5 = sopfr(6) | |

## Structure

```
  Lung anatomy:

  Right (3):   Left (2):
  [Sup]        [Sup]
  [Mid]        [Inf]
  [Inf]        (heart)

  Right lobes = 3 = P1/2
  Left lobes = 2 = phi(6)
  Total = 5 = sopfr(6)
```

## Structural Meaning

Pulmonary architecture: P1/2 right + phi(6) left = sopfr(6) total.

## Grade

🟩 EXACT -- 5 lung lobes is standard human anatomy

## Limitations
- Some variation exists (accessory lobes); left has 2 due to cardiac notch
