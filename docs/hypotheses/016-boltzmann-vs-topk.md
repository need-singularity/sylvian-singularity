# 가설 검토 016: 볼츠만 라우터가 Top-K보다 우수한가 ✅

## 가설

> 볼츠만 소프트 게이팅(T=e)이 기존 Top-K 하드 게이팅보다 Expert 활용과 다양성에서 우수하다.

## 검증 결과: ✅ 볼츠만 2/3 승

```
  메트릭               │ Top-K (K=8) │ 볼츠만 (T=e) │ 승자
  ────────────────────┼─────────────┼──────────────┼──────
  평균 활성 Expert     │        8.00 │         6.27 │ ─
  활성 수 변동(σ)      │        0.00 │         1.79 │ 볼츠만 ✅
  Expert 활용 불균형   │        0.03 │         0.01 │ 볼츠만 ✅
  조합 다양성(패턴 수) │      787    │       1000   │ 볼츠만 ✅
```

## 의미

- Top-K: 항상 같은 8개 → **경직, 조합 787가지**
- 볼츠만: 매번 다른 조합 → **유동적, 조합 1000가지**
- 볼츠만이 Expert를 더 **균등하게** 활용 (불균형 σ: 0.03 → 0.01)
