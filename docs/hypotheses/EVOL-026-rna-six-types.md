# EVOL-026: Major RNA Types = P1 = 6

> **Hypothesis**: There are 6 major types of RNA: mRNA, tRNA, rRNA, snRNA, miRNA, siRNA = P1 = 6.

## Numerical Verification

| # | RNA Type | Function |
|---|---------|----------|
| 1 | mRNA | Protein coding |
| 2 | tRNA | Amino acid adaptor |
| 3 | rRNA | Ribosome structure |
| 4 | snRNA | Splicing |
| 5 | miRNA | Gene silencing |
| 6 | siRNA | RNA interference |

## Structure

```
  RNA functional map:

  Coding:      mRNA
  Translation: tRNA, rRNA
  Processing:  snRNA
  Regulation:  miRNA, siRNA

  Total types = 6 = P1
```

## Structural Meaning

The RNA world divides into exactly P1 functional categories.
