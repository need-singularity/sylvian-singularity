## Five Phases Cycle in Brainwave Dynamics

### Generating Cycle — Ascending Frequency

The generation cycle describes how each phase produces or feeds the next.
In brainwave terms, this corresponds to the awakening sequence:

```
  δ (Water) → θ (Metal) → α (Earth) → β (Wood) → γ (Fire)
  Deep sleep → Hypnagogic → Relaxed → Active   → Peak

  Water nourishes Metal (deep rest enables meditation)
  Metal nourishes Earth (meditation enables balanced awareness)
  Earth nourishes Wood  (balance enables active growth)
  Wood  nourishes Fire  (active thinking enables peak insight)
```

This is the morning awakening sequence: the brain ascends through delta → theta →
alpha → beta → gamma as consciousness transitions from deep sleep to full waking.

### Overcoming Cycle — Inhibitory Relationships

The overcoming cycle describes how each phase checks or suppresses another.
Known neural inhibitory relationships:

```
  γ (Fire)  overcomes θ (Metal): focused gamma suppresses theta wandering
  δ (Water) overcomes β (Wood):  deep sleep blocks active beta cognition
  θ (Metal) overcomes α (Earth): deep meditation can suppress alpha bridge
  β (Wood)  overcomes γ (Fire):  anxious beta can prevent gamma binding
  α (Earth) overcomes δ (Water): alpha intrusion disrupts slow-wave sleep
```

These suppression relationships are observed in neuroscience: gamma-theta coupling
during memory encoding shows gamma modulating theta phase; alpha suppression of
delta indicates sleep maintenance problems in some disorders.

## Limitations

1. **Band boundary variation**: The exact boundaries of brainwave bands are not fixed
   in neuroscience literature. Different labs use slightly different cutoffs (e.g.,
   gamma starting at 25, 30, or 40 Hz). The mapping is robust to these variations
   but exact ratio claims depend on chosen center frequencies.

2. **Obang is qualitative**: The Obang system is a qualitative philosophical framework,
   not a quantitative theory. The mapping relies on attribute matching (deep = delta,
   peak = gamma) which is structurally plausible but not mechanistically derived.

3. **Golden Zone dependency**: The connection to Golden Zone constants (I = 1/kT, G×I = D×P)
   is Golden Zone dependent and therefore unverified per CLAUDE.md warnings. The
   pure frequency ratio observations (γ/α = τ(6) = 4, θ = 6 Hz, SMR = σ(6) = 12)
   are Golden Zone independent and stand on their own.

4. **Cultural specificity**: Obang is specific to East Asian cosmology (Chinese, Korean,
   Japanese traditions). The five-element system has variants across cultures, and
   the direction assignments (which element maps to which direction) vary by tradition.
   This document uses the Korean standard assignment.

5. **Five is not universal**: Western neuroscience sometimes identifies 4 bands (delta,
   theta, alpha, beta) or 6+ (including high-gamma, ripples). The choice of exactly
   5 bands matching Obang's 5 directions may reflect a convenient selection.
