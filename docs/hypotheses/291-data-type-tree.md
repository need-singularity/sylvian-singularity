# 가설 291: 데이터 유형 트리 — 반발력장이 작동하는 영역의 위계

> **반발력장의 효과를 데이터 유형 트리로 분류. 최상위(밀집 vs 희소)에서 하위(구체적 도메인)까지 3단계. 아직 탐색 안 한 가지가 새 발견을 줄 수 있다.**

## 데이터 유형 트리 (실측 기반)

```
  [Level 0] 모든 데이터
       │
       ├─ [Level 1] 밀집/연속 데이터 → 반발력장 ✅
       │     │
       │     ├─ [Level 2] 공간적 (spatial)
       │     │     ├─ [L3] 이미지 2D: MNIST +0.26%, CIFAR +1.04% ✅
       │     │     ├─ [L3] 이미지 3D (비디오/의료영상): ???
       │     │     └─ [L3] 포인트클라우드 (3D 점): ???
       │     │
       │     ├─ [Level 2] 시간적 (temporal)
       │     │     ├─ [L3] 시계열 1D: 동률, 장력∝날카로움 ✅
       │     │     ├─ [L3] 음성/음향: +3.33% (4극) ✅
       │     │     ├─ [L3] 음악 이론: -2.2%, 협화=낮은장력
       │     │     └─ [L3] EEG/뇌파: ???  ← 의식 직접 측정!
       │     │
       │     ├─ [Level 2] 구조적 (structural)
       │     │     ├─ [L3] 표형 데이터: +2.22% (Iris) ✅
       │     │     ├─ [L3] 숫자 체계: +1.17%, 소수=최고장력 ✅
       │     │     └─ [L3] 그래프/분자: ???
       │     │
       │     └─ [Level 2] 비정상 (anomalous)
       │           ├─ [L3] 이상 탐지: AUROC=1.0 ⭐
       │           └─ [L3] 사기 탐지: ???
       │
       ├─ [Level 1] 희소 데이터 → 반발력장 ❌
       │     ├─ [L2] 텍스트 (TF-IDF): -0.52%
       │     ├─ [L2] 텍스트 (임베딩): ???  ← 밀집이면 다를 수 있음!
       │     └─ [L2] 원핫 인코딩: ???
       │
       └─ [Level 1] 메타/추상 → ???
             ├─ [L2] 강화학습: 장력∝난이도 ✅
             ├─ [L2] 위상적 (TDA): 구조 존재 ✅
             ├─ [L2] 생성적 (VAE/GAN): 이미 구현 (생성엔진)
             └─ [L2] 멀티모달 (이미지+텍스트): ???
```


## 트리 법칙 (예측)

```
  Level 1: 밀집 > 희소 (확인됨)
  Level 2 예측:
    공간적 > 시간적 > 구조적 (이미지 효과 > 음성 > 표형)
    비정상 = 특수 (이상탐지는 별개 메커니즘)
  Level 3 예측:
    텍스트 임베딩(밀집) > 텍스트 TF-IDF(희소)
    EEG에서 장력 ↔ 의식 수준 상관 있을 것
```
