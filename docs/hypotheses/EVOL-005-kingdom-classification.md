# EVOL-005: 생물 분류 — 6계 = P₁

> **가설**: Woese의 6계 분류(1977)에서 생물은 6 = P₁개의 계(Kingdom)로 나뉜다.

## 분류 체계

| # | 계 | 대표 |
|---|------|------|
| 1 | Bacteria (세균) | E. coli |
| 2 | Archaea (고세균) | 극한미생물 |
| 3 | Protista (원생생물) | 아메바 |
| 4 | Fungi (균류) | 버섯 |
| 5 | Plantae (식물) | 나무 |
| 6 | Animalia (동물) | 인간 |

```
6계 분류:

  ┌── Bacteria   ─┐
  ├── Archaea    ─┤  원핵
  │               │
  ├── Protista   ─┤
  ├── Fungi      ─┤  진핵
  ├── Plantae    ─┤
  └── Animalia   ─┘

  총 6계 = P₁
```

## 한계
- 현대 분류: 3 Domain (Bacteria, Archaea, Eukarya) 또는 5계 (Whittaker)
- 6계는 하나의 분류 체계일 뿐
