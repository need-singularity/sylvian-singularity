# MUSIC-004: 기타 — 6현 = P₁

> **가설**: 표준 기타의 현 수 6 = P₁은 인간 손의 sopfr(6)=5개 손가락과 P₁=6개 현의 최적 조합이다.

## 수치 검증

| 속성 | 값 | P₁ 연결 |
|------|------|---------|
| 표준 현 수 | 6 | P₁ |
| 연주 손가락 | 4 (검지~소지) | τ(6) |
| 프렛 손가락 | 4 (검지~소지) | τ(6) |
| 표준 튜닝 음정 | E-A-D-G-B-E | |
| 튜닝 간격 | 5-5-5-4-5 (반음) | sopfr(6) 우세 |

```
기타 구조의 P₁:

  현:     6 = P₁
  프렛 수: ~20 (표준)
  
  튜닝 간격 (반음):
  E→A: 5 = sopfr(6)
  A→D: 5 = sopfr(6)
  D→G: 5 = sopfr(6)
  G→B: 4 = τ(6)
  B→E: 5 = sopfr(6)
  
  → 5개 간격 중 4개 = sopfr(6), 1개 = τ(6)
```
