# NOTESHAPE-004: 위상 전이점 — 4분↔2분 경계

> **가설**: 4분음표(●)에서 2분음표(○)로의 전환은 위상적 상전이(topological phase transition)이다. D² → Annulus에서 기본군이 0 → Z로 점프한다.

## 구조

```
  위상 전이 다이어그램:

  음표:   32nd  16th  8th  Quarter │ Half   Whole
  머리:    ●     ●     ●     ●     │  ○      ○
  π₁:     0     0     0     0     │  Z      Z
  χ:      1     1     1     1     │  0      0
  시가:   ⅛    ¼    ½    1      │  2      4
                                  │
               Phase I (D²)       │  Phase II (Ann)
               단순연결            │  비단순연결
               수축 가능           │  구멍 있음
                                  ↑
                           위상 전이점!
                        1박 = 심장 박동 1회
```

## 물리적 해석

| Phase | 머리 | χ | π₁ | 물리적 의미 |
|-------|------|---|-----|------------|
| I (●) | 채움 | 1 | 0 | 수축기(systole): 에너지 응축 |
| II (○) | 빈 | 0 | Z | 이완기(diastole): 에너지 확산 |

전환점 = 1박 ≈ 심장 박동 1주기 (♩=60~80 BPM)

이것은 음표 기호법이 **무의식적으로** 위상적 상전이를 인코딩한 것!
