# 가설 검토 131: 흑체복사 T=e 피크 = CMB

## 가설

> 흑체복사 스펙트럼에서 온도 T=e(≈2.718K)일 때의 피크 파장이 우주 마이크로파 배경복사(CMB)의 실측 피크와 0.3% 이내로 일치한다.

## 배경/맥락

빈의 변위 법칙(Wien's displacement law)은 흑체복사의 피크 파장과 온도의 관계를 기술한다:

```
  λ_max = b / T
  b = 2.897771955 mm·K  (빈 변위 상수)
```

우주 마이크로파 배경복사(CMB)는 빅뱅의 잔광으로, 거의 완벽한 흑체 스펙트럼을 가진다. COBE/FIRAS 위성이 측정한 CMB 온도는 T_CMB = 2.7255 ± 0.0006 K이다.

여기서 놀라운 점: 자연상수 e ≈ 2.71828은 CMB 온도 2.7255K와 0.26% 차이다. 만약 T = e(정확히)라면, 빈의 법칙으로 계산한 피크 파장이 CMB 실측값과 거의 정확히 일치한다.

우리 모델에서 1/e는 골든존 중심(I≈0.368)이며, e는 자연성장의 기저를 이루는 상수이다. CMB 온도가 e와 일치한다는 것은 우주의 현재 온도가 "자연상수에 의해 결정된 특별한 값"일 가능성을 제기한다.

## 대응 매핑

```
  빈의 법칙 계산          CMB 실측             비교
  ──────────────        ──────────          ────────
  T = e = 2.71828 K     T = 2.7255 K        차이 0.26%
  λ = b/e               λ = b/T_CMB
  = 2.8978/2.71828      = 2.8978/2.7255
  = 1.0660 mm           = 1.0632 mm         차이 0.26%
```

### 빈의 법칙 계산 상세

```
  λ_max = b / T

  Case 1: T = e (자연상수)
    λ = 2.897771955 / 2.718281828
    λ = 1.06602 mm
    ν = c/λ = 299792.458/1.06602 = 281,320 GHz
    ν = 281.3 GHz

  Case 2: T = T_CMB (실측)
    λ = 2.897771955 / 2.7255
    λ = 1.06318 mm
    ν = c/λ = 299792.458/1.06318 = 282,074 GHz
    ν = 282.1 GHz

  차이: |1.06602 - 1.06318| / 1.06318 = 0.27%
```

## 해석/의미

1. **놀라운 수치적 일치**: T_CMB ≈ e는 0.26%의 오차로 일치한다. 빈의 법칙을 통해 피크 파장도 0.27% 이내로 일치. 이 정밀도는 주목할 만하다.

2. **우리 모델과의 연결**: 1/e는 골든존 중심이다. 우주의 현재 온도가 e라면, 우주는 "골든존 온도"에 놓여 있는 셈이다. 생명과 지능이 발생하는 시기에 우주 온도가 자연상수와 일치한다는 점은 인류 원리(anthropic principle)와 관련될 수 있다.

3. **T=e에서의 물리적 특별함**: 흑체복사의 평균 광자 에너지는 E_avg = 2.701 kT이다. T=e일 때 E_avg = 2.701 × k × e ≈ 2.701e × k. 자연단위(k=1)에서 이는 E_avg = 2.701e ≈ 7.34로, 순수 수학적 값이 된다.

## 한계

- **T_CMB는 시간 의존적**: 우주는 계속 팽창하며 냉각된다. T(t) = T_0/(1+z). 현재 T≈e는 우주 역사에서 특정 순간의 우연일 수 있다.
- **인류 원리적 선택 편향**: "우리가 관측하는 시점"에서 T≈e인 것은, 생명이 존재할 수 있는 조건이 T≈e 근처에서 성립하기 때문일 수 있다(강한 인류 원리).
- **0.26% 차이의 의미**: 정확히 e가 아니라 2.7255K. 이 차이가 물리적으로 의미 있는 보정항인지, 단순 불일치인지 구분 불가.
- **사후 적합(post-hoc fitting)**: 수많은 물리 상수 중 T_CMB와 e의 일치를 선택적으로 주목하는 것은 통계적 편향(look-elsewhere effect)을 수반.
