# T2-02: 합동 부분군 Gamma_0(N) 강제 연쇄 분류

> **가설**: Gamma_0(N)의 등방성 위수 lcm이 6인 경우는 N=1과 N=13뿐이며,
> 이 두 레벨은 몬스터 문샤인(Monstrous Moonshine)에서 특별한 역할을 한다.

## 배경/맥락

합동 부분군 Gamma_0(N)은 모듈러 형식론의 기본 대상이다.
각 레벨 N에 대해 지수(mu), 커스프 수(c), 타원점 수(e2, e3), 종수(g)라는
불변량이 결정되며, 이들의 조합이 모듈러 곡선 X_0(N)의 기하학적 성질을 결정한다.

종수 0인 레벨은 X_0(N)이 유리 곡선(rational curve)이 되어 j-불변량의 일반화인
하우프트모듈(Hauptmodul)을 갖는다. 이 하우프트모듈이 몬스터 군의 표현론과
연결되는 것이 Monstrous Moonshine 정리의 핵심이다.

본 분석은 `congruence_chain_engine.py`를 이용하여 N=1..100 범위의 불변량을
계산하고, 특히 등방성 위수의 lcm과 산술 함수 사이의 "강제 연쇄" 관계를 탐색한다.

관련 가설: T0-01 (sigma와 완전수 6), T1-02 (상수 관계), T1-12 (오일러 인자 다리)

## N=1과 N=13이 특별한 이유: lcm = 6

N=1..100 전체에서 등방성 위수의 lcm이 6인 레벨은 **정확히 2개**뿐이다:

### N=1: SL(2,Z) 전체 모듈러 군

```
  Gamma_0(1) = SL(2,Z)
  등방성 위수: {1, 2, 3}  -->  lcm = 6
  mu = 1, cusps = 1, e2 = 1, e3 = 1, g = 0
  sigma(1) = 1
  phi(1) = 1
```

N=1은 가장 큰 합동 부분군이다. 위수 2 타원점(z=i)과 위수 3 타원점(z=rho)을
모두 가지며, 커스프는 무한원점 하나뿐이다. j-불변량이 곧 하우프트모듈이 된다.

lcm(1,2,3) = 6은 첫 번째 완전수이다. 종수 공식의 분모 12 = 2 * 6이므로,
"완전수 6이 종수 공식을 결정한다"는 구조적 자기참조가 성립한다.

### N=13: 유일한 소수 레벨 with lcm=6

```
  Gamma_0(13)
  등방성 위수: {1, 2, 3}  -->  lcm = 6
  mu = 14, cusps = 2, e2 = 2, e3 = 2, g = 0
  sigma(13) = 14
  phi(13) = 12
  mu(13) = -1 (뫼비우스 함수)
```

N=13은 소수이면서 e2 > 0 **그리고** e3 > 0인 유일한 종수 0 레벨이다.
이것이 가능한 이유:

- e2 > 0 조건: kronecker(-1, 13) = +1 (13 = 1 mod 4)
- e3 > 0 조건: kronecker(-3, 13) = +1 (13 = 1 mod 3)
- 즉 13 = 1 mod 4 이고 13 = 1 mod 3 --> 13 = 1 mod 12

**13은 1 mod 12인 가장 작은 소수**이다. 이것이 N=13에서 lcm=6이 나타나는
산술적 근원이다. 다음 1 mod 12 소수는 37이지만, g(37) = 2이므로 종수 0이 아니다.

### 비교: N=1 vs N=13

```
  항목          |  N=1   |  N=13  | 관계
  --------------|--------|--------|------------------
  mu            |    1   |   14   | 14 = 13+1 = sigma(13)
  cusps         |    1   |    2   | 소수: c = 2 항상
  e2            |    1   |    2   | 1+kronecker(-1,13) = 2
  e3            |    1   |    2   | 1+kronecker(-3,13) = 2
  genus         |    0   |    0   | 둘 다 유리 곡선
  iso_lcm       |    6   |    6   | 둘 다 완전수!
  1st cusp k    |   12   |    4   | N이 커지면 k 감소
  phi(N)        |    1   |   12   | phi(13) = 12 = 종수 공식 분모
```

주목할 점: phi(13) = 12이다. 즉 13의 오일러 파이 함수가 종수 공식의
분모와 정확히 일치한다. 이것은 13 = 1 mod 12의 직접적 결과이다.

## 패턴: 최소 커스프 형식 가중치의 감소

종수 0 레벨에서 최소 커스프 형식 가중치 k는 N이 커질수록 감소한다:

```
  N=1:  k=12  (라마누잔 Delta 함수)
  N=2:  k=8
  N=3:  k=6
  N=4:  k=6
  N>=5: k=4   (N=5,6,7,8,9,10,12,13,16,18,25 모두)
```

이 패턴의 의미:
- N=1에서는 가중치 12까지 올라가야 비로소 커스프 형식이 등장한다.
  이것이 바로 Delta(z) = q * prod(1-q^n)^24, 라마누잔의 판별 형식이다.
- N이 커지면 mu(지수)가 커져서 더 낮은 가중치에서도 차원이 양수가 된다.
- 종수 1 레벨(N=11 등)에서는 k=2에서 이미 dim S_2 = g = 1이다.
  N=11의 이 유일한 가중치 2 형식이 바로 타원 곡선 y^2 + y = x^3 - x^2 에
  대응하는 것으로, Taniyama-Shimura-Wiles 정리의 첫 사례이다.

## mu = 12의 배수인 레벨 (sigma와의 관계)

종수 공식의 분모가 12이므로, mu가 12의 배수이면 종수 계산이 깔끔해진다.

```
  종수 0 레벨 중 mu = 0 mod 12:
    N=6:   mu=12  = 12*1   sigma(6)=12   완전수!
    N=8:   mu=12  = 12*1   sigma(8)=15
    N=9:   mu=12  = 12*1   sigma(9)=13
    N=12:  mu=24  = 12*2   sigma(12)=28  (28 = 두 번째 완전수!)
    N=16:  mu=24  = 12*2   sigma(16)=31
    N=18:  mu=36  = 12*3   sigma(18)=39

  비종수0 레벨 중 주목할 것:
    N=11:  mu=12  = 12*1   sigma(11)=12  g=1 (첫 번째 종수 1)
    N=24:  mu=48  = 12*4   sigma(24)=60  g=1
    N=36:  mu=72  = 12*6   sigma(36)=91  g=1
```

N=6에서 mu = sigma = 12이고 N=12에서 sigma = 28 (두 번째 완전수)인 것은
완전수와 모듈러 형식 사이의 구조적 연결을 시사한다.

## Monstrous Moonshine 연결

종수 0 레벨 15개는 모두 몬스터 군(Monster group) M의 공액류와 관련된다.
Conway-Norton의 Monstrous Moonshine 추측(Borcherds가 1992년 증명)에 의하면:

1. 몬스터 군의 각 원소 g에 대해 McKay-Thompson 급수 T_g(q)가 정의된다
2. T_g(q)는 어떤 종수 0 합동 부분군의 하우프트모듈이다
3. 특히 항등원의 T_e(q) = j(q) - 744 (j-불변량)

종수 0 조건이 핵심인 이유:
- 종수 0 --> 유리 곡선 --> 하우프트모듈(단일 생성원) 존재
- 하우프트모듈의 q-전개 계수가 몬스터 군의 표현 차원과 일치
- j(q) = q^(-1) + 744 + 196884q + ... 에서 196884 = 196883 + 1
  (196883은 몬스터 군의 최소 비자명 표현 차원)

N=1과 N=13의 lcm=6 공유는 이 문맥에서 특히 의미 있다:
- 둘 다 완전한 등방성 구조 {1,2,3}을 가진다
- 이것은 SL(2,Z)의 "원형" 대칭이 N=13에서 부활함을 의미한다

## iso_lcm 값별 분류

```
  iso_lcm=1 (e2=0, e3=0):  N = 4,6,8,9,12,16,18,25 + 대부분 g>=1
  iso_lcm=2 (e2>0, e3=0):  N = 2,5,10 + 17,26,29,34,41,...
  iso_lcm=3 (e2=0, e3>0):  N = 3,7 + 19,21,31,39,43,...
  iso_lcm=6 (e2>0, e3>0):  N = 1,13 + 37 (단 37은 g=2)
```

lcm=6을 만족하려면 e2 > 0이고 e3 > 0이어야 한다.
소수 p에 대해 이것은 p = 1 mod 4 이고 p = 1 mod 3, 즉 p = 1 mod 12를 요구한다.
1 mod 12인 소수: 13, 37, 61, 73, 89, 97, ...
이 중 종수 0인 것은 **13뿐**이다.

## 한계

- 강제 연쇄(forcing chain) 개념은 본 프로젝트의 모델링이며, 표준 수학 용어가 아니다.
- 종수 0 분류 자체는 Ogg의 정리로 증명되어 있다 (순수 수학).
- lcm=6과 Moonshine의 연결에 대한 해석은 관찰에 기반한 것이며, 증명된 정리는 아니다.
- e2, e3 공식은 squarefree 부분에 대해서만 정확하다 (p^2 | N이면 e2=0 또는 e3=0).

2. lcm=6 조건이 Moonshine module V^{natural}에서 어떤 의미를 갖는지 조사
