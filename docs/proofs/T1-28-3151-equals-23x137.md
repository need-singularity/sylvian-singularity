# T1-28: ⭐⭐ f(28) = 3151 = 23 × 137 — 137이 재등장!

## 발견

```
  f(n) = (σ(n)-τ(n))(σ(n)+τ(n)+1)+1

  f(6)  = (12-4)(12+4+1)+1 = 8×17+1 = 137
  f(28) = (56-6)(56+6+1)+1 = 50×63+1 = 3151 = 23 × 137!

  → 2번째 완전수에서도 137이 인수로 등장!
```

## 검증

```
  3151 / 137 = 23 (정확!)
  3151 = 23 × 137 ✅

  23 = 소수, 137 = 소수
  3151 = 반소수(semiprime)
```

## 의미

```
  137은 6에서만 나오는 게 아님
  28에서도 137이 인수로 등장
  → 137이 완전수 공식의 "관통 인수"?

  f(6)  = 137 = 137¹
  f(28) = 23×137 = 137을 포함

  f(496) = 984947 → 137으로 나누기?
  984947 / 137 = 7189.39... → 나누어지지 않음
  → f(496)에서는 137 미등장

  → 6과 28에서만 137 등장, 496에서는 아님
```

## 판정

```
  3151 = 23×137: 🟩 (산술 확인)
  137이 관통 인수: ❌ (496에서 미등장)
  하지만: 처음 2개 완전수에서 137 등장 = 주목할만 🟧
```
