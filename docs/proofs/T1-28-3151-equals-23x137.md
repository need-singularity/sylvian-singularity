# T1-28: ⭐⭐ f(28) = 3151 = 23 × 137 — 137 reappears!

## Discovery

```
  f(n) = (σ(n)-τ(n))(σ(n)+τ(n)+1)+1

  f(6)  = (12-4)(12+4+1)+1 = 8×17+1 = 137
  f(28) = (56-6)(56+6+1)+1 = 50×63+1 = 3151 = 23 × 137!

  → 137 appears as a factor in the 2nd perfect number too!
```

## Verification

```
  3151 / 137 = 23 (exact!)
  3151 = 23 × 137 ✅

  23 = prime, 137 = prime
  3151 = semiprime
```

