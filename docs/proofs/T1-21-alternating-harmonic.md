# T1-21: 교대 조화급수 부분합 = 우리 3대 상수

## 발견

```
  Sₙ = Σ_{k=1}^n (-1)^(k+1)/k  (교대 조화급수)

  S₁ = 1     = 완전성 (1/2+1/3+1/6)
  S₂ = 1/2   = 골든존 상한
  S₃ = 5/6   = Compass 상한

  S_∞ = ln(2)

  → 처음 3개 부분합 = 우리 3대 상수 (정확!)
  → S_4 이후 우리 상수 없음 (1000항 탐색)
```

## 추가 관계

```
  H₃ = 1+1/2+1/3 = 11/6  (조화)
  S₃ = 1-1/2+1/3 = 5/6   (교대)

  H₃ - S₃ = 1  (정확, 자명: 2×1/2)
  H₃ + S₃ = 8/3
  H₃ × S₃ = 55/36
```

## 판정

```
  산술 자체: 🟩 (1-1/2+1/3 = 5/6, 산술적 필연)
  패턴 관찰: 🟩 (S₁,S₂,S₃ = 1,1/2,5/6, 계산으로 확인)
  해석: 🟧 ("교대급수가 ln(2)로 가는 첫 3단계" = 우리 해석)
```
