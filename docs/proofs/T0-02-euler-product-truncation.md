# T0-02: 오일러 곱 p=2,3 절단 → 정수 2

## 명제

오일러 곱을 소수 p=2,3에서 절단하면 약수함수가 정확히 정수 2를 산출하며, 이는 유일한 정수 산출 조합이다.

## 오일러 곱 공식

리만 제타함수의 오일러 곱 표현:

```
ζ(s) = Π_p 1/(1 - p⁻ˢ)    (모든 소수 p에 대한 곱)
```

## p=2,3 절단 (s=1 형식)

```
Π_{p∈{2,3}} 1/(1 - 1/p) = 1/(1 - 1/2) × 1/(1 - 1/3)
                          = 1/(1/2) × 1/(2/3)
                          = 2 × 3/2
                          = 3
```

## 약수함수 형태

σ₋₁(6) = Π_{p|6} (1 + 1/p):

```
(1 + 1/2)(1 + 1/3) = (3/2)(4/3) = 12/6 = 2
```

## 다른 소수 조합의 전수 검사

| 소수 집합 | Π(1+1/p) | 정수 여부 |
|-----------|----------|-----------|
| {2} | 3/2 = 1.500 | 아니오 |
| {3} | 4/3 = 1.333 | 아니오 |
| {2, 3} | **2.000** | **예** |
| {2, 5} | 9/5 = 1.800 | 아니오 |
| {2, 3, 5} | 12/5 = 2.400 | 아니오 |
| {2, 3, 7} | 16/7 = 2.286 | 아니오 |
| {2, 3, 5, 7} | 96/35 = 2.743 | 아니오 |

## 정수가 되는 이유

```
Π_{p|n}(1 + 1/p) = σ₋₁(n)  (n이 무제곱수일 때)
σ₋₁(n) ∈ ℤ  ⟺  σ(n) ≡ 0 (mod n)  ⟺  n은 완전수
2 × 3 = 6 = 최소 완전수
```

따라서 p=2,3에서의 정수 산출은 6이 완전수라는 사실의 직접적 귀결이다.
