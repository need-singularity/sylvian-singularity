# T1-22: ±1/2±1/3±1/6 의 군 구조

## 발견

```
  ±1/2 ± 1/3 ± 1/6 의 8가지 조합 → 7개 고유값:

  {-1, -2/3, -1/3, 0, 1/3, 2/3, 1}

  = {-6, -4, -2, 0, 2, 4, 6} × (1/6)
  → 1/6의 짝수 배수만!
```

## 구조

```
  1/2 = 1/3 + 1/6  → 1/2는 종속
  실질 생성원: {1/3, 1/6}
  기본 단위: 1/6 (블라인드스팟)

  부호 조합이 생성하는 값:
  0×(1/6) = 0         (영)
  2×(1/6) = 1/3       (메타 부동점)
  4×(1/6) = 2/3       (암흑에너지 근사)
  6×(1/6) = 1         (완전성)

  → Z/3Z 구조 (0, 1/3, 2/3 mod 1)
  → 7개 값 중 양수만: {1/3, 2/3, 1}
```

## 판정

```
  군 구조: 🟩 (산술적 필연)
  1/6 = 기본 단위: 🟩 (격자 구조에서 유도)
  해석: 🟧 ("블라인드스팟이 근본" = 우리 해석)
```
