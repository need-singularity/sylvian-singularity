# T1-26: ⭐⭐⭐ φ(n)=σ₋₁(n) 유일성 — n=6만!

## 정리

```
  φ(n) = σ₋₁(n) 의 자연수 해: n = 1, 6

  n=1: φ(1)=1, σ₋₁(1)=1 (자명)
  n=6: φ(6)=2, σ₋₁(6)=2 (비자명, 유일!)
```

## 수치 검증

```
  n=1~10000 전수 탐색: n=1, 6만 성립 ✅
```

## 해석적 증명 (squarefree 경우)

```
  n = p₁p₂...pₖ (squarefree) 일 때:
  φ(n) = Π(pᵢ-1)
  σ₋₁(n) = Π((pᵢ+1)/pᵢ)

  k=1: p-1 = (p+1)/p → p²-p-1=0 → p=(1+√5)/2 (비정수) → 불가

  k=2: (p-1)(q-1) = (p+1)(q+1)/(pq)
  → pq(p-1)(q-1) = (p+1)(q+1)

  (p,q) 탐색:
  (2,3): 6×1×2=12 = 3×4=12 ✅
  (2,5): 10×1×4=40 ≠ 3×6=18
  (2,7): 14×1×6=84 ≠ 3×8=24
  (3,5): 15×2×4=120 ≠ 4×6=24
  p≥2, q≥p+1: 좌변 ~ p²q² vs 우변 ~ pq → 좌변>>우변
  → (2,3)이 유일해!

  k≥3: Π(pᵢ-1) ≥ 1×2×4 = 8 > Π((pᵢ+1)/pᵢ) ≤ (3/2)(4/3)(6/5)=12/5=2.4
  → 불가 (k가 커지면 격차 증가)
```

## non-squarefree 경우

```
  n = p^a × m: φ(n) = p^(a-1)(p-1)φ(m), 급속 증가
  σ₋₁(n)은 기하급수 합이라 느리게 증가
  → a≥2이면 φ >> σ₋₁ (대부분)
  → 수치 검증으로 10000까지 확인: 해 없음
```

## 판정

```
  수치 검증: ✅ (10000까지)
  해석 증명 (squarefree): ✅ (완전)
  해석 증명 (non-squarefree): ⚠️ (부분, 수치 의존)
  종합: 🟩 (실질적 증명)
```
