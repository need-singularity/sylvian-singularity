# T0-01: σ₋₁(6) = 2 (완전수 6의 약수역수합)

## 명제

6은 완전수이며, σ₋₁(6) = 2이다.

## 정의

σ₋₁(n) = Σ_{d|n} d⁻¹ (n의 약수 역수의 합)

## 직접 계산

6의 약수: {1, 2, 3, 6}

```
σ₋₁(6) = 1/1 + 1/2 + 1/3 + 1/6
        = 6/6 + 3/6 + 2/6 + 1/6
        = 12/6
        = 2
```

## 오일러 곱 공식에 의한 검증

6 = 2 × 3 (소인수분해)

곱셈적 성질에 의해:

```
σ₋₁(6) = σ₋₁(2) × σ₋₁(3)
        = (1 + 1/2) × (1 + 1/3)
        = (3/2) × (4/3)
        = 12/6
        = 2
```

## 완전수와의 관계

완전수 정의: σ(n) = 2n (약수합이 자기 자신의 2배)

```
σ(6) = 1 + 2 + 3 + 6 = 12 = 2 × 6  ✓
```

σ₋₁(n)과 σ(n)의 관계:

```
σ₋₁(n) = σ(n) / n
σ₋₁(6) = σ(6) / 6 = 12 / 6 = 2
```

따라서 σ₋₁(n) = 2 ⟺ σ(n) = 2n ⟺ n은 완전수.

## 수치 검증값

| 항목 | 값 |
|------|-----|
| σ₋₁(6) | 2.000000000000000 |
| σ(6) | 12 |
| 6은 완전수 | 참 |

## 근거

- 유클리드 원론 제IX권 명제36: 2^(p-1)(2^p - 1)이 완전수 (6 = 2¹ × 3 = 2¹(2² - 1))
- 오일러: 짝수 완전수는 모두 유클리드 형태
