# T0-07: Γ(n, λ) = n개 Exp(λ)의 합

## 명제

독립인 n개의 지수분포 확률변수의 합은 감마분포 Γ(n, λ)를 따른다.

## 정리

X₁, X₂, ..., Xₙ이 독립이고 각각 Exp(λ)를 따르면:

```
Y = X₁ + X₂ + ... + Xₙ ~ Γ(n, λ)
```

## 적률생성함수(MGF)에 의한 증명

Exp(λ)의 MGF:

```
M_X(t) = λ/(λ - t),  t < λ
```

독립이므로 합의 MGF는 곱:

```
M_Y(t) = [M_X(t)]ⁿ = [λ/(λ - t)]ⁿ
```

이것은 Γ(n, λ)의 MGF와 정확히 일치한다. ∎

## n=2 특수 경우

```
Y = X₁ + X₂,  X₁, X₂ ~ Exp(λ) 독립
Y ~ Γ(2, λ) = Erlang(2, λ)

PDF: f_Y(y) = λ²y·e^{-λy},  y > 0
```

## 모델 적용

D, P ~ Uniform(0, 1) 독립일 때:

```
-ln(D) ~ Exp(1)     (역변환 정리)
-ln(P) ~ Exp(1)     (역변환 정리)
```

따라서:

```
-ln(D×P) = -ln(D) + (-ln(P))
         = (Exp(1)) + (Exp(1))    [독립]
         = Γ(2, 1)
```

## G의 분포

G ∝ D×P이므로:

```
-ln(G) ~ Γ(2, 1) + const
```

이는 α = 2가 D, P라는 **2개 변수**의 곱이라는 사실에서 수학적으로 결정됨을 의미한다.

## 수치 검증

| 항목 | 이론값 | 실측값 |
|------|--------|--------|
| α (형상 모수) | 2 | 2.03 |
| KS 검정 p-값 | — | 0.934 |

p = 0.934 ≫ 0.05이므로 Γ(2, 1) 귀무가설을 기각할 수 없다.
