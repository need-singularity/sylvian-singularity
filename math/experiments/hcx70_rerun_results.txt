## Interpretation

- Bottleneck dim = 21 = floor(64 * phi(6)/6) = floor(64 * 2/6)
- phi(6) = Euler totient = #{k : gcd(k,6)=1, 1<=k<=6} = #{1,5} = 2
- Bottleneck ratio = 2/6 = 0.3333 ≈ 1/3
- If Phi-bottleneck achieves similar loss with fewer params -> efficiency gain confirmed
- If loss degrades significantly -> bottleneck too narrow for this task
