# H-TOP-3: 매듭 불변량과 6

> **가설**: Trefoil 매듭(2,3)-torus knot의 불변량이 σ,τ를 반영한다.

## 배경 (R35에서 이미 관찰)
- Trefoil = T(2,3): 소인수 (2,3) = 6의 소인수!
- Alexander 다항식: Δ(t) = t-1+t⁻¹
- Jones 다항식: V(t) = -t⁻⁴+t⁻³+t⁻¹
- Trefoil → B₃ (braid group) → PSL(2,Z) → modular forms → Δ → Leech

## 대응

```
  매듭 이론              σ,τ 체계
  ──────────────────    ──────────
  T(2,3) = trefoil     6 = 2×3
  crossing number = 3   σ/τ = 3
  B₃ (braid group)     PSL(2,Z) = Z/2*Z/3
  bridge number = 2     φ = 2
  genus = 1             1 = unit
```

## 검증 방향
1. [ ] T(2,3) Jones 다항식 계수와 σ,τ 대응
2. [ ] 매듭 보수(knot complement) 부피 ≈ σ,τ 관련?
3. [ ] 가설 체인: trefoil → B₃ → PSL₂ → weight 12 → Δ → Leech
