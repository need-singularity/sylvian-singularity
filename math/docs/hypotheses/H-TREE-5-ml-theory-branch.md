# H-TREE-5: ML 이론 가지 — R(d)와 일반화 한계

> **가설**: 뉴럴넷의 차원 d에서 R(d) 값이 일반화 한계(generalization bound)에 등장한다.

## 아이디어
```
  PAC-Bayes bound: 일반화 오차 ≤ f(complexity, data)
  complexity = 모델 차원 d의 함수

  가설: R(d) = σ(d)φ(d)/(dτ(d))가 "산술적 복잡도"로서
  일반화 한계에 자연스럽게 등장?

  근거: R(d)는 d의 약수 구조를 반영
  → 가중치 행렬의 대칭성 ↔ d의 약수 ↔ R(d)
```

## 왜 새 가지인가
ML 이론에서 "왜 특정 차원이 잘 작동하는가?"는 미해결.
R(d) ≈ 1 (d=6 근처)에서 최적이라면, 산술함수가 ML을 안내.

## 파급력: ★★★★★ (ML이론 혁신 가능, 극투기적)
