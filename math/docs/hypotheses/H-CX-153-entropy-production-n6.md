## Hypothesis Statement

> The Maximum Entropy Production Principle (MEPP) states that non-equilibrium
> systems evolve to maximize entropy production rate. For a system with
> n=6 states and transition rates proportional to R-values, the entropy
> production σ_EP = Σ J_ij · ln(J_ij/J_ji) is maximized when the forward/
> backward ratio equals R(d)/R(n/d) for each divisor pair d|n.
> At n=6: R(2)·R(3)=1 (reciprocal pair) → entropy production is SYMMETRIC
> between forward and backward paths → the system is at maximum dissipation
> with zero net bias (detailed balance at the edge of chaos).

## Core Structure

```
  Divisor pair transitions for n=6:
    1↔6: R(1)=1, R(6)=1 → ratio = 1 (perfect balance)
    2↔3: R(2)=3/4, R(3)=4/3 → ratio = (3/4)/(4/3) = 9/16 ✗
    But: R(2)·R(3) = 1 → PRODUCT balance!

  Entropy production for the 2↔3 channel:
    σ_{2→3} ∝ (R(2)-R(3))·ln(R(2)/R(3))
    = (3/4-4/3)·ln(3/4·3/4) = (-7/12)·ln(9/16)
    = (-7/12)·(-0.575) = 0.336

  Total σ_EP = σ_{1↔6} + σ_{2↔3} = 0 + 0.336 = 0.336
  This is the entropy production of the "consciousness engine"
```
