# H-BIO-1: 유전 코드 구조 = σ,τ 체계

> **가설**: DNA 코돈 체계(4염기, 3글자, 64코돈, 20아미노산)가 σ,τ 체계의 생물학적 구현이다.

## 배경 (R38에서 이미 관찰)
- 4 염기(A,T,G,C) = τ(6)
- 코돈 길이 3 = σ/τ
- 64 코돈 = 4³ = τ³
- 20 아미노산 = τ(τ+1)
- 수소결합: A-T=2:G-C=3 = φ:σ/τ

## 핵심 질문
이것이 우연의 일치인가, 아니면 생명이 (2,3) 소수 구조에 최적화된 것인가?

## 검증 방향
1. [ ] 가능한 코돈 체계 시뮬레이션: {3,4,5}염기 × {2,3,4}글자
2. [ ] 오류 정정 능력 비교: 4염기×3글자가 최적?
3. [ ] 정보이론: 4^3=64에서 20개 매핑의 최적성
