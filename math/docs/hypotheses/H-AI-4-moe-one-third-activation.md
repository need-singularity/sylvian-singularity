# H-AI-4: MoE 최적 활성 비율 = 1/3 (메타부동점)

> **가설**: Mixture-of-Experts에서 최적 expert 활성 비율이 1/3 ≈ φ(6)/P₁ (메타부동점)이다.

## 배경
- σφ=nτ에서: φ/n = 2/6 = 1/3 = τ/σ
- 골든MoE: I = 1-활성비율 = 0.375 ≈ 1/e ≈ 0.368
- 1/3 = 0.333 vs 1/e = 0.368 — 근접하지만 다름
- 메타부동점: f(I)=0.7I+0.1 → I*=1/3

## 검증 방향
1. [ ] 골든MoE 학습 완료 후 활성 비율 sweep: 0.2, 0.25, 1/3, 0.375, 0.5
2. [ ] 각 비율에서 PPL 비교
3. [ ] 기존 MoE 문헌에서 최적 활성 비율 데이터 수집
