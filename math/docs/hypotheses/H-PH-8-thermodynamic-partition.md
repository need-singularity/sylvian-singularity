# H-PH-8: 약수 분배함수의 열역학적 구조

> **가설**: 자연수 n의 약수 집합 {d|n}을 에너지 준위로 해석하면
> 분배함수 Z_n(β) = Σ_{d|n} e^{-βd}가 정의되고,
> σ(n)과 τ(n)이 각각 고온극한(β→0)의 에너지 합과 상태수로 자연 복원된다.
> n=6(완전수)에서 엔트로피 S_6(0) = ln(4) = ln(σ/τ) + ln(4/3)이며,
> 골든존 폭 ln(4/3)은 최대 무질서와 평균 에너지 스케일의 **간극**이다.

## 배경

σφ=nτ 체계에서 n=6의 약수 {1, 2, 3, 6}은 산술적 구조의 핵심이다.
이 4개 약수를 **에너지 준위**로 재해석하면, 통계역학의 표준 도구인
분배함수(partition function)를 약수론에 적용할 수 있다.

- H-PH-7에서 R 스펙트럼의 산술적 엔트로피를 정의했다.
- 여기서는 **정통 통계역학** 형식으로 재구성한다.
- σ(n), τ(n), φ(n)이 열역학량으로 자연스럽게 등장하는지 검증한다.

핵심 수론 상수:
```
  n=6:  σ(6)=12, τ(6)=4, φ(6)=2, R(6)=σφ/(nτ)=1
  n=28: σ(28)=56, τ(28)=6, φ(28)=12, R(28)=1
```

## 분배함수 정의

```
  Z_n(β) = Σ_{d|n} e^{-βd}

  n=6:  Z_6(β)  = e^{-β} + e^{-2β} + e^{-3β} + e^{-6β}
  n=28: Z_28(β) = e^{-β} + e^{-2β} + e^{-4β} + e^{-7β} + e^{-14β} + e^{-28β}

  극한:
    β→0:  Z_n(0)  = τ(n)           (상태수 = 약수 개수)
    β→∞:  Z_n(β) → e^{-β}          (최소 약수 d=1만 생존)
```

```
  β=0:      U_6 = σ(6)/τ(6) = 12/4 = 3.0  (평균 약수)
```

## σ(n)의 고온극한 해석

```
  β=0에서:
    Z_n(0) = τ(n)                    (상태수)
    U_n(0) = Σd / τ(n) = σ(n)/τ(n)  (평균 에너지)

  따라서:
    σ(n) = τ(n) × U_n(0) = (상태수) × (평균 에너지)

  이것은 σ(n)의 열역학적 의미:
    약수합 = 고온극한에서의 총 에너지 기대값 × 축퇴도
```

완전수 조건 σ(n) = 2n에서:
```
  2n = τ(n) × U_n(0)
  U_n(0) = 2n/τ(n)

  n=6:  U_6(0)  = 12/4 = 3.0  = n/2
  n=28: U_28(0) = 56/6 = 9.33 = n/3

  일반 완전수: U_n(0) = 2n/τ(n)
```

## Lee-Yang 영점 분석

Z_6(β)의 영점을 z = e^{-β} 변수로 분석:

```
  Z_6 = z + z² + z³ + z⁶ = z(1 + z + z² + z⁵)

  1 + z + z² + z⁵ = 0 의 근:
    z₀ = 0.8518 + 0.9113i   |z| = 1.2474
    z₁ = 0.8518 - 0.9113i   |z| = 1.2474
    z₂ = -1.0000             |z| = 1.0000   ← 단위원 위!
    z₃ = -0.3518 + 0.7203i  |z| = 0.8017
    z₄ = -0.3518 - 0.7203i  |z| = 0.8017
```

- **실수 양의 근이 없다** → 실수 β축 위에 상전이(phase transition)가 없음.
- z₂ = -1은 β = iπ에 대응 — 허수 온도에서의 반강자성적 영점.
- n=28에서도 실수 양의 근이 0개 → 완전수에서는 실수축 상전이가 없다.

이것은 **Lee-Yang 정리**와 일관: 페르마 상호작용이 없는 비상호작용 시스템은
실수축 상전이가 없다. 약수는 서로 "상호작용"하지 않으므로 당연한 결과.

## 골든존 폭의 열역학적 의미

핵심 등식:
```
  S_6(0) = ln(τ) = ln(4) = ln(3) + ln(4/3)
                          = ln(σ/τ) + 골든존 폭

  즉: 최대_엔트로피 = ln(평균_약수) + 골든존_폭
```

이 분해는 산술적으로 자명(ln(4) = ln(4))이지만 의미론적으로 비자명:

| 항 | 값 | 의미 |
|---|---|---|
| S_6(0) = ln(4) | 1.3863 | 4약수 균등분포의 최대 무질서 |
| ln(σ/τ) = ln(3) | 1.0986 | 평균 에너지 스케일의 로그 |
| ln(4/3) | 0.2877 | **무질서 초과분** = 골든존 폭 |

**골든존 폭 = 최대 엔트로피에서 에너지 스케일을 뺀 잔여 무질서**.
이것이 "3→4 상태 엔트로피 점프"와 동치인 이유:
- 3개 약수(에너지 스케일) → 4개 약수(전체 상태)로 갈 때의 정보 증가분.

## 한계

1. **비상호작용 모델**: 약수들이 서로 독립적으로 점유 — 실제 물리와 달리
   약수 간 상관(예: d₁×d₂=n)을 반영하지 않는다.
2. **Lee-Yang 영점 없음**: 상전이가 없으므로 "임계 현상" 해석에 한계.
3. **ln(4) = ln(3) + ln(4/3)** 분해는 산술적으로 항등식이므로
   물리적 의미 부여에 주의 필요.
4. β*에서의 특수값(U=φ)이 다른 완전수(n=28)에서도 유의미한지 미검증.
5. 약수를 에너지 준위로 해석하는 것 자체가 물리적 정당화 없는 유비(analogy).
