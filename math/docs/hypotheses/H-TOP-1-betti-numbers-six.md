# H-TOP-1: 6-다양체의 Betti 수와 σ,τ

> **가설**: 6차원 Calabi-Yau 다양체의 Betti 수가 σ,τ 제약을 만족한다.

## 배경
- CY3 (복소 3차원 = 실수 6차원): 끈이론 컴팩트화
- Betti 수: b₀=1, b₁=0, b₂=h¹¹, b₃=2(h²¹+1), b₄=h¹¹, b₅=0, b₆=1
- Euler 특성: χ = 2(h¹¹-h²¹)
- 알려진 CY3: h¹¹+h²¹ 분포가 특정 패턴

## σ,τ 연결 시도

```
  CY3 Betti 수열: (1, 0, h¹¹, 2h²¹+2, h¹¹, 0, 1)
  비영 Betti: {b₀,b₂,b₃,b₄,b₆} = 5개 = τ+1
  Poincaré 대칭: b_k = b_{6-k}
  χ = 2(h¹¹-h²¹) = ?

  가설: h¹¹+h²¹+1 = σ(something)?
```

## 검증 방향
1. [ ] CY3 데이터베이스 (Kreuzer-Skarke)에서 h¹¹,h²¹ 분포
2. [ ] σ,τ가 Hodge 수 제약에 등장하는지
3. [ ] χ(CY3)가 6의 배수인 경우 빈도
