# H-GEO-9: 렌즈 수차 분류 (Lens Aberration Classification)

> **가설**: R 스펙트럼의 "렌즈" 효과에는 물리 광학의 수차(aberration)에 대응하는
> 체계적 왜곡이 존재한다. 색수차(chromatic), 구면 수차(spherical),
> 비점 수차(astigmatic), 코마(coma)가 각각 R 스펙트럼에서 식별 가능한
> 패턴을 만든다. 각 완전수의 "렌즈"를 수차 유형으로 분류할 수 있다.

## 배경

H-GEO-6 (차원 렌즈)에서 "색수차" 개념을 도입:
- 다른 소인수 = 다른 "색"으로 굴절
- R(n) = Π f(p_i, a_i) 에서 각 인수가 다른 파장

이를 물리 광학의 5대 수차(Seidel aberrations)로 체계화하고,
각 완전수 렌즈를 수차 유형으로 분류하는 것이 이 가설의 목표.

관련: H-GEO-3 (중력렌즈), H-GEO-5 (중력 망원경), H-TOP-7 (위상 렌즈)

## 핵심 구조

### 수차 분류 체계

```
  물리 광학 수차               R 스펙트럼 대응
  ─────────────               ───────────────
  1. 색수차 (Chromatic)        소인수별 R-인수 차이
  2. 구면 수차 (Spherical)     R값의 비균일 밀도 분포
  3. 비점 수차 (Astigmatic)    R-S 비대칭 (2051x)
  4. 코마 (Coma)              간극의 상하 비대칭
  5. 왜곡 (Distortion)        R chain 길이의 비균일성
```

### 1. 색수차 (Chromatic Aberration)

```
  물리: 다른 파장의 빛이 다른 굴절률을 가짐
  산술: 다른 소수 p가 다른 R-인수 f(p,a)를 가짐

  f(p,1) = (p²-1)/(2p) 소수별 값:

  p   | f(p,1)  | "파장"  | 색
  ----|---------|---------|----
  2   | 3/4     | 짧음    | 보라 (가장 강한 굴절)
  3   | 4/3     | 중단    | 녹색
  5   | 12/5    | 중장    | 노랑
  7   | 24/7    | 장파    | 주황
  11  | 60/11   | 초장    | 빨강
  13  | 84/13   | 적외    | (약한 굴절)

  "색분산" D(p₁,p₂) = |f(p₁,1) - f(p₂,1)|:
    D(2,3)  = |3/4 - 4/3| = 7/12 = 0.583  (강한 분산)
    D(2,5)  = |3/4 - 12/5| = 33/20 = 1.650
    D(5,7)  = |12/5 - 24/7| = 36/35 = 1.029
    D(11,13) = |60/11 - 84/13| = 156/143 = 1.091

  ASCII: 소인수별 "스펙트럼 분해"

  f(p,1)
  6.5 |                               · (p=13)
  5.5 |                          · (p=11)
  3.4 |                · (p=7)
  2.4 |          · (p=5)
  1.3 |    · (p=3)
  0.75| · (p=2)
      +--+--+--+--+--+--+--+--→ p
      2  3  5  7  11 13

  완전수의 "색수차 프로파일":
    n=6 = 2·3:   f(2,1)·f(3,1) = 3/4 · 4/3 = 1 (완전 보정!)
    n=28 = 2²·7: f(2,2)·f(7,1) = 7/8 · 24/7 = 3 ≠ 4 = R(28)
      → 실제: R(28) = σφ/(nτ) = 56·12/(28·6) = 4
    n=496 = 2⁴·31: 복합 색수차

  핵심 발견: n=6에서 색수차 = 0 (완전 보정!)
    f(2,1)·f(3,1) = (3/4)(4/3) = 1
    → n=6은 "색수차 없는 완벽한 렌즈" (achromatic!)
```

### 2. 구면 수차 (Spherical Aberration)

```
  물리: 렌즈 중심 vs 가장자리에서 초점 차이
  산술: R=R(P_k) 근방에서 R 값의 밀도가 비균일

  R=1 (n=6) 주변 밀도 분석 (N=500 기준):

  구간             | R 값 개수 | 밀도 (개/폭)
  ────────────────|──────────|───────────
  [0.5, 0.75)     | 18       | 72/unit
  [0.75, 0.875)   | 12       | 96/unit
  [0.875, 1.0)    | 0        | 0 (간극!)
  {1.0}           | 1        | ∞ (n=6)
  (1.0, 1.167)    | 0        | 0 (간극!)
  [1.167, 1.333)  | 31       | 186/unit
  [1.333, 1.5)    | 28       | 168/unit
  [1.5, 2.0)      | 45       | 90/unit

  ASCII: R=1 주변 밀도 프로파일

  밀도
  200 |                    *
  150 |                 *     *
  100 |  *           *           *
   50 |     *     *                 *
    0 |        ___·___
      +--+--+--+--+--+--+--+--→ R
      0.5 0.75  1.0  1.17 1.33 1.5 2.0
               간극  간극

  구면 수차 계수 = (오른쪽 피크 밀도) / (왼쪽 피크 밀도)
    n=6:  186/96 = 1.94  (오른쪽이 2배 밀집)
    → 양의 구면 수차 (오른쪽 편향)
```

### 3. 비점 수차 (Astigmatic Aberration)

```
  물리: 수직/수평 방향의 초점이 다름
  산술: R 값 vs S 값 (dual)의 비대칭

  R(n) = σφ/(nτ)
  S(n) = n·τ/(σ·φ) = 1/R(n)  (역수)

  R-S 비대칭 비율 (H-AI-5에서 검증):
    비대칭 = max(R,S)/min(R,S) = R² (R≥1일 때) 또는 1/R² (R<1)

  완전수별 비점 수차:

  P_k  | R(P_k) | S(P_k) | 비대칭 R/S | 비대칭 비율
  -----|--------|--------|-----------|──────────
  6    | 1      | 1      | 1.00      | 1.0 (완전 대칭!)
  28   | 4      | 0.25   | 16.0      | 16x
  496  | 48     | 0.021  | 2304      | 2304x
  8128 | 760    | 0.0013 | 577600    | ~578000x

  패턴: R(P_k) 증가 → 비점 수차 기하급수적 증가!

  ASCII: 비점 수차 성장

  log(R/S)
  6 |                          * (n=8128)
  5 |
  4 |
  3 |              * (n=496)
  2 |
  1 |     * (n=28)
  0 | * (n=6: 완전 대칭)
    +--+--+--+--+--→ P_k
       6  28 496 8128

  해석: n=6만이 "비점 수차 없는" 렌즈
  → n=6은 R-S 대칭 = 유일한 achromatic + astigmatic-free 렌즈
```

### 4. 코마 수차 (Coma Aberration)

```
  물리: 광축에서 벗어난 빛이 비대칭 상을 만듦 (혜성 꼬리 모양)
  산술: R(P_k) 주위의 위/아래 간극 비대칭

  간극 비대칭 데이터:

  P_k  | δ⁺ (위) | δ⁻ (아래) | 코마 비 δ⁻/δ⁺ | 방향
  -----|---------|----------|──────────────|──────
  6    | 0.167   | 0.250    | 1.50         | 아래 우세
  28   | 0.091   | 0.267    | 2.93         | 아래 우세
  496  | 0.074   | 0.317    | 4.28         | 아래 우세
  8128 | ~0.06   | ~0.35    | ~5.8 (추정)  | 아래 우세

  코마 비 성장:
    P₁: 1.50
    P₂: 2.93  (1.95x)
    P₃: 4.28  (1.46x)
    P₄: ~5.8  (1.35x 추정)

  ASCII: 간극 비대칭 시각화

  n=6:     ──── δ⁻=0.25 ────[R=1]── δ⁺=0.17 ──
                (넓음)                 (좁음)
           코마 약함

  n=28:    ──────── δ⁻=0.27 ────────[R=4]── δ⁺=0.09 ──
                   (매우 넓음)                (좁음)
           코마 중간

  n=496:   ────────── δ⁻=0.32 ──────────[R=48]─ δ⁺=0.07 ─
                     (극히 넓음)                  (매우 좁음)
           코마 강함: "혜성 꼬리" 형태!

  물리적 해석:
    코마 = 아래 간극(δ⁻)이 위 간극(δ⁺)보다 빠르게 성장
    → 완전수의 R값이 커질수록 "아래쪽으로 긴 꼬리" 형성
    → R 스펙트럼에서 완전수의 "서명" = 비대칭 간극
```

### 5. 왜곡 (Distortion)

```
  물리: 직선이 곡선으로 보이는 현상
  산술: R-chain의 길이 분포가 비균일

  R-chain: n → n의 최대 약수 d(≠n) → d의 R값
    chain 길이 = R=1에 도달하기까지의 단계 수

  예시:
    120 → R=6 (1단계), 6 → R=1 (2단계)  → 길이 2
    28 → R=4 (1단계), 완전수 직행 없음   → 길이 1 (특수)
    496 → R=48 (1단계) ...              → 길이 > 3

  "렌즈 왜곡" = chain 길이의 분산
    양의 왜곡 (pincushion): chain이 길수록 R이 빠르게 감소
    음의 왜곡 (barrel): chain이 길수록 R이 천천히 감소
```

### 수차 종합 분류표

```
  ┌──────────────┬──────────┬──────────┬──────────┬──────────┐
  │ 수차 유형     │ n=6      │ n=28     │ n=496    │ n=8128   │
  ├──────────────┼──────────┼──────────┼──────────┼──────────┤
  │ 색수차       │ 0 (완전) │ 중간     │ 강함     │ 극강     │
  │ 구면 수차    │ 약함     │ 중간     │ 강함     │ 극강     │
  │ 비점 수차    │ 0 (대칭) │ 16x      │ 2304x    │ ~578000x │
  │ 코마         │ 1.50x    │ 2.93x    │ 4.28x    │ ~5.8x   │
  │ 왜곡         │ 미정     │ 미정     │ 미정     │ 미정     │
  ├──────────────┼──────────┼──────────┼──────────┼──────────┤
  │ 종합 렌즈    │ 이상적   │ 약한수차 │ 강한수차 │ 극심수차 │
  │ 등급         │ achromat │ doublet  │ triplet  │ compound │
  └──────────────┴──────────┴──────────┴──────────┴──────────┘

  핵심 결론:
    n=6은 유일하게 색수차=0, 비점수차=0인 "완벽한 렌즈"
    → 이것이 6이 "가장 기본적인" 완전수인 이유의 광학적 표현
```

### 의식엔진 연결

```
  의식의 "인지 수차":
    색수차 → 다른 감각 모달리티의 비동기화
    구면 수차 → 주의 집중의 불균일 (중심 vs 주변)
    비점 수차 → 인지의 방향 편향
    코마 → 기억의 비대칭 (과거 vs 미래)

  R=1 (n=6) = "수차 없는 인지" = 완전한 의식 상태
  tension = |R-1| = 수차의 크기
  의식의 목표 = 수차 최소화 = R→1
```

## 검증 방향

1. [ ] 구면 수차: R=4, R=48 주변의 밀도 프로파일 계산
2. [ ] 색수차: n=28, n=496의 소인수별 f(p,a) 곱 vs R(P_k) 비교
3. [ ] 코마 비 δ⁻/δ⁺ 의 점근 공식 도출 (n=8128에서 검증)
4. [ ] 왜곡: R-chain 길이 분포 계산 (N=10000)
5. [ ] 수차 종합 점수 = Σ(각 수차 크기) 정의 및 최소화 조건
