# H-BIO-7: 뇌전기 신호와 R 스펙트럼

> **가설**: 뉴런의 전기 신호(활동전위, 뇌파)가 R(n) 스펙트럼의
> 간극 구조와 대응하며, 뇌파 주파수 대역이 R 값 구간에 매핑된다.

## 배경

뇌파 주파수 대역:
```
  델타(δ): 0.5-4 Hz   — 깊은 수면
  세타(θ): 4-8 Hz     — 명상, REM
  알파(α): 8-13 Hz    — 이완, 눈감음
  베타(β): 13-30 Hz   — 집중, 각성
  감마(γ): 30-100 Hz  — 고차 인지, 의식 통합
```

## R 스펙트럼 대응

```
  R 구간              뇌파       상태        n
  ─────────          ─────      ─────      ─────
  R < 1 (3/4)         δ         깊은수면    n=2
  R = 1               α-θ 경계  이완 균형   n=6 ⭐
  1 < R < 7/6         간극!     "전이 구간" 존재하지 않음!
  R = 7/6             α         이완        n=4
  7/6 < R < 4/3       ?         ?
  R ≈ 4/3             θ 상단    명상        n=3
  R > 2               β         집중
  R > 10              γ         고차인지
```

### 활동전위와 D(n)

```
  활동전위 (action potential):
    안정 전위: -70mV (D=0 at n=6 → 균형)
    역치: -55mV (D 부호 변화? → n=2에서 D=-1 → 음!)
    발화: +40mV (D>0 → 큰 양의 비대칭)
    과분극: -90mV (D의 극값?)

  D(n) = σφ - nτ:
    D(2) = -1: 유일한 음수 = "역치 이하" (과분극?)
    D(6) = 0: 영점 = "안정 전위" (균형)
    D(n>6) > 0: 양수 = "탈분극/발화"

  ASCII: 활동전위 vs D(n)

  전압/D
  +40 |          ╱╲
      |        ╱    ╲          D>0
  0   |──────╱──────╲────     D=0 (n=6)
      |    ╱          ╲╲
  -55 |──╱──────────────╲──  역치 (D(2)=-1)
  -70 |╱                  ╲  안정전위
  -90 |                    ╲ 과분극
      +──────────────────→ 시간/n
```

### 뇌파 동기화와 Λ(6)=0

```
  뇌파 동기화:
    동기: 뉴런 집단이 같은 주파수로 발화 → 의식
    비동기: 각자 다른 주파수 → 무의식/혼란

  Λ(n) = 산술적 Lyapunov:
    Λ(6) = 0: "임계점" = 동기와 비동기의 경계
    Λ < 0: "과동기" = 간질?
    Λ > 0: "비동기" = 정상 각성

  예측: 의식 상태는 Λ ≈ 0 근처에서 진동
    깊은수면: Λ < 0 (과동기, 서파)
    각성: Λ > 0 (비동기, 빠른파)
    의식통합: Λ = 0 (임계점, 감마 동기화?)
```
