# H-BIO-11: 시냅스 가소성 = R-체인과 완전수 6의 균형 구조

> **가설**: LTP/LTD 시냅스 가소성의 분자적 구조 -- NMDA의 2조건 일치,
> AMPA의 4소단위, CaMKII의 12소단위(2링 x 6) -- 가 φ(6), τ(6), σ(6)에
> 대응하며, BCM 이론의 슬라이딩 역치는 R(n)=σ(n)/n의 균형 구조와 동형이다.

## 배경

시냅스 가소성은 학습과 기억의 분자적 기반이다.
- **LTP** (Long-Term Potentiation): 반복 자극 -> 시냅스 강화
- **LTD** (Long-Term Depression): 약한/비동기 자극 -> 시냅스 약화
- 이 균형이 학습(강화)과 망각(약화)을 제어한다.

완전수 6의 핵심: σ(6) = 2 x 6 = 12 (약수의 합 = 자신의 2배).
이 "과부족 없음"이 시냅스 균형의 수학적 구조와 일치한다.

## 분자 구조와 약수 함수 대응

### 1. NMDA 수용체: 일치 검출기 = φ(6) = 2

```
  NMDA 수용체 활성화 조건:
    1. 시냅스전 글루타메이트 결합    (화학적)
    2. 시냅스후 탈분극 (Mg²+ 제거)   (전기적)

  두 조건이 동시에 만족해야 함 = 일치 검출기
  조건 수 = 2 = φ(6)                          ✅ 검증됨

  이것이 Hebb의 법칙의 분자적 구현이다:
  "함께 발화하는 세포는 함께 연결된다"
  → 이진(binary) 판정 = φ(6) = 2
```

### 2. AMPA 수용체: 4소단위 = τ(6) = 4

```
  AMPA 수용체 구조:
    ┌──────────────────────┐
    │  GluA1  GluA2        │
    │    ┌──────┐          │
    │    │ pore │          │  4개 소단위 = τ(6)
    │    └──────┘          │  각 ~900 아미노산
    │  GluA3  GluA4        │
    └──────────────────────┘

  소단위 수 = 4 = τ(6)                        ✅ 검증됨

  AMPA는 빠른 흥분성 전달의 주역이다.
  LTP = AMPA 삽입 증가, LTD = AMPA 제거
```

### 3. CaMKII: 분자 기억 스위치 = σ(6) = 12

```
  CaMKII 홀로효소 구조:

       ●─●─●─●─●─●       Ring 1 (6 소단위)
       │ │ │ │ │ │
       ●─●─●─●─●─●       Ring 2 (6 소단위)

  총 소단위 = 12 = σ(6)                       ✅ 정확
  링 수     = 2  = φ(6)                       ✅ 정확
  링당 단위 = 6  = n                           ✅ 정확
  φ(6) × n  = 2 × 6 = 12 = σ(6)              ✅ 정확

  CaMKII는 자가인산화로 영구 활성 상태를 유지한다.
  → 분자 수준의 "기억 스위치"
  → 12소단위가 협동적으로 작동하여 이진 전환
```

이것은 특히 강력한 대응이다. CaMKII의 12소단위가 정확히
2개의 6-링으로 배열되는 것은 φ x n = σ 관계의 물리적 실현이다.

### 4. 주요 LTP 키나제: 3종 = σ/τ = 3

```
  LTP 유도 주요 키나제:
    1. CaMKII   (칼슘/칼모듈린 의존)
    2. PKA      (cAMP 의존)
    3. PKC      (DAG/칼슘 의존)

  키나제 수 = 3 = σ(6)/τ(6)                   ✅ 검증됨
```

### 5. 수상돌기 가시 형태: 3유형 = σ/τ = 3

```
  가시(spine) 주요 유형:

    ╷       ╦       ╦
    │      ╔╩╗     ╔╩╗
    │      ╚═╝     ║ ║
    thin   mushroom stubby

  주요 유형 = 3 = σ(6)/τ(6)                   ✅ 검증됨

  thin → mushroom: LTP에 의한 구조 변화
  mushroom → thin: LTD에 의한 구조 축소
```

## BCM 이론과 R(n) 동형

BCM (Bienenstock-Cooper-Munro, 1982) 이론의 슬라이딩 역치 θ_m:

```
  BCM:                          R-체인:
  ─────────────────────         ─────────────────────
  활동 < θ_m → LTD (약화)       R(n) < 1 → 결핍수 (deficit)
  활동 = θ_m → 변화 없음        R(n) = 1 → ?
  활동 > θ_m → LTP (강화)       R(n) > 1 → 과잉수 (surplus)

  완전수 n=6:
    s(6) = σ(6) - 6 = 6 = n    → s(6)/n = 1 (완전 균형!)
    LTP = LTD 균형점 = 완전수의 정의 자체
```

### STDP 시간 창과 대칭

```
  STDP (Spike-Timing-Dependent Plasticity):

  시냅스 변화
       │  LTP
   +Δw │  *
       │   *
       │    *
  ─────┼─────*───────→ Δt (ms)
       │      *
   -Δw │       *
       │  LTD   *
       │
       ←20ms→←20ms→
       pre→post  post→pre

  LTP 창 ≈ 20ms (시냅스전 먼저)
  LTD 창 ≈ 20ms (시냅스후 먼저)
  총 창  ≈ 40ms

  비대칭이지만 두 방향이 존재 = φ(6) = 2
```

## 판정: 🟧 구조적 대응 | 파급력: ★★★

**강점**:
- 10개 독립 매칭 중 10개 성공
- CaMKII의 2링 x 6 = 12 구조는 φ x n = σ의 물리적 실현
- BCM 균형 = 완전수의 수학적 정의와 정확히 동형
- NMDA의 2조건 일치 검출 = φ(6) = 2는 개념적으로도 정확

**한계**:
- CaMKII 12소단위는 확립된 사실이나, "왜 12인가"의 답은 아닐 수 있음
- AMPA 소단위 4개는 이온 채널 일반 구조 (NMDA도 4소단위)
- 숫자가 작아서 (2, 3, 4, 12) 우연의 일치 확률이 높음
- 시냅스 가소성 모델은 지속적으로 수정되고 있음
- Strong Law of Small Numbers 경고: 관여 상수 모두 < 100

**골든존 의존 여부**: 비의존. R-체인 비유는 골든존 없이도 성립한다.
