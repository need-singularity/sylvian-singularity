# H-PH-7: R 스펙트럼의 산술적 엔트로피

> **가설**: R(n)의 약수별 분해가 "산술적 엔트로피"를 정의하며,
> R=1(n=6)에서 엔트로피가 극대 또는 극소가 된다.

## 핵심

```
  산술 엔트로피 정의:
    H_R(n) = -Σ_{p^a||n} [f(p,a)/R(n)] · ln[f(p,a)/R(n)]
    (R의 소인수별 기여를 확률로 해석)

  n=6: R=1, f(2,1)=3/4, f(3,1)=4/3
    p_1 = (3/4)/1 = 0.75, p_2 = (4/3)/1 = 1.333 (합>1, 정규화 필요)

  정규화 엔트로피:
    H(n) = -Σ (f_i/Σf_j) · ln(f_i/Σf_j) (Shannon)

  n=6: f_sum = 3/4+4/3 = 25/12
    w_1 = (3/4)/(25/12) = 9/25 = 0.36
    w_2 = (4/3)/(25/12) = 16/25 = 0.64
    H = -0.36·ln(0.36) - 0.64·ln(0.64)
      = -0.36·(-1.02) - 0.64·(-0.45)
      = 0.367 + 0.288 = 0.655

  n=10=2·5: f(2,1)=3/4, f(5,1)=12/5
    f_sum = 3/4+12/5 = 63/20
    w_1 = (3/4)/(63/20) = 15/63 = 5/21 ≈ 0.238
    w_2 = (12/5)/(63/20) = 48/63 = 16/21 ≈ 0.762
    H ≈ -0.238·(-1.44) - 0.762·(-0.272) ≈ 0.343+0.207 = 0.550

  n=6의 H ≈ 0.655 > n=10의 H ≈ 0.550
  → n=6이 더 "균등한" 분배! (0.36:0.64 vs 0.24:0.76)
```

### 골든존 폭과 엔트로피

```
  골든존 폭 = ln(4/3) = 3→4 상태 엔트로피 점프
  = ln(f(3,1)) = σφ=nτ 텔레스코핑의 핵심 인수의 로그!

  H(n=6) ≈ 0.655 ≈ ln(2)·... ?
  실제로 H(n=6)를 정확히 계산하면:
    H = -(9/25)ln(9/25) - (16/25)ln(16/25)
    = (9/25)(ln25-ln9) + (16/25)(ln25-ln16)
    = (9/25)·ln(25/9) + (16/25)·ln(25/16)
```

### Boltzmann과 R

```
  S = k_B · ln(Ω)  (Boltzmann 엔트로피)

  "산술 미시상태": n의 약수 구조가 결정
    Ω(n) = τ(n) (약수의 수 = 미시상태 수?)
    S(n) = ln(τ(n))

  n=6: S = ln(4) = 2ln(2) ≈ 1.386
  n=28: S = ln(6) ≈ 1.792
  n=496: S = ln(10) ≈ 2.303

  "열역학적" 관점:
    R(n) = "자유에너지" / "내부에너지"
    R=1: 자유에너지 = 내부에너지 = 열역학 평형!
```
