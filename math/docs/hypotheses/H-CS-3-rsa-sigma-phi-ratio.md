# H-CS-3: σφ/(nτ) 비율과 RSA 보안

> **가설**: n=pq에서 σφ/(nτ)가 1에서 멀수록 소인수분해가 어렵고, RSA 보안이 강하다.

## 배경
- RSA: n=pq, 보안 = φ(n)=(p-1)(q-1) 계산 어려움
- n=6: σφ/(nτ)=1 (균형, 쉬움)
- 큰 pq: σφ/(nτ)≫1 (불균형, 어려움)

## σφ/(nτ) for n=pq
```
  n=pq: σφ/(nτ) = (p²-1)(q²-1)/(4pq)
  p,q 클수록 → ≈ pq/4 → n/4 (선형 성장)
```
