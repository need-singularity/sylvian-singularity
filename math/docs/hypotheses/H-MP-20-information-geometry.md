# H-MP-20: 약수 분포의 정보 기하학

> **가설**: 각 자연수 n은 약수 위의 확률분포 p_n(d) = (1/d) / sigma_{-1}(n) 을 정의하며,
> 이 분포들의 통계적 다양체(statistical manifold)에서 완전수는 구별되는 기하학적 성질을 갖는다.
> 특히 완전수 간 Jensen-Shannon 거리가 정확히 ln(2)/4 로 고정됨을 보인다.

## 배경

sigma-phi 프로젝트의 핵심 비율 R(n) = sigma(n) * phi(n) / (n * tau(n)) 에서
완전수 n=6은 R(6)=1인 유일한 자연수다 (H-MP-10, H-MP-12 참조).

약수의 역수합 sigma_{-1}(n) = sum_{d|n} 1/d 는 완전수에서 정확히 2가 된다.
이 성질을 이용하면 각 자연수의 약수 위에 자연스러운 확률분포를 정의할 수 있다:

```
  p_n(d) = (1/d) / sigma_{-1}(n)    (d | n)
```

완전수 n에 대해 sigma_{-1}(n) = 2 이므로 p_n(d) = 1/(2d).

이 분포는 "작은 약수에 높은 확률"을 부여하는 자연스러운 가중치로,
정보 기하학(Information Geometry)의 도구를 적용하면 정수론적 구조가
기하학적 언어로 번역된다.

## 1. 통계적 다양체 M_n

각 자연수 n은 tau(n)개 약수 위의 확률분포 p_n 을 정의한다.
이는 (tau(n)-1)-단체(simplex) 위의 한 점이다.

## 2. Fisher 정보 계량 (Fisher Information Metric)

범주형 분포의 Fisher 정보 행렬은 대각행렬이다:

```
  g_{ij} = delta_{ij} / p_n(d_i)
```

### n=6에서의 Fisher 계량

| 약수 d | p_6(d) | g(d) = 1/p_6(d) |
|--------|--------|------------------|
| 1      | 1/2    | 2                |
| 2      | 1/4    | 4                |
| 3      | 1/6    | 6                |
| 6      | 1/12   | 12               |

**완전수의 Fisher 대각선**: g(d) = 1/p_n(d) = 2d

### Fisher 계량의 불변량

| n    | tr(g)  | det(g)    | tr(g)/tau(n) |
|------|--------|-----------|--------------|
| 6    | 24     | 576       | 6.00         |
| 28   | 112    | 1,404,928 | 18.67        |
| 496  | 1,984  | 3.07e+16  | 198.40       |

**정리 (Fisher trace 공식)**:
완전수 n에 대해:

```
  tr(g_n) = sum_{d|n} 2d = 2 * sigma(n) = 2 * 2n = 4n
```

증명: sigma(n) = 2n (완전수 정의) 이고 g(d) = 2d 이므로
tr(g) = sum_{d|n} 2d = 2 * sigma(n) = 4n. QED.

**정리 (Fisher determinant 공식)**:

```
  det(g_n) = prod_{d|n} 2d = 2^{tau(n)} * prod_{d|n} d = 2^{tau(n)} * n^{tau(n)/2}
```

마지막 등식은 prod_{d|n} d = n^{tau(n)/2} (잘 알려진 정수론 결과).

## 3. Jensen-Shannon 발산 (JSD)

KL 발산 D_KL(p_6 || p_28) 은 지지(support)가 다르므로 무한대이다:
- supp(p_6) = {1, 2, 3, 6}
- supp(p_28) = {1, 2, 4, 7, 14, 28}
- supp(p_6) 가 supp(p_28) 에 포함되지 않음 (3, 6 이 없음)

대신 Jensen-Shannon 발산(JSD)을 사용한다.

### JSD 분해 구조

완전수 6과 28의 JSD를 분해하면:

| 성분         | 약수          | 기여      |
|--------------|---------------|-----------|
| 공통 {1,2}   | p=q 이므로    | 0.000000  |
| 6만의 {3,6}  | 질량 = 1/4    | 0.086643  |
| 28만의 나머지 | 질량 = 1/4    | 0.086643  |
| **합계**     |               | **0.173287** |

### 핵심 정리: 완전수 간 JSD 고정값

> **정리**: 공통 약수 {1,2}만 공유하는 두 짝수 완전수 m, n에 대해
> JSD(p_m, p_n) = ln(2)/4 = 0.173287...

**증명**:

짝수 완전수 m, n에 대해 p_m(1) = p_n(1) = 1/2, p_m(2) = p_n(2) = 1/4.
공통 약수에서 p = q 이므로 JSD 기여는 0.

m만의 약수 질량 = 1 - 3/4 = 1/4.
n만의 약수 질량 = 1 - 3/4 = 1/4.

비공통 약수 d에서 혼합분포 m(d) = p(d)/2 이므로:
- p(d) * ln(p(d)/m(d)) = p(d) * ln(2)

따라서 각 비공통 부분의 기여 = (1/2) * (1/4) * ln(2) = ln(2)/8.
양쪽 합: JSD = 2 * ln(2)/8 = **ln(2)/4**. QED.

### JSD 거리표

| 쌍             | JSD      | sqrt(JSD) | 비고                |
|----------------|----------|-----------|---------------------|
| (6, 28)        | 0.173287 | 0.416277  | ln(2)/4 (완전-완전) |
| (6, 496)       | 0.173287 | 0.416277  | ln(2)/4 (완전-완전) |
| (28, 496)      | 0.086643 | 0.294353  | ln(2)/8 (공통 더 많음) |
| (6, 12)        | 0.052260 | 0.228605  | 6의 배수, 가까움    |
| (6, 10)        | 0.145503 | 0.381449  | 비완전              |
| (6, 8)         | 0.156361 | 0.395425  | 비완전              |
| (6, 15)        | 0.177926 | 0.421813  | 비완전              |
| (6, 24)        | 0.074882 | 0.273647  | 6의 배수            |

```
  JSD(p_6, p_n)
  0.20 |           *        (15)
       |      *  *     * *  (28,496)
  0.15 |    *               (10)
       |   *                (8)
  0.10 |
```

**관찰**: JSD(6,28) = JSD(6,496) 이 정확히 일치하고, JSD(28,496) 은 정확히
그 절반이다. 이는 공통 약수 구조에 의한 것으로 우연이 아닌 구조적 결과다.

## 4. 곡률 (Curvature)

Fisher 계량을 가진 (k-1)-단체는 상수 단면곡률 kappa = 1/4 을 갖는
(잘 알려진 정보 기하학 결과, Amari & Nagaoka, 2000).

스칼라 곡률: S = (k-1)(k-2) * kappa / (적절한 조합수)

| n    | k=tau(n) | dim=k-1 | 스칼라 곡률 S = (k-1)(k-2)/4 |
|------|----------|---------|------------------------------|
| 6    | 4        | 3       | 1.50                         |
| 28   | 6        | 5       | 5.00                         |
| 496  | 10       | 9       | 18.00                        |
| 8128 | 14       | 13      | 39.00                        |

완전수 2^(p-1)(2^p - 1) 의 약수 개수는 tau = 2p 이므로:

```
  S(n) = (2p-1)(2p-2)/4 = (2p-1)(p-1)/2
```

p=2 -> S=1.5, p=3 -> S=5, p=5 -> S=18, p=7 -> S=39.

## 5. R(n)=1 의 기하학적 의미

R(n) = sigma(n)*phi(n)/(n*tau(n)) = 1 은 n=6에서만 성립한다.

정보 기하학적으로 재해석하면:

```
  R(n) = sigma(n) * phi(n) / (n * tau(n))
       = [tr(g_n)/2] * phi(n) / (n * tau(n))     (완전수: tr(g)=2*sigma)
       = tr(g_6) * phi(6) / (2 * 6 * 4)
       = 24 * 2 / 48 = 1
```

R(n)=1 은 Fisher trace 와 Euler totient 의 균형 조건:

```
  tr(g_n) * phi(n) = 2 * n * tau(n)
```

n=6에서: 24 * 2 = 2 * 6 * 4 = 48. 성립.
n=28에서: 112 * 12 = 1344, 2 * 28 * 6 = 336. 1344/336 = 4. R(28)=4.

## 6. 측지선 (Geodesics)

완전수들이 통계적 다양체의 측지선 위에 놓이는가?

각 완전수는 서로 다른 차원의 단체에 살고 있다 (tau(6)=4, tau(28)=6, ...).
따라서 동일 다양체 위의 측지선 개념은 직접 적용되지 않는다.

그러나 **공통 약수 {1,2} 위의 사영(projection)**을 통해
모든 짝수 완전수를 2-단체 위에 사영할 수 있다:

```
  pi(p_n) = (p_n(1), p_n(2), 1 - p_n(1) - p_n(2)) = (1/2, 1/4, 1/4)
```

**모든 짝수 완전수가 사영 후 동일한 점에 수렴한다!**

이는 완전수의 "정보 기하학적 부동점"이다:
측지선이 아닌, 단일 점으로의 수렴.

## 해석 및 의미

1. **sigma_{-1}(n) = 2 가 정보 기하학적 정규화 조건**: 완전수는 약수 역수합이
   정확히 2이므로, 1/d 가중치가 자연스러운 확률분포를 형성한다.

2. **JSD = ln(2)/4 정리**: 완전수 간 정보 거리가 보편 상수(ln2)로 고정된다.
   이는 완전수의 "정보론적 등간격" 성질이다.

3. **tr(g) = 4n**: Fisher 정보의 총량이 n에 선형 비례. 정보 용량이
   수의 크기에 정비례한다.

4. **사영 수렴**: 모든 짝수 완전수가 (1/2, 1/4, 1/4) 로 사영된다.
   완전수의 "보편적 정보 프로필"이 존재한다.

## 한계

1. 각 완전수가 다른 차원의 단체에 살므로 단일 다양체 위의 비교에 한계가 있다.
2. JSD = ln(2)/4 정리는 공통 약수가 {1,2}인 경우에만 적용 (28과 496은
   공통 약수가 {1,2,4}로 더 많아 JSD = ln(2)/8).
3. 홀수 완전수(미발견)에는 적용 불가 (2가 약수가 아니므로 구조가 다름).
4. 스칼라 곡률은 단체 자체의 성질이지, 분포 위치에 의존하지 않는다.
