# H-CX-2: 골든존 I∈[0.21, 0.50] = R-인수 범위

> **가설**: 골든존의 Inhibition 범위 [1/2-ln(4/3), 1/2]이 R-인수 f(p,a)의 값 범위와 대응된다.

## 배경
- 골든존 상한: 1/2 = 1/σ₋₁(6)
- 골든존 하한: 0.2123 ≈ 1/2-ln(4/3)
- R-인수: f(2,1)=3/4=0.75, f(3,1)=4/3=1.33
- f(2,1)=3/4가 유일한 1 미만 값 → 이것이 "억제"?

## 대응

```
  R-인수 체계             골든존
  ───────────────────    ──────────────────
  f(2,1) = 3/4 < 1      → 억제 (Inhibition)
  f(3,1) = 4/3 > 1      → 촉진 (Plasticity)
  f(2,1)×f(3,1) = 1     → 균형 = 골든존 중심

  3/4 = 0.75 ∈ [0.21, 1.00]?
  → 골든존은 [0.21, 0.50], f값은 [0.75, ∞)
  → 직접 대응 안 맞음. 변환 필요?
```

## 검증 방향
1. [ ] 1-f(p,a) 또는 1/f(p,a)로 변환 후 골든존 범위 매칭
2. [ ] MoE 활성 비율 = f(2,1)=3/4? (75% 활성 = I=0.25 ∈ 골든존!)

## 흥미로운 관찰
- f(2,1) = 3/4 → 활성 비율 해석 시 I = 1-3/4 = 1/4 = 0.25
- **0.25 ∈ [0.21, 0.50] = 골든존!**
- 골든MoE 실험의 I=0.375 ≈ 1/e와 비교

## 검증 결과 (2026-03-24)
- f(2,1)=3/4 → I=1-3/4=0.25 → **골든존 [0.21,0.50] 내!** ✅
- GZ 하단 13.1% 위치 (1/e=0.368에서 0.118 거리)
- **유일**: 12개 (p,a) 조합 중 GZ 내 값은 f(2,1)뿐
- n=6: 억제(3/4)×증폭(4/3)=1 → net I=0 (완전 상쇄)
- **부분 확인**: I=0.25 ∈ GZ이지만 1/e 최적과 거리 있음
