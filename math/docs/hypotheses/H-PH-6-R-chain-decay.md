# H-PH-6: R-체인과 입자 붕괴 연쇄

> **가설**: R-체인의 감소 구조가 입자 물리의 붕괴 연쇄(decay chain)와 구조적으로 유사.

## 대응
```
  R-체인: 193750 → 6048 → 120 → 6 → 1
  붕괴:   heavy → medium → light → stable → ground

  6 = 탄소 Z (안정 원소)
  1 = 수소 Z (최경량)
```

## 검증 방향
- 실제 붕괴 연쇄 (U-238 → Pb-206)의 단계 수와 비교
- 안정 원소(C,Fe 등)의 원자번호와 R 고정점 관계
