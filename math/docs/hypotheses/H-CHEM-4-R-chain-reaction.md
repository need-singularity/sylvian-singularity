# H-CHEM-4: R-체인과 화학 반응 연쇄

> **가설**: R-체인 n→R(n)→R²(n)→...→1의 구조가 화학 반응의 에너지 단계와 유사.

## 대응
```
  R-체인 단계        화학 유비
  ──────────        ────────
  n (시작점)         반응물
  R(n) (감소)        중간체 (활성화 에너지 하강)
  R(6)=1            생성물 (안정)
  R(1)=1            기저 상태
```

## 검증 방향
