# H-TREE-3: 확률론 가지 — R(n)의 분포 이론

> **가설**: 랜덤 n에서 R(n)의 분포가 특정 확률분포에 수렴하며, n=6의 R=1이 이 분포의 극단점이다.

## 아이디어
```
  X_N = R(uniform random n ∈ [2,N])

  현재 알려진 것:
  - E[R(n)] ~ cN (평균이 N에 비례, 즉 발산)
  - R(n)/n → 0 (상대값은 0으로)
  - P(R(n)=1) = 1/N → 0 (n=6만)

  미지:

```
