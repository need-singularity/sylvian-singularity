# H-MP-20: p-진 심층 해석 — R-인수의 p-adic 구조

> **가설**: R(n) = σ(n)φ(n)/(nτ(n))의 p-adic valuation v_p(R(n)-1)은
> n의 소인수 분해와 체계적으로 연결되며, R(n)≡1 (mod p^k)를
> 만족하는 n의 집합은 k→∞에서 {1, 6}으로 수렴한다.
> 이는 완전수 6의 p-adic 유일성을 모든 소수에서 동시에 확인하는 것이다.

## 배경

R(6) = σ(6)φ(6)/(6·τ(6)) = 12·2/(6·4) = 1 (정확). H-TREE-2에서
R(6)=1이 모든 p에 대해 p-adic 단원(unit)임을 확인했다.

이 가설은 더 깊이 들어간다:
- R(n)-1의 p-adic valuation 구조는 무엇인가?
- Hensel lifting으로 R≡1 (mod p^k)인 n을 추적하면 어떤 패턴이 나타나는가?
- R(n) 값들 사이의 초거리(ultrametric) 거리는 어떤 군집을 형성하는가?
- Bernoulli 수 B_2=1/6, Kummer 합동식과 연결이 있는가?

관련: R(n)은 곱셈적 분해 R(n) = Π_p f(p,a)를 가진다 (H-MP-16).
f(p,a) = (p^(a+1)-1)·(p-1) / (p·(a+1)·(p^a - p^(a-1)))를 통해
개별 소수 기여를 분석할 수 있다.

| 3 | 4 | 2 | 2 | 8 | 6 | 2 | 1.3333 | 1 | 0 | 0 | 0 |

## 2. v_p(R(n) - 1) 정밀 분석

R(n)-1의 p-adic valuation은 R(n)이 1에 "p-adic으로 얼마나 가까운가"를 측정한다.
v_p(R-1) > 0이면 R≡1 (mod p), v_p(R-1) < 0이면 R-1의 분모에 p가 있다.

| n | R(n) | v₂(R-1) | v₃(R-1) | v₅(R-1) | v₇(R-1) |
|--:|-----:|--------:|--------:|--------:|--------:|
| 1 | 1 | ∞ | ∞ | ∞ | ∞ |
| 2 | 3/4 | -2 | 0 | 0 | 0 |
| 3 | 4/3 | 0 | -1 | 0 | 0 |
| 6 | 1 | ∞ | ∞ | ∞ | ∞ |
| 10 | 9/5 | 2 | 0 | -1 | 0 |
| 11 | 60/11 | 0 | 0 | 0 | 2 |
| 22 | 45/11 | 1 | 0 | 0 | 0 |
| 26 | 63/13 | 1 | 0 | 2 | 0 |
| 36 | 91/27 | **6** | -3 | 0 | 0 |
| 38 | 135/19 | 2 | 0 | 0 | 0 |

### n=36의 특이성

```
  R(36) = 91/27
  R(36) - 1 = 64/27 = 2⁶/3³

  v₂(R(36)-1) = 6   ← 2-adic으로 1에 극도로 근접!
  v₃(R(36)-1) = -3   ← 3-adic으로는 매우 먼
  |R(36)-1|₂ = 2⁻⁶ = 1/64
  |R(36)-1|₃ = 3³ = 27
```

36 = 6²이므로 완전수의 제곱에서 2-adic 근접도가 폭발적으로 증가한다.
64 = 2⁶ 은 바로 36 = 6² 에서 f(2,2) = 7/6, f(3,2) = 13/9 의 곱에서 나온다:
R(36) = (7/6)·(13/9) = 91/54 ... 아니, 정확히는 R(36) = 91/27.

### Hensel 수렴 도식

```
  k=1:  ███████████████  (8~13개)    다양한 n이 p-adic 1 근방
  k=2:  ████████         (3~5개)     급격히 축소
  k=3:  ██               (2~3개)     거의 {1,6}만 남음
  k=4:  ██               (2개)       {1, 6} 확정
           ↓
  k→∞:  {1, 6}  ← 모든 소수 p에서 동일하게 수렴!
```

**핵심 정리 후보**: 모든 소수 p에 대해, k가 충분히 크면
R(n) ≡ 1 (mod p^k)를 만족하는 자연수는 {1, 6}뿐이다.

이는 6이 "p-adic 완전성"을 가짐을 뜻한다: R(6)=1이 근사가 아닌 정확한 등식이기 때문에
모든 p-adic 근방에서 살아남는 유일한 비자명 해이다.

단, p=2에서 n=36이 k=4까지 생존하는 것은 주목할 만하다.
이는 36=6²이 완전수의 "p-adic 그림자"를 형성하기 때문이다.

## 4. 초거리 군집 구조

|R(m)-R(n)|₂ 를 기준으로 군집을 분석한다. v₂가 높을수록 2-adic으로 가깝다.

```
  2-adic 거리 행렬 (v₂ 표시, 높을수록 가까움):

          1    2    3    6   10   12   20   36
    1     .   -2    0  inf    2    0    0    6
    2    -2    .   -2   -2   -2   -2   -2   -2
    3     0   -2    .    0    0    1    1    0
    6   inf   -2    0    .    2    0    0    6
   10     2   -2    0    2    .    0    0    2
   12     0   -2    1    0    0    .    3    0
   20     0   -2    1    0    0    3    .    0
   36     6   -2    0    6    2    0    0    .
```

**군집 패턴**:
- Cluster A: {1, 6} — 거리 0 (동일, R=1)
- Cluster B: {1, 6, 36} — v₂=6으로 매우 가까움 (2-adic)
- Cluster C: {12, 20} — v₂=3으로 중간 근접
- Outlier: {2} — 모든 것과 v₂=-2 (가장 멀리 떨어짐)

```
  2-adic 덴드로그램:

  v₂=∞  ─┬─ 1
         └─ 6
  v₂=6  ─┬─ {1,6}
         └─ 36
  v₂=3  ─┬─ 12
         └─ 20
  v₂=2  ─┬─ {1,6,36}
         └─ 10
  v₂=1  ─┬─ 3
         └─ {12,20}
  v₂=0  ─── (대부분의 n)
  v₂=-2 ─── 2 (고립점)
```

n=2의 고립은 R(2)=3/4의 분모가 4=2²이기 때문이다.
n=36의 {1,6}과의 근접은 R(36)-1=2⁶/3³에서 분자의 높은 2-멱이 원인이다.

## 5. p-adic 보간 가능성

R(n)을 Z_p 위의 p-adic 해석함수로 확장할 수 있는가?

R(n)은 곱셈적 함수의 비율이므로, Dirichlet 급수
Σ R(n) n^(-s) 의 p-adic analogue를 구성할 수 있다.

그러나 R(n)이 곱셈적이지 않다는 것이 문제다.
R(n) = (σ*φ)(n) / (n·τ(n)) 에서 τ(n)이 완전곱셈적이 아니기 때문이다.

가능한 접근:
1. **p-adic 측도**: μ_p(n) = |R(n)-1|_p 를 Z_p 위의 측도로 해석
2. **Kubota-Leopoldt**: σ(n) = Σ_{d|n} d 는 Eisenstein 급수와 연결.
   p-adic Eisenstein 급수 G_k,p 가 존재하므로, σ의 p-adic 보간은 자연스럽다.
3. **Iwasawa 이론**: R(p^a) = f(p,a)의 a→∞ 극한에서 p-adic 수렴 분석

## 6. Kummer 합동식과 B₂=1/6

Kummer 합동식: k ≡ m (mod p-1), k,m 짝수이면 B_k/k ≡ B_m/m (mod p).

- B₂ = 1/6, 그리고 6은 R=1인 유일한 비자명 수
- σ(n)/n = Σ_{d|n} 1/d 는 Bernoulli 수의 특수값과 관련
- σ(p^a)/p^a = 1 + 1/p + ... + 1/p^a → (1 - 1/p)^(-1) = ζ_p(1) (p-adic)

이 연결은 아직 추측 단계이지만, R(n)=1 조건이
ζ(s)의 특수값 관계와 동치일 가능성이 있다 (H-MP-8 참조).

## 한계

1. **유한 범위**: n≤50에서만 검증. n>50에서 새로운 k=3 이상 생존자가 있을 수 있다.
2. **증명 부재**: {1,6}으로의 수렴은 관측이지, 증명이 아니다.
   반례가 n>50에 존재할 가능성을 배제하지 못한다.
3. **보간 미완성**: p-adic 해석함수로의 확장은 구성이 필요하다.
4. **Bernoulli 연결**: B₂=1/6과의 관계는 아직 수치적 우연일 수 있다.
