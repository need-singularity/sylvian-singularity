# H-MP-18: R-chain basin(6)의 밀도 점근

> **가설**: basin(6) = {n : R-chain이 6을 경유하는 n}의 밀도가
> N→∞에서 수렴하며, 그 극한이 산술 상수로 표현된다.

## 데이터

```
  N=50000: basin(6) = 7007/50000 = 14.01%

  basin(6)의 정확한 극한 = ?
  후보: 1/7 = 14.29%, 1/M₃ = 14.29%
       ln(4/3) / 2 = 14.38%
       φ(6)/σ(6) = 2/12 = 16.67%
```
