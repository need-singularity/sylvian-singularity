# H-MILL-2: BSD 추측과 합동수 6

> **가설**: 6이 합동수(congruent number)인 것은 σφ=nτ 체계와 연결되며,
> (3,4,5) 피타고라스 삼중의 면적=6이 산술함수로 표현된다.

## 확인된 연결

```
  6 = 합동수: 직각삼각형 (3,4,5)의 면적
  y²=x³-36x: rank≥1 (BSD: L(E_6,1)=0)

  (3,4,5)와 σ,τ의 대응:
    3 = σ(6)/τ(6) = 12/4
    4 = τ(6)
    5 = σ(6) - M₃ = 12-7
    면적 = 3·4/2 = 6 = P₁!

  곡선의 conductor:
    y²=x³+1: conductor=36=6² (P₁²)
    y²=x³-36x: conductor=2048·... (더 복잡)

  Catalan 연결:
    3²-2³=1 → (3,4)가 연속 정수 → (3,4,5) 피타고라스
    → 면적 6 = 합동수 = 완전수!
    ALL from "2와 3의 근접성"
```
