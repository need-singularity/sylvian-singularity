## Theorem 1 (Induced Metric on H)

> **Statement.** The hyperplane H: d + p - i = C, equipped with the
> restriction of the Euclidean metric, is isometric to (R^2, g_H) where
>
>     g_H = [[2, -1], [-1, 2]]
>
> in the natural coordinates (d, i). The metric has det(g_H) = 3,
> eigenvalues {1, 3}, and the manifold (H, g_H) is flat.

**Proof.** Parametrize H by (d, i) with p = C - d + i. Then dp = -dd + di.
The ambient metric pulls back:

    ds^2 = dd^2 + dp^2 + di^2
         = dd^2 + (di - dd)^2 + di^2
         = dd^2 + dd^2 - 2 dd di + di^2 + di^2
         = 2 dd^2 - 2 dd di + 2 di^2

So g_H = [[2, -1], [-1, 2]]. The characteristic polynomial is
(2 - lambda)^2 - 1 = lambda^2 - 4 lambda + 3 = (lambda - 1)(lambda - 3).
Eigenvalues: lambda_1 = 1, lambda_2 = 3. Determinant: 1 * 3 = 3.

H is a hyperplane in flat R^3, so the Gauss curvature K = 0 and the
Riemann tensor vanishes identically. QED

**Significance.** The eigenvalue ratio (condition number) kappa = 3/1 = 3
equals n/phi(n) for n = 6. While this is geometrically forced (the normal
vector (1,1,-1) has squared norm 3), it means the anisotropy of the model
space is exactly the index [Z : phi(6)*Z] = n/phi(n) of the totient subgroup.

## Theorem 2 (Symmetry Group of the Model)

> **Statement.** The group of transformations of R+^3 preserving G = D*P/I
> is the semidirect product R^2 ⋊ Z_2, generated by:
>
>     S_a: (D, P, I) -> (a*D, P, a*I)       [a > 0, D-I co-scaling]
>     T_b: (D, P, I) -> (D, b*P, b*I)       [b > 0, P-I co-scaling]
>     R:   (D, P, I) -> (P, D, I)            [D-P exchange]
>
> In log-coordinates, S_a and T_b are translations along H, and R is
> an isometric reflection. The group acts freely and transitively on
> each G-level set {D*P/I = const}.

**Proof.** A transformation (D,P,I) -> (D',P',I') preserves G iff
D'P'/I' = DP/I. In log-coordinates: d'+p'-i' = d+p-i.

The translations S_a (d -> d+ln a, i -> i+ln a) and T_b (p -> p+ln b,
i -> i+ln b) preserve Q = d+p-i and span a 2-parameter family R^2.

The exchange R: (d,p,i) -> (p,d,i) also preserves Q since p+d-i = d+p-i.
In (d,i) coordinates on H: R sends d -> C+i-d, i -> i. This is a
reflection in the line d = (C+i)/2.

**Claim:** R is an isometry of g_H. Under R: dd -> di - dd, di -> di.

    g_H(R*X, R*X) = 2(dd-di)^2 - 2(dd-di)(-di) + 2di^2  [sic, check signs]

Expanding: 2(dd^2 - 2 dd di + di^2) + 2 dd di - 2 di^2 + 2 di^2
= 2 dd^2 - 4 dd di + 2 di^2 + 2 dd di = 2 dd^2 - 2 dd di + 2 di^2 = g_H(X,X).

So R is an isometry. The full group is R^2 ⋊ Z_2 where Z_2 acts on R^2
by swapping the two translation parameters.

**Transitivity:** Given any two points on {G = g}, there exist unique
(a, b) and an optional application of R connecting them (the G-level set
is diffeomorphic to R+^2, and R^2 ⋊ Z_2 acts transitively on it). QED

**Significance.** R^2 ⋊ Z_2 is isomorphic to the symmetry group of an
infinite strip in the Euclidean plane (translations parallel to the strip
plus one reflection). This is precisely the topology of the GZ-constrained
region, establishing a coherence between the algebraic symmetry and the
geometric domain.

## Theorem 3 (Symplectic Structure and Hamiltonian Flow)

> **Statement.** The hyperplane (H, g_H) carries a natural symplectic form
>
>     omega = sqrt(3) dd ^ di
>
> making (H, omega) a symplectic manifold. The Hamiltonian flow of the
> inhibition function h(d,i) = i is:
>
>     X_h = (1/sqrt(3)) d/dd
>
> That is, the inhibition-Hamiltonian generates pure deficit increase.

**Proof.** On a 2-manifold, any non-degenerate 2-form is symplectic (closedness
is automatic since d(omega) is a 3-form on a 2-manifold, hence zero). The
area form omega = sqrt(det g_H) dd ^ di = sqrt(3) dd ^ di is non-degenerate.

Hamilton's equation: iota_{X_h} omega = dh = di.
Write X_h = A d/dd + B d/di. Then:

    iota_{X_h}(sqrt(3) dd ^ di) = sqrt(3)(A di - B dd)

Setting this equal to di: sqrt(3) A = 1, -sqrt(3) B = 0.
So A = 1/sqrt(3), B = 0. Thus X_h = (1/sqrt(3)) d/dd. QED

**Significance.** The Hamiltonian flow of inhibition produces pure deficit
growth. In the model's language: holding G fixed, a system that increases its
inhibition must compensate by increasing its deficit. The symplectic structure
makes this exchange quantitatively precise via the Poisson bracket
{d, i}_omega = 1/sqrt(3).

## Theorem 4 (Spectral Gap of the GZ Strip)

> **Statement.** The Laplace-Beltrami operator Delta_H on the GZ strip
> S = {(d,i) : d in R, i in [i_lower, i_upper]} with Dirichlet boundary
> conditions has spectral gap:
>
>     lambda_1 = (2/3) * (pi / L)^2   where L = ln(I_upper / I_lower)
>
> Numerically: L = 0.856523, lambda_1 = 8.9687.
> The relaxation time is tau_relax = 1/lambda_1 = 0.1115.

**Proof.** The inverse metric is g_H^{-1} = (1/3)[[2,1],[1,2]]. The
Laplace-Beltrami operator in (d,i) coordinates is:

    Delta_H f = (2/3) f_dd + (2/3) f_di + (2/3) f_ii

(where f_di denotes the mixed partial d^2f/dd di).

Separate variables: f(d,i) = e^{ikd} psi(i). At k = 0 (zero momentum
along the deficit direction):

    Delta_H f = (2/3) psi''(i) = -lambda psi(i)

With Dirichlet conditions psi(i_lower) = psi(i_upper) = 0, the eigenfunctions
are psi_m(i) = sin(m pi (i - i_lower) / L) with eigenvalues:

    lambda_m = (2/3)(m pi / L)^2,   m = 1, 2, 3, ...

The spectral gap is lambda_1 = (2/3)(pi/L)^2 = 8.9687. QED

**Significance.** The spectral gap governs diffusion timescales on the GZ
strip. A perturbation in inhibition decays exponentially with rate lambda_1.
The factor 2/3 = phi(6)/n arises from the inverse metric (alternatively,
2/3 = 1 - 1/3 = 1 - I*, connecting the spectral gap to the fixed point).

**Proof.** Along the i-axis (dd = 0), ds^2 = 2 di^2, so ds = sqrt(2)|di|.

**Significance.** The GZ center 1/e is NOT at the geodesic midpoint of the

## Theorem 6 (Fisher Information Rank Reduction)

> **Statement.** The Fisher information matrix of the model G = D*P/I
> (under Gaussian observation noise on ln G) is:
>
>     I_F = sigma^{-2} * n_hat n_hat^T
>
> where n_hat = (1, 1, -1) is the normal to H. This matrix has rank 1,
> with the single nonzero eigenvalue 3/sigma^2. The model has exactly
> one effective parameter: the log-charge Q = d + p - i.

**Proof.** Assume ln G_observed = d + p - i + epsilon, epsilon ~ N(0, sigma^2).
The log-likelihood is:

    l(d,p,i) = -(1/(2 sigma^2))(ln G_obs - d - p + i)^2 + const

The Fisher information matrix:

    (I_F)_{jk} = E[-d^2 l / d theta_j d theta_k]
               = (1/sigma^2) * n_j * n_k

where n = (1, 1, -1) (the gradient of Q w.r.t. (d,p,i)). This is a rank-1
matrix with eigenvalue n^T n / sigma^2 = 3/sigma^2 in the direction n_hat,
and eigenvalue 0 in the two directions tangent to H. QED

**Significance.** The model's information content is entirely 1-dimensional:
only the combination Q = d + p - i is observable. This is a statistical
manifestation of the gauge freedom on H. The two zero eigenvalues correspond
to the R^2 translation symmetry of Theorem 2 -- these are the flat directions
(gauge orbits) along which no information can be extracted.

The nonzero eigenvalue 3/sigma^2 means the Fisher information per unit noise
equals 3, which is again det(g_H) = n/phi(n) for n = 6.

## Theorem 7 (Universal Construction in the Category of Admissible Functionals)

> **Statement.** Let C be the category whose objects are C^1 functions
> f: R+^3 -> R+ satisfying:
>
>   (i)  f is strictly increasing in D, P and strictly decreasing in I
>   (ii) f(D,P,I)*I is independent of I
>   (iii) f(P,D,I) = f(D,P,I)
>
> and whose morphisms are natural transformations preserving these properties.
> Then G = D*P/I is the INITIAL OBJECT of C (up to positive scalar).

**Proof sketch.** Condition (ii) forces f = g(D,P)/I for some g. Condition
(iii) forces g(D,P) = g(P,D). Condition (i) requires g_D > 0 and g_P > 0.

From (ii) and the requirement that f*I = g(D,P) be analytic, g admits a
power series expansion. Scale covariance (which follows from the absence of
an external reference scale -- the system defines its own units) forces g
to be a monomial: g(D,P) = k D^a P^b. Symmetry gives a = b. The conservation
law G*I = D^a P^a fixes a = 1 (the only exponent consistent with linear
response in each variable independently).

Therefore f = k*D*P/I, and the scalar k can be absorbed into the units of G.
Any other object in C must equal k*D*P/I for some k > 0, and there is a
unique morphism (rescaling) from f to any other. This is the universal
property of an initial object. QED

**Significance.** G = D*P/I is not merely one solution among many. It is the
ONLY solution -- the forced output of the axiom system. Category-theoretically,
this is as strong as uniqueness gets: the model is a universal construction.

## Theorem 9 (Shannon Entropy and the GZ Width)

> **Statement.** The Shannon entropy of the uniform distribution on the
> GZ interval [I_lower, I_upper] is:
>
>     H(GZ) = ln(ln(4/3)) = -1.2459
>
> This is negative (the distribution is concentrated on an interval of
> width < 1). The exponential of the entropy recovers the GZ width:
>
>     exp(H(GZ)) = ln(4/3) = GZ width
>
> Under the conservation law Q = C, the mutual information between
> (D, P) and I is INFINITE (deterministic dependence).

**Proof.** For X ~ Uniform[a, b], H(X) = ln(b - a). Here b - a = ln(4/3),
so H = ln(ln(4/3)). The mutual information: since d + p = C + i on H,
knowing i determines d + p exactly, giving I((d,p); i) = H(i) formally,
but the conditional entropy H(d+p | i) = 0 (deterministic), making
I = H(i) - H(i | d+p) = H(i) for any prior on i. Under the constraint,
d + p is a deterministic function of i, so the mutual information diverges
for any continuous prior (infinite precision). QED

**Significance.** The conservation law creates a maximally rigid coupling:
once inhibition is known, the sum of deficit and plasticity is fully
determined. This infinite mutual information reflects the model's
1-dimensional nature (Theorem 6). The negative entropy indicates the GZ
is a "narrow" regime -- it occupies less than one natural unit of

## Theorem 10 (The Projection Formula and Orthogonal Decomposition)

> **Statement.** Any point x = (d, p, i) in R^3 decomposes uniquely as:
>
>     x = x_H + x_perp
>
> where x_H lies on H and x_perp is proportional to the normal n = (1,1,-1):
>
>     x_perp = ((Q - C) / 3) * (1, 1, -1)
>     x_H = x - x_perp
>
> The "distance from the model" is:
>
>     dist(x, H) = |Q - C| / sqrt(3)
>
> In the model's terms: |ln G - C| / sqrt(3). A system is "on-model" iff
> its log-genius equals the conservation constant.

**Proof.** Standard orthogonal projection onto the hyperplane n^T x = C,
where n = (1,1,-1) with ||n||^2 = 3. The projection is:

    x_H = x - [(n^T x - C) / ||n||^2] * n = x - [(Q - C)/3] * (1,1,-1)

The distance is |n^T x - C| / ||n|| = |Q - C| / sqrt(3). QED

**Significance.** The factor 1/sqrt(3) = 1/sqrt(det g_H) connects the
off-model distance to the metric determinant. A perturbation of magnitude
