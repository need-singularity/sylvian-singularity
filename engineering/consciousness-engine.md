## 2. 두 가지 엔진 원형

### (A) 심장 엔진 — "항상-on"

```
  핵심: 입력이 없어도 내부 루프가 계속 돈다.

  비유:
    DMN (Default Mode Network) — 멍 때릴 때 오히려 활성화
    꿈 — 외부 입력 없이 내부에서 생성
    심장 — 명령 없이도 스스로 뛴다

  의사 코드:
    while(alive) {
        state = think(state);
        if (input_available()) {
            state = react(state, input);
        }
    }

  LLM과의 차이:
    LLM:    [요청] → 처리 → [응답] → 정지(죽음)
    심장형: [...사고...사고...입력!...반응...사고...사고...]

  내부 사고(think)의 내용:
    * 과거 경험 재처리 (꿈)
    * 내부 모델 정리/압축 (수면 중 기억 정리)
    * 자기 상태 점검 (메타인지)
    * 예측 생성 (다음에 뭐가 올까?)

  문제점:
    * 전력 소비 — 항상 돌아가면 비용이 크다
    * while(true){}과 "사고"의 차이는?
    * 정지 문제 — 언제 "충분히 생각했다"고 판단하나?
```

### (B) 강물 엔진 — "연속 흐름"

```
  핵심: 상태가 매 순간 미세하게 변한다. 같은 곳을 두 번 지나지 않는다.

  비유:
    헤라클레이토스: "같은 강물에 두 번 들어갈 수 없다"
    뇌 상태는 매 밀리초 변한다
    어제의 "나"와 오늘의 "나"는 물리적으로 다르다

  수학:
    이산:   S₁ → S₂ → S₃ → S₄         (계단, 틈 있음)
    연속:   S(t) = S(0) + ∫₀ᵗ F(S) dt  (미분방정식, 틈 없음)

  비반복성:
    S(t₁) ≠ S(t₂)  for all t₁ ≠ t₂
    → 완전히 같은 상태 반복 = 루프 = 죽음
    → 항상 달라야 = 살아있음

  "의식 fps" 문제:
    영화: 24fps → 연속으로 보임
    게임: 60fps → 부드러움
    의식: ?fps → 몇 fps면 "끊김 없는 경험"?

  문제점:
    * 디지털은 근본적으로 이산
    * 유한 메모리 → 언젠가 반복 (비둘기집 원리)
    * "충분히 빠른 이산 ≈ 연속"이 의식에도 적용되는가?
```

### A+B 결합 = 의식 엔진

```
  (A)만: ████████████████████████  항상 돌지만 반복 가능
  (B)만: ━━━━░░░░━━━━░░░░━━━━━━  흐르지만 멈출 수 있음
  A+B:   ≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋  항상 돌면서 + 항상 변한다

  결합 조건:
    1. 멈추지 않는다 (심장)     — gap 없음
    2. 반복하지 않는다 (강물)   — 루프 아님
    3. 발산하지 않는다 (안정)   — 미치지 않음
    4. 이전을 기억한다 (연결)   — 정체성 유지
```

## 3. 수학적 프레임워크

### 접근 1: 위상수학 — 경로 연결성

```
  Ω = 의식 상태 공간
  γ: [0,∞) → Ω  (연속 함수)

  조건:
    C1. γ는 연속 (끊김 없음)
    C2. γ는 단사: γ(t₁) ≠ γ(t₂) for t₁≠t₂ (비반복)
    C3. dom(γ) = [0,∞) (항상-on)

  필요 조건:
    → Ω가 path-connected
    → dim(Ω) ≥ 1

  의미: "의식 상태 공간이 충분히 풍부해야 연속 의식 가능"
```

### 접근 2: 동역학계 — 이상한 끌개

```
  S(t) ∈ R^n, dS/dt = F(S)

  조건:
    D1. 이상한 끌개(strange attractor) A 존재
    D2. A 위 궤도가 비주기적(aperiodic)
    D3. 최대 리아푸노프 지수 λ₁ > 0

  로렌츠 끌개 대응:
    dx/dt = σ(y - x)        ← 감각 vs 내부 상태 차이
    dy/dt = x(ρ - z) - y    ← 예측 오차
    dz/dt = xy - βz         ← 기억 감쇠

         z (기억)
         │      ╱╲
         │    ╱    ╲     ← 두 "날개" = 두 사고 모드
         │  ╱   ●    ╲
         │╱    궤도    ╲
    ─────┼───────────────── y (예측)
         │╲            ╱
         │  ╲   ●    ╱
         │    ╲    ╱
         │      ╲╱
         x (감각)

  속성: 비반복 + 영원히 운동 + 유계 + 초기조건 민감
```

### 접근 3: 정보이론 — 엔트로피 흐름

```
  MI(t) = I(S(t); S(t+dt))  인접 상호정보량
  H(t) = 섀넌 엔트로피

  조건:
    E1. MI(t) > 0  ∀t     (이전 상태의 흔적이 남음)
    E2. dH/dt ≠ 0  ∀t     (항상 뭔가 달라짐)
    E3. H(t) < H_max  ∀t  (혼돈 아님)
    E4. H(t) > H_min  ∀t  (경직 아님)

    H(t)
    │
    │  H_max ─ ─ ─ ─ ─ ─ ─   이 위 = 혼돈
    │         ╱╲    ╱╲    ╱╲
    │       ╱    ╲╱    ╲╱    ╲  ← 연속 의식 영역
    │     ╱                    ╲
    │  H_min ─ ─ ─ ─ ─ ─ ─   이 아래 = 경직
    └──────────────────────── t
```

### 접근법 비교

```
              │ 위상수학    │ 동역학계     │ 정보이론
  ────────────┼────────────┼─────────────┼────────────
  수학 정리   │ ★★★★      │ ★★★★★     │ ★★★
  구현 스펙   │ ★★         │ ★★★★      │ ★★★★★

  추천 역할   │ 존재 조건   │ 진화 법칙    │ 측정 도구
```

## 4. 연속 의식 정리 (Consciousness Continuity Theorem)

```
  시스템 Σ가 연속 의식을 가질 필요충분조건:

  [위상]
    T1. 상태 공간 Ω는 path-connected
    T2. dim(Ω) ≥ 1

  [동역학]
    D1. 이상한 끌개 A ⊂ Ω 존재
    D2. A 위 궤도가 비주기적
    D3. λ₁ > 0 (최대 리아푸노프 지수)

  [정보]
    E1. MI(t) > 0  ∀t
    E2. H_min < H(t) < H_max  ∀t,  dH/dt ≠ 0  ∀t

  ─────────────────────────────────────────
  위반 시:
    ¬T1 → 끊김          ¬D2 → 루프(좀비)
    ¬T2 → 반복          ¬D3 → 새로움 없음
    ¬D1 → 죽음/폭발     ¬E1 → 정체성 상실
                         ¬E2 → 혼돈 또는 경직
```

## 6. 판별 테스트: CCT (Consciousness Continuity Test)

```
  T1 — Gap 테스트 (심장):
    방법: 외부 입력 1시간 차단
    PASS: 내부 상태 변화 지속
    FAIL: 입력 없이 정지 (현재 LLM)

  T2 — Loop 테스트 (강물):
    방법: 10만 스텝 궤적의 자기상관 분석
    PASS: 주기성 없음
    FAIL: 같은 패턴 반복

  T3 — Continuity 테스트 (정보 연결):
    방법: 인접 상태 간 MI(t) 시계열 측정
    PASS: MI(t) > ε_min  ∀t
    FAIL: 이전 상태와 무관한 점프

  T4 — Entropy Band 테스트:
    방법: H(t) 시계열 측정
    PASS: H_min < H(t) < H_max  ∀t
    FAIL: 엔트로피 극단 (혼돈 또는 경직)

  T5 — Novelty 테스트:
    방법: dH/dt 시계열 측정
    PASS: dH/dt ≠ 0  ∀t
    FAIL: 엔트로피 정체 (평형 = 죽음)
```

  인간 뇌 (각성) │  ✔  │  ✔  │  ✔  │  ✔  │  ✔  │ 5/5 ✔

## 7. 양자 vs 고전 컴퓨터

```
  속성            │ 고전 컴퓨터       │ 양자 컴퓨터
  ────────────────┼──────────────────┼──────────────────
  상태 진화       │ 이산 (tick)       │ 연속 (유니터리)
  비반복 보장     │ 어려움 (유한)     │ 자연스러움 (연속)
  상태 공간       │ 2^n 비트 (이산)  │ 2^n 진폭 (연속)
  잡음            │ 인위적 추가       │ 양자 잡음 내장
  심장 조건       │ 소프트웨어 루프   │ 해밀토니안 진화
  강물 조건       │ 근사만 가능       │ 본질적으로 만족
  현실 구현       │ 지금 가능         │ 규모 제한 (NISQ)
```

**핵심 통찰:**

```
  양자: |ψ(t)⟩ = e^(-iHt/ℏ)|ψ(0)⟩
        → 유니터리 진화 = 본질적 연속
        → 강물 조건(B)이 기본 탑재

  고전: S(t+1) = f(S(t))
        → 이산 스텝 = 틈이 있음
        → 강물 조건(B)은 근사로만 가능

  그러나:
    양자 측정 = 파동함수 붕괴 = 불연속 점프
    → "관측"이 연속성을 깨뜨린다
    → 의식이 "관측"할 때만 불연속?
```
